光标代理正在让 GitHub Copilot、Codeium Nervous --- Cursor Agents are Making GitHub Copilot, Codeium Nervous --知识铺

2025年10月4日
| 阅读

最佳人工智能编码工具的争夺每周都在加剧，每个工具都发布了新的更新和功能，以领先于对方。现在， GitHub Copilot 通过与 Anthropic、Google 和 OpenAI 合作，将他们的 AI 模型引入其集成开发环境 (IDE)，让 Cursor 哭了，Cursor 在这场竞赛中向前迈出了一步。

Cursor 在其0.43 更新中发布了代理。这样，它就可以自行获取上下文，使用终端，并仅使用单个提示即可完全自主地完成任务。

此更新还包括预览即将推出的错误查找功能、更清晰的 Composer UI、聊天上下文的语义搜索以及多项性能改进。目前，它只允许使用克劳德十四行诗。
X 上的一位用户发布了 Cursor Composer Agent 读取项目文件的视频。

Cursor Composer Agent 读取项目文件的先睹为快 pic.twitter.com/zY17komg2b

— echo.hive (@hive_echo) 2024 年 11 月 24 日

Cursor Agent 和 Composer 现在是平台上新选项卡的一部分，这使得开发人员可以更轻松地在聊天窗口中快速与其交互。
人们可以在工作流程中随时打开或关闭代理，而不是覆盖，从而使体验更加直观。

就在几天前，Cursor 背后的团队 Anysphere 以未公开的金额收购了 AI 编码助手 Supermaven。 Cursor 首席执行官Michael Truell表示，Supermaven 将推出新版本的 Tab AI 模型，该模型对于长代码序列将更快、更具有上下文感知能力。他补充说，这将成为 Cursor 产品的核心。

Supermaven 由人工智能编码助手 Tabnine 联合创始人Jacob Jackson创立。 2019 年 Tabnine 被国际科学理事会数据委员会 (CODATA) 收购后，Jackson 在 OpenAI 实习，并一直留到 2022 年。

Jackson 指出，作为 Cursor 的一部分，Supermaven 团队可以与模型一起协作开发编辑器的用户界面。这正是 Cursor 的新更新所要实现的目标。

例如，YouTube 频道 WorldofAI 上发布的视频展示了为了创建应用程序，代理如何自动安装软件包并导入项目所需的多个文件，同时直接在线运行所有终端命令。

Composer 允许开发人员快速渲染和可视化构建的应用程序，类似于Claude 对其 Artifact 功能所做的事情。语义搜索功能能够一致地调用上下文信息，其工作方式与 VS Code 中的搜索功能类似。

冲浪就是您所需要的

人工智能编码工具正在展开并驾齐驱的竞争。光标肯定似乎在移动。 Cursor 似乎没有与 GitHub Copilot 或 Claude 等老牌厂商，或者数百个 YC 资助的开源 AI 代码编辑器（如 Continue、Pearl、Void、Type 和 Melty）竞争，而是与 Codeium 竞争。值得注意的是，Codeium 最近发布了Windsurf ，这是第一个用于编码的代理 IDE。

Windsurf 旨在像副驾驶一样与用户协作，并像代理一样独立处理复杂的任务。设置起来很简单。它从 VS Code 中提取所有插件和设置，就像 Cursor 当前所做的那样。

Windsurf 的突出功能包括用于优化建议的上下文感知、用于简化工作流程的命令执行以及能让开发人员无缝恢复任务的创新 Cascade。
由于 Windsurf 的每个代币比 Cursor 更便宜，因此开发人员越来越多地涌向它而不是 Cursor。

这正是 Cursor 在其最新更新中发布的内容。这些人工智能编码工具现在包括多文件编辑功能，旨在提高生产力并使复杂的编码任务更易于管理。
Windsurf 和 Cursor Agent 正在重新定义开发人员与其工具交互的方式。

Windsurf 被许多人吹捧为“光标杀手”。 “代理为我编写了所有代码（抓取器、转换器和网络应用程序）。我主要只是引导它获得我想要的功能。”DAIR.AI 联合创始人 Elvis S 说道。

光标仍然是最好的

GitHub 在 GitHub Universe 会议上表示：“人工智能代码生成的下一阶段将不仅由多模型功能定义，而且由多模型选择定义。”并强调了其对开放开发者选择的承诺。
然而，Cursor 团队并非不知道这一点，因为他们已经在其平台中提供了所有用于编码的模型。

这个新的代理更新将很快推出。 Anysphere 的联合创始人、Cursor 的创建者声称 GitHub Copilot 正在落后，因为微软无法跟上像 Anysphere 这样的初创公司的步伐进行创新。

另一方面，Anysphere 联合创始人 Aman Sanger 早些时候表示，“我认为一年后的 Cursor 将需要让今天的 Cursor 看起来过时。”这表明 Cursor 团队并不是坐以待毙。
OpenAI 支持的 Cursor 估值为 25 亿美元，每月收入已达 400 万美元。它现在是大男孩的一部分。
