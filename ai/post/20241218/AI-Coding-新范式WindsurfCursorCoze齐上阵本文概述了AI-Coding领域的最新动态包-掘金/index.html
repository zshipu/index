AI Coding 新范式：Windsurf、Cursor、Coze齐上阵本文概述了AI Coding领域的最新动态，包 - 掘金 --知识铺

| 阅读

本文主要介绍了 AI Coding 领域的新动态，包括 Codeium 的 Windsurf 及其多方面的创新和优势，如超强上下文感知、支持图片上传等；Cursor 的更新及新功能，如 Composer Agent 和 Bug Finder；Coze 在无代码开发、应用搭建等方面的情况。这些为不同水平的读者提供了 AI Coding 的技术革新和应用知识。

关联问题: Windsurf怎样收费？ Cursor新功能好用吗？ Coze应用如何变现？

原文链接：www.2agi.net/blog/ai-cod…

随着人工智能技术的飞速发展，AI Coding领域迎来了前所未有的变革。Codeium的Windsurf、Cursor的agent模式更新、Copilot的新版本以及Coze的AI应用能力，都在推动着编程领域的创新。本期我们将深入探讨这些新兴工具和技术，为不同背景的读者提供洞见，无论您是技术专家还是编程新手，都能在AI Coding的浪潮中找到自己的立足点。让我们一起探索这些技术革新背后的原理和实际应用，把握AI Coding带来的机遇。

概要

本文概述了AI Coding领域的最新动态，包括Codeium的Windsurf、Cursor的更新、Copilot的新版本和Coze的AI应用能力。文章将详细介绍 Windsurf 如何凭借其创新和优势进入 AI Coding 市场，Cursor 的最新重大更新内容，Copilot 如何在 AI Coding 热潮中定位自己，以及 Coze AI 应用的功能和用途。

通过这些介绍，读者可以全面了解 AI Coding 背后的技术革新和实际应用，无论技术水平如何，都能从中获得宝贵的知识和见解。

文章精选（2AGI.NET）

Windsurf：这款全新免费的 AI IDE 击败了Cursor 吗？超强上下文感知 、AI代理和工作流支持

1、Windsurf是Codium推出的一款新的AI IDE，它支持多平台并且与Cursor一样基于VS Code，宣称是首个具备“代理”功能的IDE，强调工作流和实时协作。

2、Windsurf的核心特点是超强的上下文感知能力，能够自动获取开发者的最新操作并保持同步，减少了开发者需要手动提供上下文信息的需求。

3、Windsurf提供了一个免费计划和相对便宜的Pro计划，与Cursor相比，它在某些方面可能提供更多的功能和更经济的价格，这使得Windsurf成为提升开发者生产力的一个有力竞争者。

Windsurf再升级，支持图片上传！

1、Windsurf的这次更新不仅满足了开发者的核心需求，还通过创新的AI操作简化了繁琐的工作流程，使得开发者能够更高效、更有趣地完成工作。

2、Windsurf 1.0.5版本引入了颠覆性的功能，特别是支持图片上传，这使得AI可以分析截图或图片中的关键信息，以提升问题诊断和自动化修复的效率。

3、更新还包括AI问题修复功能，即自动识别问题并提供解决方案，甚至直接执行修复，以及快捷键绑定优化，提高了代码差异和文件间切换的效率。

新晋AI编程神器干翻Cursor！

1、Windsurf结合了copilots（协作型智能体）和agents（独立型智能体）的能力，创造出了名为「Flow」的全新协作型智能体，能够深度理解代码库并实时感知程序员的操作。

2、Windsurf的新功能Cascade允许用户通过自然语言指令与IDE中的操作保持同步，自动执行代码编辑和终端命令，显著提高了编程效率。

3、Codeium公司已完成1.5亿美元的C轮融资，估值达到12.5亿美元，其平台支持约70种编程语言，并与多个流行开发环境集成，拥有超过70万用户和1,000多家企业客户。

更新了！带Agent的Cursor太疯狂了

1、Cursor 0.43版本推出了Composer Agent功能，这使得Cursor具备了完整的项目理解和编辑能力，能够深度理解代码库并实时感知用户操作。

2、Composer Agent可以通过聊天窗口启用，或在设置中启用“Enable Tools”选项，用户已经开始使用这项新功能来解析整个文件，并进行代码编辑和增强。

3、除了Composer Agent，Cursor的更新还包括Bug Finder功能，旨在检测潜在代码问题，帮助开发者提前修复，尽管这一功能目前还处于Beta阶段且不是免费的。

Cursor 搭配神级Prompt(Thinking Claude)，无敌编程利器

1、文章提供了一个详细的教程，展示如何利用Cursor和Thinking Claude Prompt进行高效的编程，使得即使是非专业编程人员也能轻松理解和使用这些工具。

2、通过使用Thinking Claude Prompt，Cursor能够进行深入的思考和分析，从而提供更全面和深思熟虑的编程解决方案。

3、文章强调了这种组合工具的强大潜力，表明它们可以成为编程领域的一个无敌利器，特别是对于那些寻求提高编程效率和质量的开发者来说。

AI 应用无代码开发教程：工作流模式详解

1、文章详细介绍了如何使用国产AI应用开发平台扣子（Coze）来构建一个简单的AI文言文生成器，无需编程基础。

2、通过图形化界面，作者展示了如何搭建后端AI模型和前端用户界面，并将其连接起来，实现用户输入白话文生成文言文的功能。

3、文章还提供了一个活动预告，邀请读者参加扣子开发者日，与字节跳动的AI工程师面对面交流AI应用的开发和推广。

大事记: 终于可以用coze搞钱变现了! — 不是付费模板

1、coze平台推出了新功能，使用户能够通过图形化界面“拖拉拽”创建网页，这一变化显著提升了coze的应用潜力和用户体验。

2、新功能的推出让coze从一个玩具变成了具有变现能力的实用工具，用户可以创建动态页面跳转和单词解释网站等，体验类似“独立站”的感觉。

3.、文章鼓励读者加入社区，共同学习如何使用coze的新功能，并提到英伟达官方直播也在推广社区链接，显示了社区的影响力和活跃度。

手把手教你搭建扣子(coze)AI前端应用

1、扣子（Coze）平台进行了重大更新，发布了自己的AI应用搭建IDE，使得用户可以轻松创建AI应用，其界面和功能与Zion函子类似，但目前功能上不如Zion全面细致。

2、文章通过实战案例，手把手教读者使用Coze搭建AI应用，从创建AI应用、编辑业务逻辑到用户界面的搭建，以及最终的应用发布。

3、文章还对比了Coze和Zion在UI设计、组件丰富度、屏幕适配和模拟器预览功能等方面的异同，指出Coze在某些细节上不如Zion全面，但也展现出Coze作为新兴平台的潜力和发展前景。
