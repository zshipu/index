访问量 500 Posted 2024-03-29 Updated 2024-09-12 53~69 min read AI开源项目繁多，本页面精选了业界全面的开源项目列表，涵盖大模型、AIGC工具、解决方案等多个方面。每个项目都详细列出了其特性和功能，方便读者根据自己的需求选择合适的开源项目，避免在网上大量搜索摸索。

大模型

Llama-Chinese 9K

  • 描述: Llama中文社区提供的完全开源可商用的中文Llama大模型。
  • 开源地址: GitHub

Baichuan-7B (支持中文) 5.7K

  • 描述: 采用LLaMA相同架构，在中文上进行预训练，可商用。
  • 开源地址: GitHub

OpenLLaMA 7.3K

  • 描述: 参考LLaMA代码，在Apache 2.0 license下重新实现和训练，使用RedPajama训练集合。
  • 开源地址: GitHub

Alpaca 29.3K

  • 描述: 基于lama7b + self-instruct数据指令微调，最早迈出LLaMA+SFT这一步的模型。
  • 开源地址: GitHub

Vicuna 36.2K

  • 描述: llama13b + ShareGPT对话数据微调，受协议限制，公布的权重为delta值。
  • 开源地址: GitHub

GPT4ALL 68.6K

  • 描述: LLaMA用80w的GPT3.5数据(code, story, conversation)微调而来。
  • 开源地址: GitHub

Chinese-LLaMA-Alpaca (支持中文) 18.1K

  • 描述: LLaMA + 扩词表 + lora，属于局部参数预训练，仿照Alpaca训练了Chinese Alpaca。
  • 开源地址: GitHub

StableVicuna 15.8K

  • 描述: LLaMA + FT + RHLF，完整复现ChatGPT路径。
  • 开源地址: GitHub

llama.cpp 63.8K

  • 描述: LLaMA推理优化，用C/C++实现的推理，不依赖显卡。
  • 开源地址: GitHub

框架和工具

Open-webui 11.5k

  • 描述: 专为LLM打造的先进Web界面工具，支持多种LLM大模型，兼容Ollama和OpenAI API。
  • 开源地址: GitHub

GPT_API_free 14.7k

  • 描述: 免费ChatGPT API Key，支持GPT4 API（免费），国内可用免费转发API。
  • 开源地址: GitHub
  • 使用说明: 一个可以获取免费ChatGPT API Key的开源项目

图片类

Stable-diffusion-webui 138k

  • 描述: 稳定扩散web界面，基于grado库开发，提供友好的浏览器界面。
  • 开源地址: GitHub

整体方案

Gpt-pilot 26.7k

  • 描述: 帮助开发人员更快速地构建应用程序，通过指定应用程序类型逐步编写代码。
  • 开源地址: GitHub

DIFY 42.4k

  • 描述: 开源的大语言模型应用开发平台，融合后端即服务和LLMOps理念，支持数百个模型，提供Prompt编排界面、RAG引擎、Agent框架等。
  • 开源地址: GitHub
  • 使用说明: 官方文档
