这个 GitHub 项目牛逼！在手机上用 ClaudeCode 或 Codex 。 - 知识铺

2025年10月5日
| 阅读

你使用 Claude Code 或 Codex ，让它跑了个很复杂时间或者耗时很久的任务，然后开着电脑，自己出去玩了。

可能曾希望能在手机上随时查看它们的进度，或者在离开电脑时也能保持连接。

GitHub 上有一个叫 Happy 的开源项目正是解决这个问题的。

逛逛已经部署好，用上瘾了。

01

项目简介

happy 是一个开源的移动与网页客户端，让你随时随地访问这些 AI 编程工具，同时保障隐私和安全。

这个项目由一群来自湾区不同咖啡店和黑客空间的工程师发起。

他们是重度使用 AI 编程的人，常常想要在休息或外出时查看代码生成状态，但缺乏合适的移动端工具。

于是，他们开发并开源了这个项目。

这个开源项目的核心功能之一是跨设备无缝切换。

在手机上实时查看电脑上运行 AI 生成任务的进度，也可以控制。如果回到电脑前，只需随意按下键盘，操作权就会立即切换回电脑端。

整个过程流畅且自然。

更牛的是，支持实时语音通讯和端到端加密。你的代码数据全程加密，隐私性得到充分保障。

同时，应用还提供消息推送功能，当 AI 需要额外授权或运行遇到错误时，你会立即收到通知。

02

如何使用

使用 Happy Coder 非常简单。

首先，你需要在手机上下载 App（支持 iOS 和 Android），或在网页端直接使用。

你可以直接在美区的 App Store 或者应用商店，搜索 Happy 就行了，就是下面这个东西。没在中国的 APP Store 上架哦。

然后在电脑上通过 npm 安装配套的CLI工具。

<span leaf=""><span>npm</span>&nbsp;install -g happy-coder</span>

安装后，你只需要将原本使用的 claude 或 codex 命令替换为 happy 或happy codex，即可启动连接。

<span leaf=""><span># Instead of: claude</span></span>

之前你应该输入 Claude 运行 Claude Code ，现在输入 happy 就行了。

输入后，如果选择的是 Mobile ，就会给你弹出一个二维码。

此时打开你手机上安装的 Happy，打开相机扫描你电脑上的二维码。

就会在你手机上看到你电脑这个 Claude Code 实例了。

你可以在手机 APP 上对 Claude Code 聊天对话，查看进度。

而这个时候，你的电脑端终端里面的 Claude Code 会编程下面这种类似待机的状态。

只需要按下空格，就能继续在电脑上使用 Claude Code 了。

<span leaf="">开源地址：<span>https</span>:<span>//github.com/slopus/happy</span></span>
