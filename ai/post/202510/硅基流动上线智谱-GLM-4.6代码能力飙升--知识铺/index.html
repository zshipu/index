模型效果

在硅基流动模型广场中体验 GLM-4.6，让它用苏轼的风格，写一首中秋节纪念他夫人的诗词：

鹧鸪天·中秋怀亡妻

玉盘高悬又中秋，寒光浸透小红楼。
十年尘梦风吹散，此夜孤心只添愁。

犹记轩窗正梳妆，为我低语问寒流。
梦醒不见当时月，惟有清辉照白头。

我们在 Claude Code 中调用 GLM-4.6，让它生成一个网页版的粒子互动沙盒。

提示词：写一个网页版的粒子互动沙盒。初始化画布上布满数百个随机位置的小圆点（粒子）。粒子默认做布朗运动（微小随机移动）。当鼠标移动时，在鼠标位置产生一个力场：靠近鼠标的粒子会受到朝向鼠标（引力）或远离鼠标（斥力）的力，力的大小随距离衰减。实现按键切换力场模式（如按 ‘g’ 引力，按 ‘r’ 斥力，按 ‘v’ 漩涡）。粒子可以设置不同的颜色或根据速度改变颜色。

方式一：一键安装及配置脚本

1. 在终端中运行以下命令：

<span leaf="">bash -c "$(curl -fsSL https://static01.siliconflow.cn/cdn/assets/ccsf_v2.sh)"

2. 提示输入 API Key 时，复制粘贴你的硅基流动 API Key。

3. 提示选择模型时，使用方向键选择要在 Claude Code 中使用的 GLM-4.6 / DeepSeek-V3.1-Terminus / Kimi K2-0905 / Qwen3-Coder 等模型。

4. 根据提示，复制相应命令并重启终端运行，应用配置：

5. 执行 Claude 命令，进入 Claude Code 并使用。

注意：目前 Claude Code 并不支持添加多个自定义模型（Custom Model），你可以再次执行上述 1-5 步，选择并更新 ANTHROPIC_MODEL 环境变量的方式切换模型。

Gen-CLI

Gen-CLI 基于开源的 Gemini-CLI 改造而来，目前已在 GitHub 开源（https://github.com/gen-cli/gen-cli/），你可以通过以下方式安装。

  1. 1. 确保操作系统已经安装 node 18+

  2. 2. 设置 API key 环境变量

<span leaf="">export SILICONFLOW_API_KEY="YOUR_API_KEY"
  1. 3. 运行 Gen-CLI

通过 npx 运行 repo：

<span leaf="">npx https://github.com/gen-cli/gen-cli

或者，通过 npm 安装命令：

<span leaf="">npm install -g @gen-cli/gen-cli

开发者首选的“Token 工厂”

—— 极速 · 经济 · 可靠

硅基流动的一站式大模型服务平台致力于为开发者提供极速响应、价格亲民、品类齐全、稳定丝滑的大模型 API。

开发者用一个密钥就能启用所有顶尖开源大模型。除了 GLM-4.6，硅基流动平台已上架包括 DeepSeek-V3.1-Terminus、Qwen-Image-Edit-2509、Qwen3-Next-80B-A3B、Ling-flash-2.0、Hunyuan-MT-7B、Qwen-Image、Kimi-K2-0905、Wan2.2、step3、Qwen3-Coder、MOSS-TTSD-V0.5、Qwen3-Embedding & Reranker、Qwen2.5VL-72B、CosyVoice2 在内的各类语言、图片、音频、视频、嵌入与重排序模型。其中，DeepSeek-R1 蒸馏版（8B、7B、1.5B）、Qwen3-8B 等多款大模型可免费使用，让开发者实现“Token 自由”。

开发者可在平台自由对比、组合各类大模型，只需在开发应用时轻松调用更易用、高效的 API，为你的生成式 AI 应用选择最佳实践。
