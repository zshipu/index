Claude Code 编码能力很强，业界公认，国庆放假前，刚发布了4.5之后，跑分打榜又进入全球第一。

腾讯Codebuddy是什么？

CodeBuddy是腾讯云自主研发的一款AI代码助手，目前ide插件、ide、cli都支持了，并且区分了国内版本和国外版本。

Cli更像是Claude Code 平替，它定位于专业工程师用专业的 CLI Agent，基于npm安装，支持用自然语言驱动整个开发运维生命周期，实现极致自动化效率提升。

选择完github之后，会在浏览器弹出认证页面，选择github

会进入到这个页面，登录进去，会发送邮箱的验证码进行验证。

这个也是无限邮箱的关键步骤，不止是接码，还要接验证链接

点击完认证链接后，回到命令行页面，按esc后，再次登录，就正常进入了
