（二）工作流框架

框架流程：

输入广告需求、提示词、视频尺寸、AI大模型等信息 -> 谷歌AI根据提示词生成创作视频的提示词-> 把视频提示词存进谷歌表格 -> 调用veo3视频模型提交生成视频任务 -> 每隔五秒检查是否生成成功  -> 如果生成成功则把生成的视频链接存入谷歌表格，否则继续查询。

（三）操作手册（图文）

3.1 准备工作

  1. 获取谷歌大模型key

  • 进入 https://aistudio.google.com/prompts/new_chat ，点击左下角【Get API key】

创建密钥保存在小本子。

注意：如果你无法显示谷歌页面，一般是魔法问题，节点切换成新加坡，还是不行的话换个牛逼的魔法。

  1. 获取视频AI大模型key

  • 进入  https://kie.ai  ， 初次注册需要过图形验证，关闭魔法即可通过。

  • 登录之后，右上角点击头像，然后点 【Dashboard】

选择 左侧的【API Key】创建密钥并保存。

进入 https://docs.google.com/spreadsheets ，新建一个表格，导入下面的表格文件。

3.2 流程开始

  1. 创建 【Set】节点

  • 作用：这个节点的作用主要是为了在工作流开始配置一些需要用到的信息，比如 生成视频的需求、提示词、生成视频的尺寸等。

点击工作流的【+】，在右侧输入“ Set ” ，选择 下图第一个节点。

给Set节点输入信息，具体如下图，如果没有输入框，点 下面橙色的【Add Field】。
