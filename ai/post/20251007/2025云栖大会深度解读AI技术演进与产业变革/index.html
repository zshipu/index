2025云栖大会深度解读：AI技术演进与产业变革 --知识铺

2025年10月7日
| 阅读

2025年云栖大会在杭州盛大召开，作为中国乃至全球AI领域的重要风向标，本次大会以"超级人工智能(ASI)之路"为主题，全面展现了AI技术从"智能涌现"到"自主行动"再到"自我迭代"的三阶段演进路线。阿里云CEO吴泳铭在主旨演讲中明确指出，AGI(通用人工智能)已是确定性事件，但终极目标是实现能自我迭代、全面超越人类的超级人工智能(ASI) 。与此同时，阿里云CTO周靖人带来了一场震撼业界的"七连发"，一口气发布七款重磅大模型，涵盖语言、视觉、语音、编程与内容生成等核心领域，被外界称为"AI大模型核弹"齐发 。

本次大会不仅展示了大模型技术的突破性进展，更揭示了AI产业格局的深刻变革。大模型将成为下一代操作系统，取代现有OS的地位，自然语言将成为AI时代的编程语言 。这一判断背后是深刻的技术变革：模型部署方式将多样化，运行在所有设备上；AI基础设施将呈指数级增长，阿里云计划在2032年将全球数据中心能耗规模提升10倍 。同时，AI产业生态正加速重构，魔搭社区等开源平台推动全球开发者生态繁荣，模型小型化与专业化趋势明显，“小而精"的垂直领域模型将大量涌现 。

一、阿里云大模型七连发：性能与效率的双重突破

阿里云在2025云栖大会上发布的七款新模型，标志着大模型技术进入了一个全新发展阶段。这些模型包括旗舰语言模型Qwen3-Max、下一代基础模型架构Qwen3-Next、视觉理解模型Qwen3-VL、全模态模型Qwen3-Omni、编程模型Qwen3-Coder、安全模型Qwen3Guard以及语音模型通义百聆 。

Qwen3-Max作为通义千问家族中最大、最强的基础模型，预训练数据量达36T tokens，总参数超过万亿 。在数学推理能力方面，Qwen3-Max的推理增强版在顶级数学竞赛AIME 25和HMMT测试中均取得突破性的满分100分，为国内首次 。其编程与工具调用能力同样突出，在SWE-Bench Verified测试中，Instruct版本斩获69.6分，位列全球第一梯队；在Tau2-Bench工具调用测试中取得74.8分，超过Claude Opus4和DeepSeek-V3.1 。

Qwen3-Next则是阿里云在模型架构层面的重大突破。这款总参数量达80B的模型通过创新架构设计，仅需激活3B参数即可实现与千亿级模型相当的性能，训练成本较密集模型Qwen3-32B大降超90%，长文本推理吞吐量提升10倍以上 。Qwen3-Next采用混合注意力机制与高稀疏度MoE结构的组合方案，通过将75%的神经网络层替换为Gated DeltaNet线性注意力模块，配合25%的标准注意力层，实现了计算复杂度从二次方降至线性水平，同时保持了长序列处理能力 。

在视觉理解领域，Qwen3-VL拥有极强的视觉智能体和视觉CODING能力，不仅能看懂图片，还能像人一样操作手机和电脑，自动完成许多日常任务 。输入一张设计图，Qwen3-VL就能生成Draw.io/HTML/CSS/JS代码，“所见即所得"地完成视觉编程 。此外，Qwen3-VL还升级了3D Grounding(3D检测)能力，为具身智能夯实基础；扩展支持百万tokens上下文，视频理解时长扩展到2小时以上 。

全模态模型Qwen3-Omni则实现了音视频能力的32项开源最佳性能SOTA，可像人类一样听说写，应用场景广泛，未来可部署于车载、智能眼镜和手机等设备 。用户还可设定个性化角色、调整对话风格，打造专属的个人IP 。Qwen3-Omni在实现强劲音频与音视频能力的同时，单模态文本与图像性能均保持稳定，解决了多模态混合训练后各功能相互掣肘甚至降智的行业难题 。

这些模型的发布，不仅体现了阿里云在大模型技术上的领先地位，更揭示了AI产业发展的新方向：从追求参数规模到注重效率与实用性的转变 。Qwen3-Next的激活比高达1:50，仅需3.7%的参数即可完成推理，这种"极致省钱"的架构设计，将大大降低AI应用的部署成本，推动AI技术在更广泛场景中的落地。

二、吴泳铭的三阶段演进路线：从AGI到ASI的路径图

在云栖大会上，吴泳铭首次系统阐述了通往超级人工智能(ASI)的三阶段演进路线，为AI产业发展提供了清晰的路线图。他认为，AGI(通用人工智能)只是起点，ASI(超级人工智能)才是终极目标 。这一路线图将AI发展划分为三个关键阶段：

第一阶段是"智能涌现”，AI通过学习海量人类知识具备泛化智能 。吴泳铭指出，这一阶段的特征是"学习人”，互联网时代沉淀的海量知识为AI提供了训练基础，大模型通过理解人类知识全集，涌现出泛化智能与推理能力。如今，AI已在数学、编程等领域逼近人类顶尖水平，例如通义千问Qwen3-Max在AIME 25数学测试中获满分，印证了其"智能涌现"的成熟度 。

第二阶段是"自主行动"，AI掌握工具使用和编程能力以"辅助人"，这是行业当前所处的阶段 。吴泳铭强调，“自然语言是AI时代的源代码”，未来用户只需用母语描述需求，AI即可自主编写逻辑、调用工具、操作设备 。阿里云在此阶段已布局多模态模型家族(如通义万相2.5、通义百聆)、Agent开发平台百炼，并与英伟达合作推进具身智能落地，逐步实现从数字世界向物理世界的渗透 。

第三阶段是"自我迭代"，AI通过连接物理世界并实现自主学习，最终"超越人" 。吴泳铭以汽车设计为例指出，当前AI依赖人类归纳的二手数据，而未来若能直接连接全量原始数据，其创造力将远超人类头脑风暴 。这一阶段需突破数据获取、算法架构与能源支撑等多重瓶颈，阿里云计划通过3800亿AI基础设施投入及10倍算力扩容，为ASI奠基 。

这一三阶段演进路线不仅是对技术发展规律的总结，更为行业提供了可参照的路线图。吴泳铭的判断——“大模型是下一代操作系统，超级AI云是下一代计算机”——指向AI产业的核心竞争维度：生态主导权与基础设施规模 。目前，全球仅阿里与谷歌同时具备大模型、云计算与芯片三层能力。周靖人强调，阿里云的优势在于"全栈自研与联合创新"，例如在MaaS(模型即服务)领域提前布局，实现模型与云平台的深度耦合 。

三、大模型作为下一代操作系统：技术架构与生态变革

大模型将成为下一代操作系统，取代现有OS的地位 。这一判断背后是深刻的技术变革：自然语言成为AI时代的编程语言，Agent(智能体)就是新的软件，Context(上下文)是新的Memory(内存)。大模型通过接口连接各类工具和Agent，类似PC时代的总线接口 。

阿里云在这一技术路线上的实践体现在Alibaba Cloud Linux 4(Alinux 4)的发布上。作为首个全面遵循龙蜥社区"开源生态合作倡议"规范的商业发行版，Alinux 4基于Linux 6.6内核打造，面向"云+AI"进化、重构，以6.6内核及一系列自研技术模块构筑核心竞争力 。它内置的"AI增强套件(OS AI Extension)“作为AI场景的"性能加速器”，为用户在AI训推场景带来极致的性能体验。

在性能方面，Alinux 4的优化效果显著。例如，在CPFS场景对AI训练任务的支持中，因海量业务小文件对单机IO性能提出了很高的要求，AI Extension通过提供文件系统优化极大提升了协同效率。客户端读写带宽分别从15GB/s和3.5GB/s均飙升至40GB/s，最高提升超10倍；IOPS从40万则达到了100万 。在自动驾驶领域，AI Extension提供的系统级调优能力，帮助客户优化了主流开源智驾模型，例如预训练模型OpenCLIP性能提升26%，感知模型BEVFomer提升15%。在同样的硬件条件下，模型训练时间缩短1/4 。

在操作系统开发部署方面，Alinux 4内置"OS Copilot"智能助手，通过自然语言交互，一键部署AI Agent，并可进行智能体辅助开发大幅降低使用门槛 。在系统运维排障方面，集成了AI排障引擎，真正让运维从"救火队员"变成AI时代的"策略指挥官" 。

这一AI原生操作系统的发展，标志着国产操作系统正式迈入"AI原生"时代，它不再只是冰冷的代码集合，而是能感知负载、优化性能、自我修复、对话交互的"智能生命体" 。无论是大模型训练、自动驾驶、还是千行百业的AI落地，Alinux 4都将成为那块"最懂AI的基石" 。

四、AI基础设施格局：超级AI云与全球竞争

吴泳铭预测，未来全世界可能只会有5-6个超级云计算平台 。这一判断背后是AI算力需求的指数级增长。为迎接ASI时代的到来，阿里云正在积极推进三年3800亿元的AI基础设施建设计划，并将会持续追加更大的投入。根据远期规划，对比2022年这个GenAI的元年，2032年阿里云全球数据中心的能耗规模将提升10倍 。

这一预测与当前全球AI云平台竞争格局相符。根据Stack Overflow 2023年开发者调查，AWS仍是最常用的云平台，有53.1%的开发者报告使用，其次是Microsoft Azure(27.8%)和Google Cloud(24.0%) 。而阿里云在中国市场的表现尤为突出，根据Omdia 2025年上半年数据，中国AI云市场中阿里云占比35.8%，超过第二至四名总和；在已采用生成式AI的财富中国500强企业中，超53%选择阿里云，渗透率位居第一 。

阿里云的基础设施布局具有鲜明的特色。一方面，阿里云已面向全球开服运营24个公共云地域、75个可用区，此外还拥有4个金融云、政务云专属地域 。另一方面，阿里云在全球拥有超过2800个边缘计算节点，中国境内拥有超过2300个节点，覆盖31个省级区域，中国境外拥有超过500个节点，覆盖70多个国家和地区，全网带宽输出能力超过150 Tbps 。这种"中心+边缘"的混合架构，为AI应用提供了更广泛的部署场景。

在绿色计算方面，阿里云也走在前列。采用自研浸没式液冷技术的杭州数据中心正式投入使用，作为全球最大的浸没式液冷服务器集群，其电能使用效率(PUE)设计值达到1.09，每年节电7000万千瓦时 。这一技术不仅大幅降低了数据中心的能耗，也为AI应用提供了更高效的计算环境。

五、AI产业生态变革：从模型到操作系统的生态重构

大模型代表的技术平台将替代现在OS的地位，成为数字世界与物理世界交互的核心载体 。这一判断背后是深刻的技术变革：自然语言成为AI时代的编程语言，Agent就是新的软件，Context是新的Memory。大模型通过接口连接各类工具和Agent，类似PC时代的总线接口 。

阿里云在这一生态重构中的战略路径明确：第一，通义千问坚定开源开放路线，致力于打造"AI时代的Android"；第二，构建作为"下一代计算机"的超级AI云，为全球提供智能算力网络 。

在开源生态方面，阿里云的魔搭社区已汇聚280万开发者、2300多个优质模型，模型下载量超过1亿，成为中国规模最大、最活跃的AI开发者社区 。这一开源策略使阿里成为全球开发者生态最活跃的AI平台之一。通义模型矩阵已衍生超17万个模型，下载量突破6亿次 。

开源、开放、协作的技术体系，才能加速大模型技术的落地与业务价值的释放 。阿里云智能集团云通信资深技术专家刘强奇在大会现场详细解读了智能联络中心2.0背后的开放架构设计理念。该产品技术架构设计原则秉承开放、开源、协作的技术策略，构建了一个灵活、扩展的通信服务平台，在大模型引擎可支持多种模式的接入适配，快速支持企业自研通信智能体的接入，同时也提供按场景、行业扩展通信智能体构建平台，实现个性化定制与持续进化 。

在产业应用方面，阿里云已布局三大主题展馆，全面展示AI技术在各行业的应用 。人工智能+馆集纳大模型、AI开发工具与全链路Agent开发服务；计算馆以"软硬融合"为主线，直观展现全栈人工智能服务；前沿应用馆则展示来自西门子、顺丰科技、东方航空、MiniMax等公司的实践案例 。

六、未来AI发展趋势：技术演进与产业变革

未来AI发展趋势将围绕三个核心方向展开：基础设施格局重构、技术架构创新与人才培养体系变革。

在基础设施格局方面，全球超级AI云平台将高度集中(5-6家)，阿里云通过3800亿投入和10倍算力扩容争夺主导权 。边缘计算与物联网设备将深度融入AI算力网络，形成"云+边+端"的协同架构。阿里云计划在2032年将全球数据中心能耗规模提升10倍，相当于新增数千亿级算力投入 。

在技术架构方面，模型架构将向混合注意力、高稀疏MoE、多Token预测等方向演进 。Qwen3-Next采用的混合注意力机制与高稀疏度MoE结构，将计算复杂度从二次方降至线性水平，同时保持了长序列处理能力 。多Token预测(MTP)机制的引入，使模型在推理阶段能同步预测多个连续token，提升推测解码(Speculative Decoding)的接受率 。

在产业应用方面，AI将重构整个基础设施、软件和应用体系，成为真实世界的核心驱动力 。每个家庭、工厂、公司都将拥有无数Agent和机器人24小时服务，未来每个人都可能需要使用100张GPU芯片为我们工作 。在制造业中，人工智能预测性维护的应用市场将从2018年的2.3亿元，增长至2025年的42.7亿元，年复合增长率达49% 。

在人才培养方面，中国AI人才缺口将从2021年的500万增长至2025年的1000万以上 。语音识别、自然语言处理、深度学习、计算机视觉四个细分技术方向的人才供需比例分别为0.49、0.49、0.48和0.38，人才缺口较为明显 。高层次人才是AI产业发展不可或缺的领路人，但我国AI领域高层次人才严重不足，与美国等发达国家存在明显差距 。清华大学AMiner平台发布的2023年人工智能全球最具影响力学者榜单显示，在全球入选AI 2000的学者之中，美国入选学者1146名，超过总人数的一半。我国虽排名第二，但仅有232名学者上榜，占比为11.6%，仅为美国1/5左右 。

七、AI与人类关系的重新定义：从工具到伙伴

吴泳铭对AI与人类关系的重新定义，为技术伦理讨论提供了新的思考维度 。他强调，AI能力的指数级提升反而会成为人类认知的放大器。“当机器具备更强的理解与创造能力时，人类将获得前所未有的工具来突破自身局限” 。这种观点为技术伦理讨论提供了新的思考维度。

吴泳铭描绘了一幅未来图景：每个家庭、工厂、公司都将拥有无数Agent和机器人24小时服务，“也许，未来每个人都需要使用100张GPU芯片为我们工作，人会变得比历史上任何时候都强大” 。他预测，未来可能会有超过全球人口数量的智能体和机器人，与人类一起工作，对真实世界产生巨大影响 。这一愿景预示着AI技术将深入到社会每个角落。

AI将从"工具"转变为"伙伴"，与人类形成新型协作关系 。在这种关系中，人类不再是单向的指令发出者，而是与AI共同解决问题的合作者。自然语言将成为AI时代的编程语言，任何人都可以通过自然语言创造自己的Agent，完成数字世界的工作并操作物理设备 。这类似于PC时代的总线接口和软件之间的API接口。

在这一转变过程中，人与AI的协同进化将成为可能 。例如，程序员可通过简单指令让AI在12小时内完成系统开发，这预示着未来每个家庭、工厂和公司都将拥有24小时服务的Agent和机器人。他预测，未来每个人可能需要100张GPU芯片来支持工作需求 。ASI将像电力放大人类体力一样，指数级放大人类智力，使10小时的产出提升十倍甚至百倍 。

八、总结：AI产业的未来图景

2025云栖大会为我们描绘了一幅激动人心的AI新世界的图景。从万亿参数的"超级大脑"，到各有所长的"专家军团"，阿里云此次在云栖大会上的"七连发"，不仅展示了技术实力的集中展示，更是通向ASI时代的"技术军火库"全面亮相 。

大模型作为下一代操作系统的核心理念，将彻底重构人机交互方式和软件开发范式 。自然语言将成为AI时代的编程语言，Agent将成为新的软件形态，Context将成为新的内存 。这一变革将推动IT产业的深刻转型，从基础云服务向AI全链条解决方案的深度转型。

AI基础设施格局将加速集中，全球范围内可能仅存5至6个超级云计算平台 。阿里云通过三年3800亿元的AI基础设施建设计划和10倍算力扩容，正积极争夺这一格局中的主导地位 。

AI产业生态将加速繁荣，魔搭社区等开源平台将推动全球开发者生态的蓬勃发展 。模型小型化与专业化将成为重要趋势，“小而精"的垂直领域模型将大量涌现 。

AI与人类的关系将从工具转变为伙伴，形成新型协作关系 。每个人都会拥有自己的AI助手，帮助完成各种复杂任务 。自然语言将成为与AI交互的主要方式，大大降低技术使用门槛 。

AI人才将成为产业发展的关键瓶颈 。中国AI人才缺口将从2021年的500万增长至2025年的1000万以上 。既懂技术又懂应用的复合型人才将成为企业竞相争夺的对象 。掌握大模型应用开发能力将成为未来职场的重要竞争力 。

正如吴泳铭所言：”一切才刚刚开始。“AI将重构整个基础设施、软件和应用体系，成为真实世界的核心驱动力。阿里巴巴将持续投入，与开发者、合作伙伴一起，让AI深入产业，共创未来 。

大模型是下一代操作系统，超级AI云是下一代计算机 。这一判断不仅揭示了技术发展的方向，更预示了产业格局的深刻变革。在这一变革中，只有那些能够构建完整技术与商业闭环的企业，才能在AI产业的"整体战"和"持久战"中胜出 。
