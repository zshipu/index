你知道传统的 AI 视频制作流程吗？

整个过程下来，要不停地打磨提示词，生图，图生视频，就比较繁琐。

可以看看小意之前做 AI 视频的教程：惊呆了！即梦上线智能多帧，一键 54 秒，一镜到底无需剪辑

对小白来不是很友好，但凡有一步卡住就会劝退！

小意这次用即梦 Agent 尝试剪了一个长沙旅行小视频，四个字形容：”太简单了！”

  • 打开即梦网页版：
  • 进入平台后选择“Agent 模式”
  • 输入提示词：“做 8 张长沙像素风图片，避开橘子洲头，毛泽东等敏感场景，竖屏，然后用智能多帧串起来”

黄色标记内容可进行替换哈！

  • 发送，等待成片。

这里注意有敏感场景，要在提示词中特别说明，否则就会像小意这样，少一张图，即梦 Agent 停止工作。

不过后续补充图片后，即梦 Agent 会继续把生成的图片运用智能多帧功能串起来。

这个过程不用你干什么，你只需要负责收片就行了。

连运镜转场都不需要你想，是不是很简单？

不用你想图生视频提示词。

如果想学习的话，点击视频，就能看到整个视频的提示词了！
