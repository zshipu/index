20251027 A股复盘｜逼近四千：科技领跑，节奏不变 --知识铺

2025年10月27日
| 阅读

前言｜磋商推进，月底至11月初总体偏利好

周末经多方的报导看来，谈的还行，至少形式上看，已经是这半年来最缓和的一种调调了。这也为月底韩国的日程营造了氛围；所以今天情绪也被打开，量价齐升。

结合我们之前提到的本月底4件大事，十五五定调、降息、谈判和APEC双龙会等日程，10月尾声到11月初的宏观环境整体更偏利好：政策预期更清晰、外部不确定性边际缓和、全球风险偏好修复的概率抬升（细项见下文“当日催化与宏观”）。

盘面速览｜指数放量上行，科技链多点开花

指数与成交：沪指收涨**+1.18%至3996.94**，深成指**+1.51%至13489.40**，创业板指**+1.98%至3234.45**；科创50**+1.50%至1484.21**。两市成交约2.34万亿，上涨个股超3300只，量价匹配度明显改善。

热点与板块：

  • 存储芯片全天强势，龙头与高景气个股创新高；

  • 算力硬件/CPO延续主线地位，光模块龙头盘中再创历史高位；企业侧对1.6T等新品放量预期积极，三季报披露的光模块公司多数利润同比增长；

  • 有色金属/稀有金属走强；

  • 核电/可控核聚变方向活跃；

  • 风电设备、游戏等板块相对承压；

  • 券商尾盘前后有异动拉升，金融街论坛（10/27–10/30）召开在即也带来关注度。

机构观点节选（要点转述）：

  • 华X证券：多重利好下“慢牛”延续，大科技仍属中长期主线；

  • 中X证券：风格切换已基本结束，业绩驱动回归，线索在产业链安全与AI端侧扩散；

  • 中T证券：在宏观缓和预期下维持指数偏强，若谈判结果不及预期，则主线应灵活在“大金融+新消费”与“自主可控+稀缺资源”间切换。

为什么今天没能临门一脚踹破4K？（个人看法）

一是权重金融未形成合力——银行、券商并未组织化推升；

二是无形之手大节奏上应该更倾向慢慢来，对指数节奏更注重可持续性与结构优化而非快攻拔点；

三是财报季节奏与事件窗口尚未完全验证，资金更愿意在主线科技与有业绩支撑的品种上做结构性行情而非“顶大关口”。因此乐观不变但仍重结构：跌有底、看主线、选业绩。

强弱与龙头

  • 强：存储芯片、CPO/光模块、算力硬件链、有色（金/钨/稀有）。

  • 中坚：核电与可控核聚变链条（事件驱动+中期景气）。

  • 偏弱：风电设备、部分游戏与高位题材的兑现回落。

当日催化与宏观

  • 周末大马磋商：通报称交流“建设性”，议题覆盖双方最关切的实际领域；为后续会晤铺路定调。

  • “十五五”规划：方向明确“高水平科技自立自强、新质生产力、绿色转型、区域协调与高水平开放”等主线，政策锚更清晰（相关周末发布会要点）。

  • 外部流动性：FED处于降息通道的再确认窗口，风险资产估值中枢支撑强化（以科技为代表的成长方向弹性更大）。

  • 国内基本面：三季报陆续落地，业绩分化推动“择优抱团”，市场交易回到基本面与产业化进度。

结构判断

  • 指数：短线靠近4000点的整数关口博弈在所难免，若量能维持在**≥2.2–2.4万亿**平台，回踩后仍偏向震荡上行。

  • 主线：科技/算力链与自主可控仍是“进攻中的确定性”，有色贵金属+部分工业金属受宏观与供需共振；大金融阶段性承担“稳盘器”，但节奏仍看事件与数据。

  • 操作：坚持“业绩+景气”两条线选股，避免在关口位情绪化追涨；回避基本面缺乏兑现的高位故事。

个人持仓与操作（仅作交易记录，不构成投资建议）

  • 中性/防守仓（20%）：

  • 早盘清仓某高端白酒（季报未出但不看好）；切换至某券商龙头（三季报已披露、估值合理、具防守属性），中性仓位10%；

  • 机动→中性切回：上周五建的“某PCB上游原材料龙头”（今涨停）已挂板止盈约+15%了结，接回此前的某产业互联网平台，占10%。

  • 进攻仓（60%）：

  • 两只各30%仓位的PCB龙头分别**+6.02% / +10.14%**（继续持有，关注这两天即将发布的三季报与量价匹配）。

  • 机动仓（20%）：

  • 今日主要用于锁定个股利润并腾挪到中性，机动仓位整体留作事件与数据窗口的弹性资金。

当前总仓位约80%：60%进攻 + 20%中性兼防守 + 20%机动（结构未变，中性个股做了优化）。

明日关注

  1. 量能与宽度：总成交是否稳在2.2万亿以上，涨跌比改善与否；

  2. 大金融是否接力，关注券商在论坛与三季报落地后的持续性；

  3. 科技主线内的切换：存储↔光模块↔PCB的轮动节奏，留意业绩/订单/指引类硬信息；

  4. 谈判后续细节，降息及APEC预期发酵；

  5. 三季报密集披露下的个股分化与“高位兑现→低位承接”的节奏。

“Process over prophecy; risk first, return second.”
