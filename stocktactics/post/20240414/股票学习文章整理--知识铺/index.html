五种经典K线形态

第一种：仙人指路

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

次日收阳可以大胆买入，指一根K线冲高回落的走势，K线形态上表现是有一根长上引，我们在平时也可以把它称之为试盘线，试盘线又分为主动试盘和被动试盘。

那么主动试盘和被盘如何去区分呢？如果说是主动盘，一般就是次日就会把上引线反包住，如果说是被动试盘，他调整的周期可能会比较长。

我们来看一下最近的标的：

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

在1这一天是出现了一根长上引线，我们发现是冲高回么，这根K线到是主力在出货还是力在盘呢，其实当时我们也不知道，因为他处于一个高位了，我们就要等待第二天的股价怎么走，如果说第二天他能够把1这一根上引线完全反包住，这里就可以看作是仙人指路，也可以看作是主力主动试盘去测试上方的抛压，以及下方的一个跟风，有时候他也是做到了一个洗盘的效果，我们看2号K马上做出一根反包阳线把1这个上引线给包住了，那么在2这一天我们就可以介入，不管说半路还是打板都是可以的。

第二种：双针探底，新低不创，大涨在即。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

股价步入了一段下跌趋势，一般下跌咱们是不抄底的，但是如果说已经下跌了很多，在底部位置出现连续的两根带有长下引线的K线，股价又不再创新低，就是右侧K线比左侧的要高，这种往往就意味着这个已经见底了，后面的大涨在即，很有可能会走V型反转的走势。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

股票双针探底是由两根K线组合而成，这两根K线都是较长的下影线K线状态，是一种常见的底部反转技术形态。 K线的实体部分可以是阴线、阳线、十字星。通常股票双针探底会出现于股票的下跌趋势中，表示股价在连续的下跌过程中，出现两次探底回升，下方支撑较强，形成有效的底部区间。

股票双针探底的技术形态可以是紧密相连的两根长下影线K线，也可以是有部分K线间隔的两条长下影线，但相隔天数不多。出现股票双针探底就表示空方力量两次探底没有成功引起下跌，被多方力量逐步吞噬，证明多方力量底部具有较强的支撑，而且多方力量处于强势阶段，表示股票形成有效强势的双针探底技术形态。

一般情况下，在股票出现双针探底技术图形时，如果股票价格出现立即快速反弹，走出强势的上涨行情，就可作为趋势转折买点参考。反之，在股票出现双针探底技术图时，如果股票价格没有出现强势反弹。就表示股票价格上升动力不足，还需观察股票价格后续的走势，但下方低点位依旧具有较强的支撑性。

第三种经典K形态：上升跳空阳，上涨有希望。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

股价经过拉升出现了多根小阳线，一般情况下就会有一个加速的上涨，一个跳空阳线直接把小平台给突破了，那么这种的话就是意味着后面看涨的概率会比较大，我们可以直接突破平台跳空高开买入，当然这个跳空高开的一个幅度不要太大，就是三个点以内是最好的，有时候如果说大幅高开，往往当天主力兑现的一个概率非常大，也就是说我们买进去时候往往会出现冲高回落。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

第四种经典K线形态：高位阴包阳，下跌在眼前。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

什么是高位阴包阳呢？就是股价连续上涨，然后在高位出现一根放量的大阴线，这根大阴线直接把前面的这根K线完全给反包了，那么这种就称之为高位的阴包阳，这种是一种急剧看跌的信号，后面股价可能会出现连续的大幅调整，如一旦出现这种放量的高位阴包阳K线形态，我们赶紧出掉小心要吃面。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

第五种经典K线形态：连涨出吊线，小心要吃面。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)

股价经过连续上涨，比如说出现连续涨停，连续大阳线，那么在高位出现一根吊线，吊线也可以称之为吊顶线，他有一个非常长的上引线，这种往往意味着股价要发生反转了，当然也有可能会出现仙人指路。

出现吊线之后，股价一旦出现跌破这个吊顶线的最低价，这种就意味着股价他这一波上涨趋势已经结束了。

![图片](http://zshipu.com/t/index.html?url=data:image/svg+xml,%3C%3Fxml version=‘1.0’ encoding=‘UTF-8’%3F%3E%3Csvg width=‘1px’ height=‘1px’ viewBox=‘0 0 1 1’ version=‘1.1’ xmlns=‘http://www.w3.org/2000/svg' xmlns:xlink=‘http://www.w3.org/1999/xlink'%3E%3Ctitle%3E%3C/title%3E%3Cg stroke=‘none’ stroke-width=‘1’ fill=‘none’ fill-rule=‘evenodd’ fill-opacity=‘0’%3E%3Cg transform=‘translate(-249.000000, -126.000000)’ fill=’%23FFFFFF’%3E%3Crect x=‘249’ y=‘126’ width=‘1’ height=‘1’%3E%3C/rect%3E%3C/g%3E%3C/g%3E%3C/svg%3E)
