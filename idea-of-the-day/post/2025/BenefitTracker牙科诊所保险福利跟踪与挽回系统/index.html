BenefitTracker：牙科诊所保险福利跟踪与挽回系统

它解决的是 诊所收入流失点 + 患者体验痛点 + 运营效率低 三重问题。

下面是完整的创意 App 功能点梳理（从 MVP → 成熟产品），已按“核心模块/功能点/价值”三层结构梳理。

🦷 一、核心产品定位

一句话定位 帮助牙科诊所自动化追踪患者保险福利剩余额度，防止过期损失，通过智能提醒和自动外呼挽回收入。

核心价值公式 收入提升 =（挽回预约 × 平均单次治疗费用 × 成交率）− 运营人力成本

🧩 二、系统架构概览

┌────────────────────────────┐
│ 患者数据 + 保险数据源       │ ← PMS集成（Dentrix、Eaglesoft、OpenDental） 
├────────────────────────────┤
│ BenefitTracker 数据引擎     │
│ • 实时保险额度同步          │
│ • 剩余额计算 / 即将到期检测  │
│ • 患者分群与优先级评分      │
├────────────────────────────┤
│ 外呼与提醒引擎              │
│ • 智能短信 / 邮件 / AI语音外呼 │
│ • 自动预约链接生成           │
│ • 外呼结果跟踪与CRM联动     │
├────────────────────────────┤
│ 前台与管理仪表盘            │
│ • 剩余福利可视化             │
│ • 任务与提醒队列             │
│ • 统计与绩效分析             │
└────────────────────────────┘

🪄 三、功能模块详细拆解

1️⃣ 数据集成模块（Integration Hub）

功能点 说明 价值
PMS集成 与主流实践管理系统（Dentrix, Eaglesoft, OpenDental）API或导出文件同步患者与保险数据 无需手动输入，提高准确率
保险公司数据同步 支持主要保险运营商（Delta Dental, MetLife, Cigna等），实时查询剩余福利 实时性保障
数据清洗与映射 自动清理不完整数据（出生日期缺失、ID错误），智能匹配 降低人工审核压力

2️⃣ 福利追踪与预测引擎

功能点 说明 价值
剩余福利计算 计算每位患者的免赔额、最大额度、已使用额度、剩余额 可见化保险状态
到期预警机制 识别30天/60天/90天内即将到期的福利 提前触发营销活动
治疗机会推荐 根据剩余额匹配可行治疗项目（如洗牙、补牙、正畸） 提升客单价

3️⃣ 智能外呼与营销自动化

功能点 说明 价值
短信/邮件自动提醒 向患者发送个性化提醒：“您还有$350保险未用，将于12月31日到期” 转化率高，减少流失
一键预约链接 每条提醒附带诊所预约页面链接 减少前台操作
智能分群 优先提醒高价值患者（剩余额度高/客户忠诚度高） 优化ROI
AI语音外呼（Pro版） 自动拨打电话 + 语音交互确认预约 减轻前台负担

4️⃣ 诊所运营仪表盘

功能点 说明 价值
患者福利看板 每位患者剩余福利图表化展示 一目了然
即将过期警报 红色高亮显示“90天内到期患者数” 聚焦关键任务
任务队列 自动生成“应联系患者名单” 精准行动
成效分析 显示“挽回预约数”、“新增收入”、“触达率”等 量化ROI
员工绩效追踪 每位前台的挽回收入贡献 激励机制

5️⃣ 报告与预测分析

功能点 说明 价值
挽回收入报告 展示本月通过BenefitTracker挽回的预约和金额 可量化收益
收入预测 基于历史使用趋势预测未来到期风险 提前营销
治疗推荐分析 分析患者最常放弃的治疗项目 优化服务策略

6️⃣ 扩展功能（第二阶段）

功能点 说明
自动召回系统 按周期自动提醒常规检查
患者忠诚度积分 每使用完福利一次积1分，鼓励定期回访
AI客服助手 回答保险使用相关问题
团体保险合作 面向企业团险方案，批量提醒员工使用福利
API市场 第三方CRM可接入该数据

💰 四、商业模式（Pricing Model）

收费方式 说明
SaaS订阅制 每诊所$250/月（基础功能）
预订成功提成 每完成一例挽回预约抽取$5-10
高级分析模块 $100/月额外功能（预测 + AI外呼）
联合推广分润 与牙科协会、咨询顾问、PMS厂商分销合作

📈 五、增长与获客策略

  1. 牙科行业会议 / 协会合作（ADA、DentalPost、DentalTown）
  2. PMS系统合作集成（成为官方插件）
  3. Facebook & LinkedIn 精准广告（目标：诊所经理 + 运营主管）
  4. 试点诊所案例营销（ROI展示）
  5. Referral Program（推荐一家返现$200）

🧠 六、产品演进路线（Roadmap）

阶段 核心目标 关键功能
MVP (0-6月) 验证价值 PMS集成 + 福利跟踪 + 邮件提醒 + 简版仪表盘
v1.0 (6-12月) 扩大客户 多保险支持 + 智能分群 + 收入分析
v2.0 (12-18月) 智能化 AI语音外呼 + 预测引擎 + API生态
v3.0 (18-24月) 行业标准化 全面集成至Dental PMS生态，成为行业SaaS标准
