教你用Lovart+Midjourney轻松批量输出分镜图！学AI设计上优设 --知识铺

2025年10月10日
| 阅读

嗨大家好！我是阿真！

响应群友催更，今天我们来看点轻轻松松的，Lovart第三弹，分享关于Midjourney在Lovart工作流与其他模型批量输出分镜图的一些方法给大家。

上链接：https://www.lovart.ai/zh

Lovart最近把Midjourney接上了，这下是真·图片生成全家桶一条龙服务了。

Midjourney单独拎出来吧，生成文字输出海报是不太行的，对提示词的理解是自由发挥天马行空的，唯独挡不住的就是硬好看，美学天花板，真的非常强。AI视频的素材图片就是它最有质感。但是在Lovart中和一大堆其他模型结合起来，就可以做很多不一样的尝试了。

我之前的一个观点很多朋友赞同，**Midjourney擅长生图，NB模型擅长基于原图拓展，Seedream4.0擅长做海报周边。**Gemini Imagen 4遵循提示词非常顶，Recraft真实感和插画效果好，Ideogram做Logo和英文海报牛逼。

然后现在Lovart把这些全放进去了，一个界面一条龙完成，视频的模型也是。后悔，我Recraft和Ideogram的年费会员感觉白订阅了。

等有时间单把海报一条龙拉出来讲讲，今天主要分享分镜图生成。接下来和我一起来尝试一下吧！

往期教程：

Midjourney的提示词，首先注意提示词少废话，简洁清晰表达内容。公式可以参考官方这个：

[主体] + [环境/背景] + [动作/场景] + [风格/材质] + [光影/氛围] + [构图/镜头] + [可选参数]

首先，我们可以直接给到多组提示词，每组会有4张图，这里我直接给了8组提示词，测试后发现也可以（我是为了测试所以一次给了多组提示词，具体数量大家看自己需要）。这里比例要求可以写参数–ar x:x，也可以直接中文写出比例。记得选择指定模型

然后直接生成了8组，我从中选出自己最想要的场景图。

Midjourney生成原图（部分）：

后期使用的时候还可以点击上方Upscale提升画质，最高4倍。

这里稍微扩展一下，V7的原生模型除了真实感和质感，在美学上锦上添花的，还有它极为大量的–sref code和个人风格–p。这些在小红书上搜索可以搜索到非常多的风格代码。也有专门的sref code网站，等有时间我给大家整理一些。

收集的代码加在提示词最后，哪怕相同的提示词，加了不同的sref code之后也会有不同的质感。比如下面这些图片，提示词都是以花束为主体，大家可以看到添加sref code后不同的图片风格：

--sref 704078150

--sref 2123637092

--sref 3171847554

Sref code可以在Lovart对话框进行要求，类似于「后缀参数全部使用–sref 3171847554」，在整个对话框上下文中只要调用Midjourney，它都会使用。

在上一步中还只有选到图片，还没有具体到场景细节，我们可以选中图片作为参考图，发送到对话，选择NB模型或者Seedream 4.0模型进行下一步分镜图生成。也就是基于一张图生成多张，还能保持一致性。

我这里使用的是Seedream 4.0模型。

这里有一组提示词参考，从一张图延展12张故事序列图，作为参考。也就是发送图片到对话（右键图片add to chat）并发送下面这组提示词（提示词为AI优化，供参考）：

请分析这张图片的视觉风格、色彩、角色动作和叙事氛围。

基于这张图，创建完整的12镜分镜故事序列图片：

故事结构：
- 镜头01-03：开场建立（环境/角色出场/氛围铺垫）
- 镜头04-06：事件发展（动作开始/情节推进）
- 镜头07-09：冲突高潮（转折/关键时刻/情绪顶点）
- 镜头10-12：结局收尾（结果/情绪释放/余韵）

要求：
1. 保持原图的视觉风格、色彩基调和美术风格
2. 镜头之间有逻辑连贯性和时间顺序
3. 包含不同镜头类型（远景/全景/中景/特写/运动镜头）
4. 每个镜头包含：场景描述、镜头类型、视觉重点

输出格式：表格图，包含"镜号 | 场景描述 | 镜头类型 | 视觉重点"四列

这样就可以快速通过一张图获得多张分镜的图片啦。不过这个有限制，因为镜头和提示词都是Lovart写的，所以可能不是每个都都是我们想要的，需要筛选自己要的图。需要更好的图片效果，建议是每个场景都有自己的想法要求，自己输出场景的提示词。

场景图：

另外还生成了一张分镜故事序列图：

在上一步中我基于图片获得了很多不同的分镜，某一张或者几张分镜图，我想要这个场景的不同角度的镜头视角，也可以很快实现。

比如可以要求输出同一场景不同摄影机位（提示词为AI优化，供参考）

请分析这张照片的场景、环境、光线、氛围和视觉风格。

基于这个场景，生成8个不同角度的镜头视角的图片。

要求：
1.保持相同的场景、环境、光线和氛围
2.保持相同的视觉风格和色调
3.只改变拍摄角度和距离
4.场景中的物体/人物/元素保持一致

输出为专业分镜表格

输出图片

也可以基于现有动作推断接下来的动作（提示词为AI优化，供参考）：

基于这个动作瞬间，推断并生成接下来8个镜头的分镜图，比例为16:9。

分析当前状态：
- 角色的身体姿态和运动方向
- 物体的位置和运动轨迹
- 环境的互动可能性
- 动作的力量和速度

推断规则：
1. 遵循物理规律
2. 符合角色性格和情绪
3. 包含动作的"预备-执行-结果"
4. 可以有场景切换和时间流逝

镜头设计：
- 不同角度捕捉动作
- 穿插反应镜头和细节特写
- 营造动作的节奏和张力

输出专业分镜表格

输出图片（部分）：

也可以让AI基于已有场景推断接下来的会发生的场景，输出连续镜头分镜图片。像这样的镜头要求，同样可以基于不同需要，参考我前面的提示词格式，输出比如电影感叙事方向、同场景后续动作方向、不同人物情感和氛围、广告片连续单帧等方向的要求。

从上述案例不难发现,在Lovart基于单一场景中整合多个需求不仅可行,而且具有极大的发挥空间。那么接下来我再简单举例。

开局一张图。

那么接下来来玩一个简单的开局一张图延展一系列。

现在我只有一张这样的图片：

首先将口红替换为自己的产品图，这里选中2张图tab一下：

接下来我想使用NB模型延展系列分镜图，但是NB模型的一个特性是给什么比例输出什么比例，那么我们修改比例，选择图片扩展到16:9：

扩图完成，但背景太简单了，可以让它重新生成更适合这个氛围的背景，或者给到背景图供参考。

选择上一步的图片做参考，用来做多个静态分镜图，比如我让它生成12张16:9比例的图（提示词为AI优化，供参考）：

请分析这张广告图片的：
- 产品/品牌特征
- 视觉风格和色彩
- 目标受众和调性
- 核心卖点

基于这张广告图，生成12张静态分镜图，比例为16:9

分镜结构：
镜头01-02：开场吸引 - 建立场景，抓住注意力
镜头03-05：产品展示 - 多角度展示产品特点
镜头06-08：使用场景 - 展示应用情境和体验
镜头09-10：情感共鸣 - 人物反应，情绪高潮
镜头11：产品特写 - 强化记忆点
镜头12：品牌收尾

每张分镜图要求：
1. 保持原图的视觉风格、色彩和品牌调性
2. 清晰的构图，适合静帧截取
3. 标注镜头类型（特写/中景/全景等）
4. 考虑画面的完整性和美感

注意：每张都是独立完整的静态画面，不是动态过程的某一帧。

细节不好可以选择图片点击Tab召唤NB模型修改，其他图片有细节问题要修改也同样：

还可以参考我前面提到的基于一张图延展更多基础分镜图的不同角度，选择最合适的图片。

如果觉得场景和人物不错，需要更多构图也可以：

参考这张图片，生成8张不同角度的图片，人物的姿态也可以进一步变化

如果觉得只有人物比较单调，那么还可以基于单产品生成更多产品的展示效果图。（提示词为AI优化，供参考）：

请分析这张产品图的：
- 产品外观、材质、颜色、细节
- 产品类型和特点
- 当前拍摄角度和光线

基于这个产品，生成8组不同构图的展示效果图，比例为16:9

构图方案：
1. 极致特写 (Extreme Close-up) - 聚焦产品细节、材质、纹理、工艺
2. 产品特写 (Close-up) - 展示产品主要特征，突出设计亮点
3. 产品中景 (Medium Shot) - 产品完整展示，适度环境衬托
4. 产品全景 (Full Shot) - 产品与环境关系，场景感
5. 俯视角度 (Top-down View) - 平铺展示产品
6. 多产品排列 (Multiple Products) - 2-5个产品重复排列，阵列美感
7. 创意角度 (Creative Angle) - 45度斜角/仰视/特殊视角

每组要求：
✓ 产品外观、颜色、材质保持一致
✓ 不同的拍摄距离、角度、构图方式
✓ 适当的光影和背景配合
✓ 符合商业摄影美学标准
✓ 突出产品不同的视觉优势
