OPENAI开发者大会，放出三大王牌！ --知识铺

2025年10月8日
| 阅读

北京时间2025年10月7日凌晨，旧金山福森要塞会场，OpenAI的DevDay开发者大会再度成为全球焦点2。

OpenAI首席执行官Sam Altman面向1500名开发者宣布，公司已从单纯的AI模型提供商，转型为AI应用生态平台。

过去，一个绝妙的想法从诞生到抵达用户，需要跨越需求分析、架构设计、代码编写和市场推广的漫漫长路。无数创意因资源匮乏或工程复杂而胎死腹中。

然而，在OpenAI今年的DevDay之后，这一格局已被彻底打破。我们不再活在“编写代码”的世界，而是进入了“表达意图”的时代。

ChatGPT：从聊天工具到超级生态

OpenAI不再满足让ChatGPT仅仅做个“聊天机器人”，而是要将其打造为连接亿万用户的超级平台。

最新推出的Apps SDK，允许开发者将完整应用直接嵌入ChatGPT环境。这意味着：

  • 当用户讨论学习新技能时，Coursera应用可被直接推荐并内嵌到对话中，用户甚至能边看视频边问“刚才讲的概念是什么意思？”-1

  • 当用户头脑风暴创业想法时，可直接呼唤Canva，让它根据对话上下文一键生成海报或演示文稿-4

(Apps SDK（Application Software Development Kit）就是帮开发者快速造手机或电脑应用的“工具箱”。里面装着现成代码、教程和测试工具，让开发者不用从零写代码，就能轻松搞定登录、支付、AI功能等模块。简单说：省时省力的“App 建造砖头”）

这种集成远非简单的插件可比：

  • 上下文感知：应用能获取完整对话语境，无需用户手动传递信息

  • 沉浸式交互：支持全屏或画中画模式，提供完整应用体验

  • 商业闭环：未来将通过Agentic Commerce（代购智能体）协议实现即时结账，一句话完成从需求到交易的全过程

Agentic Commerce 就是“AI代理自动购物”。 它让你的AI助手不仅能帮你推荐商品，还能直接、自主地帮你完成购买。

传统电商购物：

  1. 你想买咖啡机 -> 2. 打开淘宝/京东 -> 3. 自己搜索、比价、看评测 -> 4. 加入购物车 -> 5. 填写地址、选择支付方式 -> 6. 输入密码完成支付。
    （整个过程都需要你亲力亲为）

Agentic Commerce 模式：

  1. 你直接对你的AI助手（比如ChatGPT）说：“我的咖啡机坏了，帮我找一款适合小家庭、预算在500元左右、口碑好的新咖啡机，并买下来。”

  2. AI助手就会自动执行：

  • 搜索并对比符合你要求的咖啡机。

  • 直接调用你预先授权和绑定好的支付方式（如信用卡、数字货币包）。

  • 无需你再跳转或输入密码，它直接在对话界面里就替你完成支付。

  1. 你只需要最后确认一下订单即可。

简单说，就是从“帮你选”升级到了“帮你买”。

你的服务，现在可以以最自然的方式部署到数亿用户的指尖。Agent Kit：复杂逻辑，简单搭建

构建可靠的AI智能体曾是个噩梦——复杂的流程编排、难以衡量的评估、与企业系统的连接，让多数项目早早夭折。

Agent Kit的出现，为开发者提供了一套端到端的解决方案：

  • 可视化工作流设计：通过拖拽节点即可设计复杂逻辑流程，如同绘制流程图般直观-4

  • 内置安全护栏：预设PII拦截等安全机制，确保智能体在敏感企业环境中安全运行

  • 全面评估系统：提供追踪评分和自动化提示优化，让智能体不断自我改进

零售巨头Albertsons已经利用此技术部署智能体：当冰淇淋销量意外下降32%时，智能体能立即分析季节性、历史趋势和外部因素，迅速给出调整建议。

这意味着，每一家企业现在都能将内部知识与行动工具结合，形成智能自动化闭环。

Codex：从写代码到表达意图

如果说前两者是平台和流程的升级，那么Codex的进化则彻底重塑了人与代码的关系。

现已正式商用的GPT-5 Codex，不再是简单的代码补全工具，而是真正的编程伙伴：

  • 全景上下文理解：Codex连接你的聊天记录、IDE、终端和GitHub，理解你的隐性意图，减少冗余提示

  • 自主问题解决：当遇到知识盲区时，Codex会自行搜索文档，找到所需信息，并编排成可执行方案

  • 实时代码修改：通过语音指令即可让Codex修改、编译并运行代码，实现真正的“意图编程”

在现场演示中，开发者只需说“将Xbox手柄连接到摄像机”，Codex便自主完成了一切：从理解Visca协议到生成完整的Node服务器和接口代码。

推特上，已经不少人开始使用codex来编程，怎么说呢，不是万能的，但是，相对其他几个来说，性价比还是最高的。

代码不再是创意的屏障，而是可被意图随时调用的资源。

随着GPT-5 Pro提供更强的推理能力，**Realtime Mini（OPENAI下面一个实时语音模型）**让语音交互更加自然，Sora 2的API实现音画同步的视频生成，技术壁垒正被迅速消除。

美泰的设计师已能通过文字将草图直接变为可分享、可迭代的动态产品概念。

在这个新时代，衡量开发者的标准不再是他能写多少行代码，而是他能产生多大、多复杂的意图。

当工具无限强大时，你对世界的思考深度，就是你作品的最终高度。唯一的障碍，只剩下想象力本身。

现在AI模型到了，大家拼命迭代，拼命发布新模型的时代，国庆假期前阿里一口气推出了7个模型。

kimi也有新模型推出。

此前M开头的智能体企业，再不更新，扭扭捏捏恐怕就明日黄花了。

OPENAI的这个开发者大会，会不会让ai应用企业成为风口呢？

拭目以待吧，反正明天港股开盘，大后天A股也要开盘了。
