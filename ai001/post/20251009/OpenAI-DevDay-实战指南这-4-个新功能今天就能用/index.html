OpenAI DevDay 实战指南：这 4 个新功能，今天就能用！ --知识铺

2025年10月8日
| 阅读

昨天，我给小可爱们介绍了 OpenAI DevDay 2025 的直播发布会。

那么今天问题来了，这些更新是 OpenAI 画的“饼”吗？

还是现在就能用？

好消息：不是“饼”。

废话不多说，直接干货走起。

01｜在 ChatGPT 里直接用 App：Apps in ChatGPT

对于个人用户来说，这可能是本次发布最实用的新功能。

你可以在 ChatGPT 里直接和这些已经集成好的 App 对话了。

并且，立马就能用，不需要等待灰度推送。

实测 ChatGPT 免费用户就能用。

Go、Plus、Pro 这些付费用户更没问题。

唯一遗憾的是，ChatGPT Team/企业用户暂时还用不了。

如何判断你的 ChatGPT 账号能不能用 Apps in ChatGPT？

很简单，点击左下角头像，菜单栏里找到 设置 Settings，如果你能看到下面这个 Apps & Connectors，那就成了。

可以看到，目前已经集成进 ChatGPT 的 App 包括：Booking.com、Canva、Coursera、Expedia、Figma、Spotify 和 Zillow。

对国内的小伙伴来说，Canva、Coursera 和 Figma 相对来说可能更实用。

使用时，直接“输入 App 名字 + 空格”就能唤起这个应用，首次使用会让你绑定自己的账号进行授权。

然后，你就能以对话的形式让 ChatGPT 调用 Canva 的接口画海报了。

或者是在 ChatGPT 里上 Coursera 的网课，还能边学边问。

或是用 Figma 画图。

如果你碰巧是 Spotify 用户，那用这个功能来创建歌单是再好不过了。

02｜用 Agent Builder 拖拽出 AI 智能体

CEO Sam Altman 这样描述 Agent Builder：“像 Canva 一样构建智能体。”

简单来说，它就是一个可视化画布，用于创建多智能体工作流。

拖拽式，零代码，轻松上手。

Agent Builder 目前也已经发布。

附上体验地址。

https://platform.openai.com/agent-builder

实际上，Agent Builder 只是 AgentKit 工具包的一部分，这个工具包里还有：

  • ChatKit：预制的聊天界面，一行代码就能嵌入

  • Evals：AI Agent 测试和优化工具

  • Connector Registry：第三方 MCP 连接器管理

好了，厨房已搭好，就等着你炒菜了。

题外话，如果你之前用过字节 Coze/扣子、Dify、n8n 或是 Zapier，那 OpenAI 的 AgentKit 应该能快速上手，毕竟内在逻辑都是一样的。

03｜Codex 发布正式版了

之前的 Codex 都是研究预览版本。

这次，正式版来了。

和 Claude Code 一样，只要你是 ChatGPT 付费用户就能用。

而且，支持的客户端也不少。

说说大家比较关心的问题：使用额度。

不同的 ChatGPT 订阅层级对应着不同的 Codex 额度，下面总结一波。

  • Plus：20 美元/月，每 5 小时 30-150 条本地消息或 5-40 个云端任务

  • Pro：200 美元/月，每 5 小时 300-1500 条本地消息或 50-400 个云端任务

  • Team/Business/Enterprise：基础额度同 Plus，支持购买额外积分扩容

同样和 Claude Code 一样，Codex 也同时加入了每周限额，使用时需要多加注意。

04｜GPT-5 Pro API，值吗

一起发布的，还有 OpenAI 最强模型 GPT-5 Pro 的 API。

价格是真高：每百万输入 tokens 15 美元，输出 120 美元。

但能力也是真强。

所以，值不值，看你怎么用。

如果你碰巧要做一个很难的项目，或者解一道竞赛题，而你又碰巧不想开通 ChatGPT Pro/Team 会员，那么可以考虑 GPT-5 Pro API。

否则还是算了，它可能会是你的 token “刺客”。

结语

OpenAI 这次确实没画饼，新功能基本都能直接用了。

Apps in ChatGPT 最实用，值得一试。

Agent Builder 适合想做点小工具的个人开发者/企业。

Codex 对程序员来说是生产力工具。

GPT-5 Pro 和 Sora 2 API，按需选择。

工具都在这，怎么用，看你了。
