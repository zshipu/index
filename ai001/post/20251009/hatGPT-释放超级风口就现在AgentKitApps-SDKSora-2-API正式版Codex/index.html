hatGPT 释放超级风口！就现在：AgentKit、Apps SDK、Sora 2 API、正式版Codex --知识铺

2025年10月8日
| 阅读

8 亿用户、400 万开发者、每分钟 60 亿 tokens 的处理能力，OpenAI 面向普通人的 AI 红利已经爆发。

为避免错过一手 AI 资讯、产生信息差，您可关注本号并点亮星标，重要更新将第一时间送达。

机会来了！而且是实打实的！

OpenAI 在 2025 年 DevDay 上一口气发布了 Apps SDK、AgentKit、Codex GA 和 Sora 2 API，直接把 ChatGPT 变成了超级平台：

  • 像 App Store 一样分发应用

  • 像 Canva 一样搭建智能体

  • 像 Figma 一样部署聊天前端

  • 像 Notion AI 一样写代码、生成视频

为了更好的抢占风口机遇，我们一起来学习本次的重磅更新有哪些

一、AgentKit 「人人都能搭建智能体」

用拖拽方式就能构建完整 Agent 流程，从创建到部署只需几分钟，杀死了 dify、fastgpt 等这类创业公司。

核心组件包括：

Agent Builder：拖拽式流程画布

Connector Registry：统一管理数据与 API

ChatKit：将聊天嵌入你的网站或应用

二、Apps SDK 「应用可以“住进” ChatGPT」

你可以在 ChatGPT 中一句话唤起 Canva、Spotify、Zillow 等应用，它们将直接以交互界面形式出现，无需下载、跳转。

开发者可通过 Apps SDK 构建原生应用，未来还将开放应用目录与变现通道。

真实演示：OpenAI 员工在聊天框中一句话生成遛狗公司海报，下一句要求生成 PPT，一气呵成。

三、Codex GA「用自然语言写代码」

正式版 Codex 支持 Slack 集成、Codex SDK 和管理后台。

OpenAI 称团队使用 Codex 后，PR 数提升 70%，几乎所有工程师都依赖它自动审查代码。

真实案例：现场演示用语音让 Codex 写完控制摄像头的服务器代码，还能用 Xbox 手柄控制镜头。

四、Sora 2 API「文本生成视频走入产品化」

最新的视频生成模型开放 API，可控性极强：长宽比、时长、分辨率、音效都能精准控制。

Sora 不再是“玩具”，而是面向电商、广告、影视、概念展示的生产力工具。

应用想象：输入产品描述+图片，即可生成带背景音的开箱视频，30 秒做完概念广告。

这是我的落地建议，不同角色该怎么抓住这波风口

一、创作者 / 自媒体

  • 用 Canva + ChatGPT 做封面图 + 海报 + PPT，一条龙生成

  • 将常见提问做成小型 Agent 嵌入你的公众号 / 小程序

二、产品经理 / 初创团队

  • 构建一个垂类 Agent，替代客服、顾问或教练角色

  • 把现有 SaaS 工具接入 Apps SDK，率先占位“聊天入口”

三、工程团队

  • 用 Codex 接 Slack + Git + CI 流程，周迭代速度翻倍

  • 使用 Sora 2 API 做产品 DEMO、动态设计稿、展示视频
