Codex CLI 0.44.0 发布！本地云端一体化 --知识铺

2025年10月8日
| 阅读

Codex CLI 0.44.0 版本已发布！

0.44.0应用更加简洁高效，没有Claude Code现在花里胡哨的开机界面，瞬间加载对话，打开就是界面用户消息，没有任何多余的元素。

这次的交互操作逻辑和脸面也全换了，更新了聊天编辑器、底部面板、模型选择器和升级提示界面，整体更加的简洁，但是操作习惯还是和Claude Code有些许的区别，不过自适应一下就应该还可以接受。

增加了对streamable HTTP MCP servers的支持。

Codex的MCP配置一直是一个很难的问题，可以使用我写的这个图形化配置工具，轻松实现原来的Claude Code MCP配置转换。

地址：https://toml.bytenote.net/json-to-toml

Codex支持自定义提示了。

自定义提示的详情配置和使用说明：

 https://link.bytenote.net/r/V0BLTY

现在自定义提示中的命名参数和位置参数，现在可以创建带参数的自定义提示模板，就像函数一样，比如下面的位置参数，数字参数：

/prompts:review $1 $2

使用时：

/prompts:review myfile.py "check for bugs"

$1 会被替换为 myfile.py
$2 会被替换为 check for bugs

本次最大的更新就是这个实验性功能，codex cloud，现在不需要来返于命令行和网页，在本地命令行就可以实现本地和云端和远一体化操作。

基本的操作按钮和命令如下

^/ (Ctrl+/) - Move（移动）
r - Refresh（刷新）
Enter（回车键） - Open（打开）
a - Apply（应用）- 目前处于禁用状态
o - Set Env（设置环境变量）
n - New Task（创建新任务）
q - Quit（退出）

目前，云端操作对用户提供了几乎无限量的使用，可以进行异步的多任务处理。
