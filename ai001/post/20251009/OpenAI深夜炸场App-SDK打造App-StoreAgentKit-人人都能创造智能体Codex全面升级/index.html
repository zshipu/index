OpenAI深夜炸场：App SDK打造App Store、AgentKit 人人都能创造智能体、Codex全面升级！ --知识铺

2025年10月8日
| 阅读

今天凌晨 OpenAI 的开发者日发布了一系列硬核更新。简单来说，OpenAI 不只想做那个最聪明的“大脑”，它还想打造一个属于 AI 的“苹果 App Store”和 “智能体（Agent）制造工厂”。

这次更新主要有三件大事：App SDK、AgentKit 和 Codex 的全面升级。每一个都可能改变我们使用 AI 和开发软件的方式。

下面，我们用最简单的方式，聊聊这次更新到底有多炸裂。

ChatGPT 的“App Store 时刻”来了

首先，最让普通用户兴奋的是，ChatGPT 终于有了自己的“App Store”。

还记得 iPhone 刚出来的时候吗？最开始大家觉得它就是个漂亮的手机。但真正改变世界的是 App Store 的出现。无数开发者在上面创造了各种 App，让 iPhone 变得无所不能。

现在，OpenAI 正在为 ChatGPT 做同样的事情。

通过这次发布的 Apps SDK，开发者可以为 ChatGPT 开发各种各样的“小程序”。这些 App 不再是简单的插件，而是能和你在聊天中深度互动、带有交互界面的应用。

举个例子。

以前，你想用 ChatGPT 规划旅行，它只能给你一堆文字建议。现在，你可以直接在聊天框里说：“帮我找找下周末在上海的酒店”，ChatGPT 就会自动调用 Booking.com 的 App，直接在对话里给你展示可以预订的酒店列表，有图有价，就像下面这样。

你可以用 Canva 的 App，把你的想法直接变成精美的 PPT。

可以用 Spotify 的 App，让它为你创建一个派对歌单。可以用 Zillow 的 App，在对话中浏览符合你预算的房子。

这就像给 ChatGPT 安装了无数个“外挂”，能力瞬间指数级增长。

对于用户来说，ChatGPT 从一个无所不知的“聊天机器人”，变成了一个无所不能的“智能操作系统”。对于开发者来说，一个拥有 8 亿用户的全新平台诞生了，这里充满了无限的机会。

根据 OpenAI 的描述，免费用户也可以使用哦！

人人都能造 Agent

如果说 App SDK 是给普通用户的大餐，那 AgentKit 就是给开发者和企业准备的“核武器”。

AgentKit 是一个用来构建、部署和优化 AI 智能体（Agent）的完整工具包。参见此前发布 OpenAI 即将发布 Agent Builder：拖拽即可定制 AI 智能

什么是“智能体”？你可以把它理解成一个能独立思考、使用工具、并完成复杂任务的“AI 员工”。比如，一个能自动帮你分析财报的金融智能体，或者一个能 7x24 小时处理客户问题的客服智能体。

以前，开发一个这样的智能体非常复杂，需要写大量的代码，处理各种繁琐的流程。现在，OpenAI 把这个过程变得像搭积木一样简单。

AgentKit 里有一个叫 Agent Builder 的工具，它提供了一个可视化的界面。开发者只需要通过拖拽，就能设计出一个完整的工作流，把不同的 AI 智能体连接起来，设定规则，就像在画流程图。

Ramp 公司用它在几个小时内就搭建了一个采购员智能体，而之前这需要耗费好几个月。

同时，AgentKit 还提供了 ChatKit 工具，让开发者可以轻松地把这些智能体聊天界面嵌入到自己的网站或 App 中，外观还可以定制，完美融入自己的产品。

简单说，OpenAI 不仅提供了最聪明的大脑，现在连“骨骼”、“四肢”和“生产线”也一起打包给你了。一个全民创造 AI 智能体的时代，可能真的要来了。

Codex 全面开放，更强更易用

聊完了应用和智能体，我们再来看看底层的开发工具——Codex。

这次，Codex 也迎来了重磅升级，并且宣布“全面可用”（Generally Available）。这意味着它已经非常成熟，所有开发者都可以大规模使用它了。

这次 Codex 的更新主要有三点：

  1. 1. 更强的“大脑”：新的 Codex 由更强大的 GPT-5-Codex 模型驱动，代码生成和理解能力都上了一个新台阶。

  2. 2. 深入你的工作流：推出了 Codex SDK（目前支持 TypeScript），让开发者可以把 Codex 的能力集成到任何自己的工具和流程中。同时，还支持了 Slack 集成，你可以在公司群里直接 @Codex，让它帮你干活，就像你的同事一样。

    import { Codex } from "@openai/codex-sdk";

const agent = new Codex({});
const thread = await agent.startThread();

const result = await thread.run(“Explore this repo”);
console.log(result);

//resume thread
const result2 = await thread.run(“Propose changes”)
console.log(result2); ```

  1. 3. 更完善的管理工具：为企业管理员提供了新的后台，可以更好地管理和监控 Codex 的使用情况，保证安全和效率。

根据 OpenAI 的数据，公司内部工程师使用 Codex 后，每周合并的 PR（代码提交）数量提升了 70%。像思科（Cisco）这样的巨头，用 Codex 把代码审查的时间缩短了 50%。

Codex 的全面开放，正在进一步降低编程的门槛，让开发软件变得前所未有的高效。

底层动力：联手 AMD 增强算力

发布了这么多强大的软件工具，底层也得跟上。

就在同一天，OpenAI 还宣布了一个重磅消息：与芯片巨头 AMD 达成战略合作。OpenAI 将采购高达 6 千兆瓦的 AMD GPU。这是什么概念？这是一个天文数字的算力，足以支撑未来更强大、更复杂的 AI 模型的训练和运行。

如果说 App SDK、AgentKit 和 Codex 是 OpenAI 展示给世界的“肌肉”，那么和 AMD 的合作，就是为这些肌肉提供源源不断能量的强大“心脏”。

这表明，OpenAI 的野心远不止于模型本身，它正在下一盘大棋，从最底层的硬件基础设施，到中间的开发者工具，再到最上层的应用生态，它全都要。

总结

从为 ChatGPT 打造应用商店，到提供智能体生产线，再到升级核心代码引擎，并增强巨量算力，OpenAI 正在一步步地构建自己的帝国。

对于普通人来说，AI 将会像手机 App 一样，以更具体、更实用的形态融入我们的生活。对于开发者和创业者来说，一个新的时代机遇已经到来。
