ChatGPT 成为“应用平台”：什么是 OpenAI 的“Apps SDK” --知识铺

2025年10月9日
| 阅读

应用SDK的功能

过去，要使用该应用程序，您必须打开智能手机或浏览器。 但是，通过使用 Apps SDK，您只需在 ChatGPT 上说“在 Canva 中创建幻灯片”、“使用 Zillow 查找匹兹堡的房产”或“在 Spotify 上播放这首歌”，AI 就会在内部启动每个应用程序并返回结果。

奥特曼说：

“这将创建交互式、自适应性和个性化的新一代应用程序。”

OpenAI的愿景很明确：将ChatGPT从“会说话的AI”演进为“作系统”。

什么是应用程序 SDK：ChatGPT 如何“与应用程序交互”

Apps SDK 是一个软件开发工具包，用于在 ChatGPT 中运行外部应用程序。
开发人员可以使用它将他们的应用程序集成到 ChatGPT 中，允许用户通过与人工智能的对话与应用程序进行交互。

此外，OpenAI 还推出了自己的“代理商务协议（ACP）”。 这也启用了应用内购买机制，在 ChatGPT 中收费提供应用程序并收取使用费。 换句话说，ChatGPT 正在转变为一个应用商店和执行环境。

技术基础：采用Anthropic倡导的MCP标准

Apps SDK 基于 Anthropic 一年前提出的开源标准 Model ContextProtocol （MCP）。 MCP 是 AI 与应用程序安全高效地通信的通用规范，允许它们调用应用程序数据或执行特定作。

在 Apps SDK 中，ChatGPT（GPT-5 和 O 系列模型）从外部应用程序获取最新数据，以确保您始终了解用户在做什么。 开发者可以按以下格式显示他们的应用程序：

  • 在线显示（卡片/轮播格式）
  • 全屏视图（例如地图和幻灯片）
  • 画中画（例如视频和现场学习）

这使您可以保持 ChatGPT 的简单 UI，同时仍提供每个应用程序的品牌体验和高级交互。

主要合作伙伴和演示示例

在演示会上，已经展示了多个应用程序链接。 典型例子包括：

  • Coursera：登录 ChatGPT 并观看讲座视频。 AI实时总结和解释讲座内容。
  • Canva：ChatGPT 根据文本指令自动编辑和生成 Canva 内模板（例如，“制作色彩鲜艳的海报”）。
  • Zillow：内联运行房地产搜索。 直接在 ChatGPT 中在地图上显示房产、价格和平面图。

此外，Booking.com、Expedia、Figma 和 Spotify 等主要参与者似乎也将加入董事会。 该文档还确认了 AllTrails、Peloton、OpenTable、Target 和 Uber 等名称。

这样，教育、旅游、电子商务、设计、音乐等各个领域的应用都可以在ChatGPT中运行。

应用 SDK 的实际应用：对话转化为应用互动

Apps SDK 的吸引力在于它们能够将传统的对话式 AI 转变为作 AI。 例如，您可以通过以下方式使用它：

  • “我在 Canva
    中制作了一张带有’Walk This Wag’标志的传单→ChatGPT 名为 Canva，并生成了自动调整字体和配色方案的海报提案。
  • “在 Coursera 上向我展示机器学习入门课程”
    →在 ChatGPT 中播放讲座视频时，如果你问“解释你正在谈论的算法”，你会立即总结。
  • “使用 Zillow 查找匹兹堡的宠物友好型房产”
    → 带有地图的搜索结果是在 ChatGPT 上开发的，如果您问“附近有狗公园吗？”，它会分析地图信息并回答。

所有这些都在 ChatGPT 聊天屏幕中完成。 无需切换应用程序，人工智能就会“接管”作。

安全和开发者指南

OpenAI为这个新的“应用生态系统”制定了严格的运营标准。 您的应用必须：

  • 有一个明确且有用的目的
  • 可预测且稳定的行为
  • 对 13 岁以上的公众安全
  • 以最少的数据采集尊重隐私

所有应用都必须有隐私政策，并且在连接时必须征得用户同意。 此外，在对外部数据进行更改（发布、发送等）时，需要进行明确的用户验证。

OpenAI 在视觉效果方面也设定了统一标准。 它遵循 ChatGPT 的配色方案、排版、间距和图标系统，同时保持对话式、直观且易于访问的用户界面。 允许使用品牌徽标和颜色，但禁止使用促销表达。

对话式作系统计划向开发人员和用户开放

随着Apps SDK的推出，ChatGPT即将从“聊天机器人”转变为“AI OS”。 在对话界面中，集成了商业、学习、设计、旅行等应用程序，使对话成为一种作手段。

对于开发人员来说，它提供了对超过 8 亿 ChatGPT 用户的直接访问，为应用程序发现和货币化创造了新的机会。 另一方面，对于用户来说，“只需说话即可使用的应用程序”的全新体验扩展了。

“这是ChatGPT应用的开始，
为用户带来新的价值，也为开发者带来新的机会。”
— 摘自OpenAI官方评论

未来挑战：数据共享与“GPT Store”的去向

然而，一些疑虑仍然存在。 首先，当第三方应用程序和 ChatGPT 交换数据时，这些信息是否会与 OpenAI 共享并用于学习？

其次，目前尚不清楚之前强调的“GPT Store”（一种允许您销售自定义 GPT 的机制）将如何与这个新的“Apps SDK”集成。

应用 SDK：摘要

Apps SDK的发布，可以说是从生成式AI“在应用中移动”时代到“成为应用本身”时代的转折点。 ChatGPT 演变成为所有应用程序提供支持的对话作系统（而不仅仅是对话工具）的未来终于成为现实。
