看完OpenAI年度发布会，突然感觉世界就是个巨大的豆包 --知识铺

2025年10月8日
| 阅读

没错，昨晚 OpenAI 年度开发者大会，又出了一堆东西。

包括：
◦ Apps SDK：GPT 现在可以直连 APP 了
◦ AgentKit：类似于扣子或 Dify 的 Agent Workflow
◦ Codex 正式版：类似于 Claude Code 的 CLI Agent
◦ 3 个新 API：GPT-5 Pro、Sora、Real-Time Mini 都上 API 了

群里都在热议的是，OpenAI 也开始下场抄扣子和 Dify 了。

豆包：奥特曼，你是 AI 吧，哈哈哈哈。

1、Apps SDK

现在直接可以在 GPT 里面丝滑的使用第三方应用了，而不是单纯的使用 API 和插件，可以返回应用界面，比如可以直接在 GPT 里播放 Coursera 的课程视频，一边看一边问问题。

开发者可以把基于 MCP 协议构建的 SDK 集成到 GPT，就可以在 GPT 中使用该 APP。

OpenAI 想把 GPT 打造成应用超级入口，首批合作伙伴包括 Figma、Canva、Spotify、Zillow 等，今天就能用。

2、Agent Kit

可以说这是 OpenAI 版的扣子全家桶

这是一个构建、部署和优化的 Agent Workflow，原生 GPT-4/5/o3 集成以及一些预建模板。

包含以下四个部分：

ChatKit：可嵌入、可自定义的聊天用户界面

Agent Builder：所见即所得的工作流创建，和扣子一样，拖拉拽工作流画布；

Guardrails：输入/输出的安全检查；

Evals: Datasets：数据集、轨迹分级、自动提示优化以及第三方模型支持

3、Codex 正式版

这玩意儿今天终于正式版了

Codex 用的是 GPT-5 Codex 模型，专门为写代码和 agentic coding 优化的。

4、API 全家桶

当然，每次发布会，最让人期待的还是 API。

这次，OpenAI 直接发布了三套模型的 API：
◦ GPT-5 Pro
◦ Sora2 & Sora2 Pro
◦ Real-Time Mini

其中 Sora 2：0.1刀/秒，pro 版本价格是 0.3-0.5 刀/秒

今年的 DevDay，感觉 OpenAI 的野心是想让 AI 变为一个连接万物的操作系统。
