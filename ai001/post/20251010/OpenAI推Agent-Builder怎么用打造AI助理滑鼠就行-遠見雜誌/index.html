OpenAI推Agent Builder怎么用？打造AI助理滑鼠就行

郭宇璇

OpenAI DevDay 2025发布引起社群热议的是AI代理平台Agent Builder新工具。取自OpenAI

ChatGPT之父、OpenAI奥特曼近期动作超多， 6日除了宣布与AMD结盟，在Open AI DevDay 2025上还发布引起社群热议的是AI代理（AI Agent）平台Agent Builder新工具。这项应用让不少人联想到热门的n8n、Dify等其他自动化工具，因为只要在萤幕上用滑鼠拖曳就能打造AI助理的工作流，零程式码就能跑起来。 Agent Builder的特色与市面上自动化工具差异在哪？又要怎么用？一文解析。

OpenAI在本周6日开启了给开发者的DevDay 2025，并发布了工具组AgentKit，为的是更加实现AI Agent的理想。不过，这其中马上引起全球社群热议的是Agent Builder，到底厉害在哪里？

AgentKit是什么？其中的Agent Builder怎么用？

AgentKit是OpenAI推出的工具组，整合了从建立、部署到优化AI代理的过程中可使用的工具，包括建立AI代理与工作流的平台Agent Builder、管理OpenAI产品间的资料与连接工具的Connector Registry，以及可让开发者在产品中加入聊天式代理的工具组ChatKit。

在这之中，Agent Builder最受关注，其用拖拉和连接节点（nodes）的方式，加上自然语言下prompt，还能使用MCP串接第三方工具，不用写程式光用滑鼠在萤幕上拖曳，就能快速打造以AI代理为核心的工作流。

根据OpenAI，建构AI代理主要可分为三个步骤：第一步为在Agent Builder上设计每个代理的任务，以及执行的逻辑流程；第二步为将设计好的流程公开发布，系统会自动建立版本与ID，方便后续追踪；最后，则是可透过ChatKit将AI代理直接嵌入网站或应用程式，也可以下载SDK程式码自行部署。

Agent Builder的应用场景

目前Agent Builder已开放给所有用户，可以在OpenAI的线上平台上使用。进入Agent Builder后，其已提供六个范例模板可直接试用，从中可窥见未来的应用场景如下：

─结构化Q&A

─内部知识助理

─文件比较

─客服助理

─工作计画制订

─汇整资料回答问题

在Agent Builder中，已有数个模板可直接试用。截自OpenAI

以「内部知识助理」（Intenral knowledge assistant）为例，其为一个包含五个步骤的工作流。首先是让使用者输入想问的问题，接着会经过两个AI代理，第一个AI代理会改写使用者的讯息，让其与内部知识库的内容更相关；

第二个则会将问题分类，判断其是属于要在内部搜寻的QA问题、或是要向外搜寻的事实查找（fact─finding）问题。第四步的If/else节点会将分类后的讯息分流，再让最后一步的三个AI代理各自处理对应的问题，对于不属于前述两类的问题，则要求使用者补充更多资讯。

OpenAI提供的模板中的「内部知识助理」。截自OpenAI

Agent Builder的安全措施 

AI代理涉及许多存取使用者资料的任务，因此，防止私人资料外泄与恶意攻击便是重要议题。对于AI代理，最常见的攻击为提示注入（Prompt Injections），有心人士可能会将恶意内容藏在资料中，试图让AI改写其原始指令，改为执行会窃取个人资料的任务。

对此，在Agent Builder中设有护栏（Guardrails）节点，可对使用者输入的内容进行初步审核，挡下可能有提示注入攻击疑虑、含有仇恨或骚扰内容，或是带有个人隐私资讯的输入讯息。

Agent Builder中的护栏功能。截自OpenAI

此外，也可以使用Agent Builder的评估（Evals）以及轨迹评分（Trace Grading）功能，评估AI代理执行流程的正确性与效能，在正式上线前找出潜在问题。

Agent Builder与n8n、Dify比较

Agent Builder的出现，让不少人担心OpenAI作为AI龙头公司，会让n8n或Dify等近期热门的自动化工具被取代。但其实，这三种工具的定位和功能并非完全相同，下表整理了完整比较：

台湾目标跃升亚洲资产管理中心！ 玉山银行领航高资产财富管理新格局

陈薪智

陈薪智

玉山银行串联国际、深耕家族财管，驱动台湾成为亚洲资产管理中心。

近年金管会积极推动台湾成为「亚洲资产管理中心」，放眼2030年将我国财富管理规模翻倍至60兆元，近期选择高雄作为全台第一个试验点。台湾欲在亚洲扮演金融要角，亟需释放高资产客群价值，解决他们过往跨境财务遭遇的限制。玉山银行作为首批获准开办高雄资产管理专区的银行之一，对高资产族群将带来哪些关键助益？

2025这一年来，川普2.0的多变政策，加上通膨压力、汇率波动、多国央行多数升息后货币紧缩效应，全球资金加速流动。另一方面，金融环境规范日益严谨，受控外国公司（CFC）制度于2023年正式上路，反避税浪潮推升，这两波驱力促使高资产族群积极寻找符合自身需求的家族财富传承、资金永续管理方案。

玉山银行领先获核准开办，持续以多元特色产品满足高端顾客需求

玉山银行已于高雄苓雅区的玉山旗舰大楼开办资产管理专区业务，提供20项媲美国际金融中心的产品与服务。为强化高资产家族的资产多元化配置与抗通膨能力，玉山银行特别在资产管理专区积极耕耘全球私募市场，并与瑞士百达集团联手，独家引进共同投资私募股权产品。该档基金瞄准ESG永续趋势，锁定环境相关产业，如：污染控制、循环经济以及赋能科技等，为高净值家族提供兼具长期增值与风险分散的私募基金投资选项。

以高雄为起点，玉山银行助力台湾迈向亚洲资产管理中心新纪元。

同时，玉山银行结合投信的力量，透过高雄资产管理专区的创新，与全球私募投资机构领导品牌Ardian携手合作，在台湾优先引进Ardian常青型私募基金商品，以提供投资人参与欧洲与北美核心基础建设的机会，其中包含能源转型、AI数位基建与交通运输等大型投资项目，为顾客带来相对稳定的现金流与抗通膨韧性，提升长期资产配置稳健度及弹性。

此外，玉山银行也提供「保费融资」服务，可搭配超过10档精选保险商品，协助高资产顾客灵活运用资金，兼顾保障与提升财务杠杆效益，真正落实从资产配置、风险控管到价值观传承的整合式家族资本管理。

响应亚资中心政策拓展家族办公室业务，陪伴您走过世代传承的每一步

截至2025年6月，玉山银行的高资产顾客户数为台湾市场第一，较去年底成长逾七成。为何玉山银行在财管2.0业务表现亮眼，且在亚资中心也以领先之姿获得核准开办？事实上，玉山银行从2016 年开始发展私人银行，多年来汇聚投资、税务、法律、信托、不动产等多领域专家，并结合外部会计师与律师资源，以因应市场波动与政策变动，持续梳理个人、家族及企业资产现况，以滚动式策略为顾客调整资产配置规划。

玉山银行深知高资产族群对「专属化、整合式、永续性」服务的高度需求，在响应亚资中心政策下，玉山银行专为资产逾新台币30亿元以上的家族打造的「家族办公室」服务具备以下三大特色。

为协助家族在市场趋势及法规快速变更的环境下即时地调整应变，玉山家族办公室采「主动式顾问咨询」模式，专业团队密切关注市场变化与家族资金运用需求，主动与家族讨论调整最佳化的资产配置。另外，针对家族资产传承规划，由内部专家主动视法规变动及家族成员动态，对家族既有传承规划的影响进行说明，并评估原订传承方案是否需要调整，以确保相关传承方案符合家族不同阶段的需求。

其次，玉山家族办公室提供「一站式整合服务」，从咨询、规划到执行与监督，搭配定期拜访，必要时串联内外部专家共同执行，并居中协助沟通，以确保服务品质及家族需求均被满足。

最后，除了法规制度面的传承规划，情感的连结更是家族永续的关键。玉山家族办公室协助打造「家族永续沟通平台」，透过制定家族宪章、召开定期会议，传递共同的价值观，让后代在精神层面富足，并凝聚多代共识，进一步巩固家族资产传承的架构，共同实现家族永续的愿景。

双核团队服务模式，串起台港星打造具竞争力境外私银平台

由于许多高资产顾客本身是企业主，其财富管理经常涉及「公司层级」与「个人／家族层级」， 加上同时有资产成长与财富传承双轨需求，单一顾问难以处理全部范畴。对此，玉山银行组成「双核团队」，成员包含企业金融关系经理（Relationship Manager）与私人银行顾问（Client Advisor），有助于深度理解顾客需求并提供综合性解决方案。

由于高资产族群的资产多跨足境内与海外，对于跨境金融服务，包含跨境资产活化、股权重组规划、资本市场服务、跨境现金管理等需求复杂度也日益提高。对此，玉山银行串联台、港、星三地的理财平台，提供顾客全方位财管咨询服务，并于2024年延揽具国际私银经验的专家加入玉山，增设香港私银中心、新加坡私银中心，持续打造具竞争力的境外私银平台，成为境内外高端客群首选私人银行。

随着2024年玉山海外分子行税前获利首次突破100亿元，创历史新高之际，玉山表示他们除了持续耕耘11个海外国家地区的33个营业据点，后续会完善亚太布局，目标于2026年完成国际私银系统专案的建置以服务更多海内外高资产人士，同时也助力台湾跃升为亚洲资产管理中心。
