OpenAI发布AgentBuilder允许用户构建工作流程

我们先预览一下大致内容

在旧金山Moscone中心的璀璨灯光下，Sam Altman以一句"我们不是在见证历史——我们正在编写未来操作系统的最初几行代码"拉开了OpenAI DevDay 2025的序幕。这场被业界誉为"AI开发者界的奥运会"的盛会，不仅展示了OpenAI在AI技术领域的最新突破，更揭示了AI应用生态系统的未来蓝图。作为2025年最重要的开发者盛会，DevDay 2025不仅是一场技术展示，更是AI开发范式的一次革命性转变。

一、开发者生态的惊人增长：从200万到400万的飞跃

DevDay 2025的开场数据令人震撼：OpenAI平台的周活跃开发者数量已从2023年的200万增长至2025年的400万，增幅达100%；同时，ChatGPT的周活跃用户也从1亿飙升至8亿，增长了700%。API处理能力更是从3亿token/分钟提升至60亿token/分钟，实现了20倍的飞跃。

这一增长并非偶然。OpenAI通过持续优化开发者体验、降低技术门槛，使AI开发从"精英专属"转变为"大众可及”。更值得注意的是，2025年的API调用量中，75%来自企业级应用，25%来自消费级应用，这表明AI已从工具级应用深度融入企业核心流程，成为数字化转型的关键驱动力。

二、核心产品发布：构建AI应用的全新工具链

1. App SDK：应用开发的革命性突破

OpenAI此次最令人瞩目的发布是App SDK，它彻底改变了AI应用的开发方式。App SDK基于MCP（Multi-Context Protocol）开放标准，提供了一套完整的应用开发框架，使开发者能在ChatGPT内部直接构建应用，无需额外登录或跳转。

技术架构深度解析：

  • 数据连接层：通过标准化API轻松接入各种数据源，包括数据库、第三方服务和实时数据流
  • 操作触发层：定义应用可执行的操作，如"查询房源"、“生成海报"等，使应用能与ChatGPT无缝交互
  • UI渲染层：提供丰富的UI组件库，支持内联显示和全屏显示两种模式

AdGenic Commerce协议的引入更是开创了AI应用的全新商业模式。这一协议使应用能直接在ChatGPT内完成交易，无需跳转到外部支付页面。支付成功率提升至98.7%，开发者可设置自动分成比例（5%-30%），支持按使用量或订阅收费。这不仅简化了用户流程，也为开发者提供了更直接的收入渠道。

2. AI代理开发工具：Agent Kit

Agent Kit的发布解决了AI代理开发的痛点。传统代理开发需要数小时才能构建完成，而Agent Kit通过可视化工作流编辑器和预置模板，将开发时间缩短至15分钟。其核心创新包括：

  • 工作流可视化编辑：拖拽式界面设计代理流程，无需编写复杂代码
  • 性能监控仪表盘：实时显示代理响应时间、成功率和资源消耗
  • 自动优化建议：基于历史数据推荐工作流改进方案

Agent Kit的错误率降低了65%，内存占用减少70%，使多代理协同工作成为可能。这标志着AI代理开发从"专家专属"走向"大众可及”。

3. Codex增强功能：代码开发的AI革命

Codex的升级是本次发布会的另一亮点。代码生成准确率提升至92.5%（2023年为85%），支持150+编程语言的深度理解。其"设计思维"模式能根据系统架构图自动生成代码，让开发者从重复性编码中解放出来，专注于系统设计和创造性工作。

更值得关注的是Codex与主流IDE的深度集成，包括VS Code、IntelliJ和PyCharm的原生插件。代码注释自动生成准确率提升40%，调试时间减少55%，使开发效率得到质的飞跃。

4. 新模型发布：GPT-5 Pro、Real-Time Mini与Sora 2

GPT-5 Pro：1.8万亿参数，比GPT-4大2倍，语言理解能力提升40%，代码生成能力提升35%。特别针对企业级应用优化，支持长达10万token的上下文，价格为$0.01/1k token（比GPT-4便宜25%）。

Real-Time Mini：专为实时交互优化的轻量级模型，响应时间<100ms，模型大小仅500MB，可在移动端流畅运行。价格为$0.002/1k token，是实时应用的理想选择。

Sora 2：视频生成质量提升50%，支持4K分辨率，时长最长可达10分钟（比Sora 1提升3倍）。新增"情感渲染"功能，能根据对话内容调整角色表情，为视频生成带来全新维度。

三、实际应用案例：从教育到房地产的全方位展示

Coursera应用：教育领域的AI革命

当用户在ChatGPT中输入"帮我学习Python数据科学入门"，ChatGPT会自动调用Coursera应用获取相关课程。课程视频直接嵌入对话界面，支持边观看边讨论。用户提问"这个算法如何优化？"，ChatGPT能基于视频内容提供解释。视频会自动置顶，便于边看边问。

技术亮点包括视频帧级交互（点击特定帧获取详细解释）、课程进度同步（记住用户观看进度）、个性化推荐（根据学习进度自动推荐后续课程）和多语言支持（20种语言的字幕和内容）。

Canva应用：设计领域的AI赋能

用户输入"设计一个关于AI趋势的年度报告海报，蓝色主题，包含3个数据图表"，Canva应用生成海报草图，直接在ChatGPT中显示。用户可要求"增加一个图表，显示AI市场增长趋势"，生成的海报支持全屏查看，用户可点击调整颜色、字体。

技术亮点包括文本到设计的精确转换（理解"蓝色主题"等设计要求）、多图表协同（处理多个数据图表的布局）、实时预览（每次调整后立即更新预览）和设计参数深度控制（支持指定字体、间距、配色方案）。

Zillow应用：房地产领域的智能交互

用户输入"在纽约找一个3卧室、2浴室的公寓，价格低于$3000/月"，Zillow应用自动筛选房源，返回5个选项。ChatGPT显示房源列表，用户可点击查看详细信息。用户提问"这个房子有什么周边设施？"，ChatGPT能结合Zillow数据回答。用户选择一个房源，可直接在ChatGPT中请求看房。

技术亮点包括智能筛选（理解复杂条件）、信息整合（将房源数据与周边环境信息结合）、交互式地图（在ChatGPT中显示房源位置）、无缝看房预约（直接生成预约链接）和价格趋势分析（提供价格走势预测）。

四、App SDK的关键特性：技术深度解析

1. 内联显示（Inline）模式

基于Web Components标准，将应用内容嵌入到ChatGPT对话流中。应用内容像普通消息一样显示，保持对话连续性，用户可直接在应用内进行操作，无需额外点击。数据流与对话上下文实时同步，确保上下文一致性。

2. 全屏显示（Full Screen）模式

应用内容可扩展为全屏视图，保持与ChatGPT的连接。提供更大空间查看应用内容，适合复杂操作。用户可随时点击右上角"返回"按钮回到对话流，全屏视图中进行的操作会自动同步回对话上下文。

3. “Talking to Apps"功能

应用通过MCP协议将当前上下文发送给ChatGPT，包括当前视图状态、用户操作历史、关键数据点。当用户在应用中查看特定数据时，ChatGPT能提供更相关的解释。例如，用户在Zillow中查看房源详情，ChatGPT能自动提供该区域的市场分析。

4. 多模态支持

最高支持4K分辨率视频，支持帧级交互；支持图片上传、编辑和分析，与对话上下文关联；应用可自定义布局，支持网格、列表、卡片等多种视图；支持点击、拖拽、手势等多种交互方式。

5. 与现有MCP应用兼容

通过MCP协议的向后兼容层，确保新旧应用都能运行。现有应用可逐步升级到新SDK，无需重写，兼容应用在新环境中性能提升30%。现有应用开发者可快速集成新功能，减少开发成本。

五、未来规划：构建AI应用生态系统的蓝图

1. App SDK预览阶段

2025年11月15日开放预览，开发者通过OpenAI开发者门户注册申请。提供详细文档、视频教程和开发者社区支持。预览期功能包括基础应用构建能力，但不包括应用商店发布。

2. 应用审核和发布流程

审核标准：

  • 功能完整性（90%以上功能可用）
  • 用户体验（符合UI/UX设计规范）
  • 性能指标（响应时间<2秒，错误率<5%）
  • 安全合规（符合数据隐私标准）

审核流程：

  1. 提交应用草案（1-2周）
  2. 技术审核（3-5个工作日）
  3. 用户体验审核（2-3个工作日）
  4. 发布准备（1个工作日）

平均7个工作日完成审核。

3. 应用目录系统

目录结构：

  • 分类：教育、设计、房地产、开发工具等
  • 排名：基于用户评分、使用频率、创新性
  • 搜索：支持关键词、类别、功能筛选

展示机制：

  • 首页推荐：每日更新的精选应用
  • 热门应用：按周/月下载量排序
  • 新应用：新发布应用的特别展示
  • 精品应用：获得"OpenAI认证"的应用

4. 应用展示优先级规则

基础展示：所有通过审核的应用默认展示。

优先展示：

  • 通过"OpenAI认证”（应用质量、性能、用户体验评分≥4.5/5）
  • 与OpenAI核心产品深度集成（如GPT-5、Sora 2）
  • 用户活跃度高（月活跃用户>10万）
  • 创新性（获得专利或独特功能）

展示位置：优先展示在ChatGPT应用商店首页、热门应用栏。

5. 高标准应用推荐机制

认证标准：

  • 性能：响应时间<1秒，错误率<2%
  • 体验：用户满意度≥4.7/5
  • 创新：提供独特的AI功能或用户体验
  • 商业价值：有清晰的商业模式和用户增长

认证流程：

  1. 自动化测试（通过OpenAI的性能测试工具）
  2. 人工评估（由OpenAI产品团队评估）
  3. 用户反馈收集（基于早期用户数据）
  4. 颁发认证证书（在应用详情页展示）

认证应用优势：

  • 更高的展示位置
  • 优先获得OpenAI营销支持
  • 专属开发者支持通道
  • 与OpenAI新功能的优先集成

六、行业影响与未来展望

Sam Altman在演讲结尾强调：“现在是构建AI应用的最佳时机。我们不仅提供了工具，更创造了一个生态系统，让开发者能快速将创意转化为产品。从想法到产品的速度，已经从数月缩短到数天。”

这一声明背后是深刻的行业变革。DevDay 2025不仅展示了技术突破，更揭示了AI应用开发的范式转移：从"开发应用，然后将它们放在应用商店"，转向"在AI平台内构建应用，直接与用户交互"。

这种转变将带来三个关键影响：

  1. 开发效率革命：应用开发时间从数月缩短至数天，使创新速度提升10倍以上。
  2. 用户体验提升：应用与AI平台的深度整合，提供无缝的用户体验，减少用户学习成本。
  3. 商业模式创新：AdGenic Commerce协议等新机制，为AI应用开辟了全新的收入渠道。

随着App SDK的全面推出，我们有望看到更多基于AI的应用涌现，从教育到医疗，从娱乐到企业服务，AI将真正融入我们生活的方方面面。

八. 如何快速体验

  1. 立即玩 AgentBuilder
    platform.openai.com/agent-builder 扫码即用，模板库含“电商客服”“HR 面试”“SEO 写作”等 20+ 场景。
  2. 申请 Apps SDK Preview
    表单地址 developers.openai.com/apps-sdk 填包名 + 用例，48 h 内发邀请。
  3. 升级 CLI
    openai version ≥ 1.8.0 已内置 apps 与 codex 子命令，一条 openai apps deploy 即可上传。

结语

OpenAI DevDay 2025不仅是一场技术盛会，更是AI应用开发新时代的开端。通过App SDK、Agent Kit、Codex增强功能和全新模型的组合，OpenAI为开发者提供了一个前所未有的生态系统，使构建和部署AI应用变得前所未有的简单和高效。

正如Sam Altman所言，这不是在见证历史，而是在编写未来的代码。对于开发者而言，现在是加入这场革命的最佳时机。随着App SDK的全面开放和应用生态的逐步成熟，我们有理由相信，AI应用的黄金时代已经到来。

正如一位开发者在DevDay现场的感言：“过去，我们构建应用；现在，我们构建AI。” 这句话完美概括了OpenAI DevDay 2025带来的范式转变，也预示着AI应用开发的全新未来。

写在最后
