免费版：入门级选择

Perplexity AI 的免费版为一般用户提供了无限制的基础搜索，全部由标准 AI 模型驱动。 此外还包括：

  • 每日 5 次 Pro 高级搜索（支持多步推理与更深度洞察）

  • 每4小时可提出5个追问

  • 支持基础档案附件

此方案完全免费，非常适合偶尔搜索与轻量使用需求。

Pro 专业版：高性价比升级

Perplexity Pro定价为20美元/月或200美元/年（节省17%），是最受欢迎的方案。 功能包括：

  • 无限制 Pro 高级搜索

  • 访问高级 AI 模型（如 Claude 3.7 Sonnet）

  • 每月 5 美元 API 点数（供开发者调用接口）

  • 无限制文件上传与分析

  • 产品功能抢先体验

  • 增强研究功能

Max 高级版：顶级个人方案

2025 年 7 月，Perplexity 推出 Max 方案，价格为 50 美元/月。 此方案专为高强度研究者与生产力用户设计，功能包括：

  • 无限制Labs使用，可建立个性化仪表板与应用

  • 优先体验新功能（如 Comet 浏览器）

  • 顶级 AI 模型支持（OpenAI o3-pro、Claude Opus 4 等）

企业版：团队与组织首选

企业方案起价为 40 美元/用户/月或400 美元/用户/年，提供：

  • 团队管理与组织级权限控制

  • 内部知识库搜索

  • 组织档案库

  • 专属客户支持与白手套服务

  • 可扩展的批量订阅

Perplexity AI 各方案功能对比

  • 免费版：无限制基础搜索，但 Pro 搜索仅限每日 5 次，无优先支持。

  • Pro：无限制 Pro 搜索，多模型切换，支持档案分析与实时协作。

  • Max：在 Pro 基础上升级，提供无限制 Labs、实时新功能体验、多模型深度研究。

  • 企业版：支持团队协作、知识管理与安全合规，适合中大型组织。

总体而言，层级越高的方案在研究深度、多模型支持与团队协作上的优势越明显。

Perplexity AI API 定价与使用限制

对于开发者而言，API 定价尤其关键：

  • 免费用户：无API权限。

  • Pro/Max 用户：每月 5 美元 API 点数。

  • 收费标准：按使用量计费，每百万代币0.2-5美元不等。

  • 企业版：支持更高额度，无使用上限。

API 功能涵盖搜索、推理、脱机聊天等，并支持自动充值与组织级使用权限管理。

订阅支付与管理方式

Perplexity 支持按月或按年支付，年缴更优惠。 支付方式包括：

  • 信用卡

  • 银行/电汇

  • 企业级发票

在管理后台，用户可进行升级/降级、组织席位分配与账单管理。 取消订阅立即生效，但部分月费不支持退款。
