第一单元：物质的变化

单元主题

物质的物理变化与化学变化

知识结构

物质的变化

├── 物质的变化类型

│   ├── 物理变化

│   └── 化学变化

├── 物质的性质

│   ├── 物理性质

│   └── 化学性质

├── 化学反应的类型

│   ├── 化合反应

│   ├── 置换反应

│   └── 复分解反应

└── 质量守恒定律

重点知识点

1.1 物质的变化类型

物理变化

  • 定义：没有生成新物质的变化

  • 特征：物质的形态、状态发生改变，但组成成分不变

  • 实例：水的三态变化、玻璃破碎、金属导电

化学变化

  • 定义：生成新物质的变化

  • 特征：有新物质生成，常伴随颜色改变、放出气体、生成沉淀、能量变化等

  • 实例：铁生锈、木材燃烧、食物腐败

物理变化与化学变化的区别

  • 根本区别：是否有新物质生成

  • 联系：化学变化过程中一定伴随着物理变化，但物理变化过程中不一定发生化学变化

1.2 物质的性质

物理性质

  • 定义：物质不需要发生化学变化就能表现出来的性质

  • 包括：颜色、状态、气味、熔点、沸点、硬度、密度、溶解性、导电性、导热性等

化学性质

  • 定义：物质在化学变化中表现出来的性质

  • 包括：可燃性、氧化性、还原性、酸碱性、稳定性等

1.3 化学反应的类型

化合反应

  • 定义：由两种或两种以上物质生成一种新物质的反应

  • 通式：A + B → AB

  • 实例：C + O₂ → CO₂

分解反应

  • 定义：由一种物质生成两种或两种以上新物质的反应

  • 通式：AB → A + B

  • 实例：2H₂O → 2H₂↑ + O₂↑

置换反应

  • 定义：一种单质和一种化合物反应，生成另一种单质和另一种化合物的反应

  • 通式：A + BC → AC + B

  • 实例：Fe + CuSO₄ → FeSO₄ + Cu

复分解反应

  • 定义：两种化合物相互交换成分，生成另外两种化合物的反应

  • 通式：AB + CD → AD + CB

  • 发生条件：生成沉淀、气体或水

  • 实例：HCl + NaOH → NaCl + H₂O

1.4 质量守恒定律

定律内容

  • 参加化学反应的各物质的质量总和，等于反应后生成的各物质的质量总和

微观解释

  • 化学反应前后，原子的种类、数目、质量都没有改变

应用

  • 解释化学反应前后物质的质量变化

  • 确定化学反应中某物质的质量

  • 推断化学反应的化学式

典型例题

例 1：物质变化类型的判断

下列变化中，属于化学变化的是（ ）

A. 冰雪融化 B. 酒精挥发 C. 食物腐败 D. 矿石粉碎

解答：

A. 物理变化（状态改变）

B. 物理变化（状态改变）

C. 化学变化（有新物质生成）

D. 物理变化（形状改变）

答案：C

例 2：化学反应类型的判断

下列反应中，属于置换反应的是（ ）

A. 2H₂O₂ → 2H₂O + O₂↑

B. Zn + H₂SO₄ → ZnSO₄ + H₂↑

C. CaO + H₂O → Ca (OH)₂

D. NaOH + HCl → NaCl + H₂O

解答：

A. 分解反应

B. 置换反应（单质 + 化合物→新单质 + 新化合物）

C. 化合反应

D. 复分解反应

答案：B

例 3：质量守恒定律的应用

在化学反应 A + B → C + D 中，已知 20g A 和 10g B 恰好完全反应，生成 5g C。若要生成 5g D，需要参加反应的 A 的质量为（ ）

A. 4g B. 5g C. 10g D. 15g

解答：

根据质量守恒定律：

参加反应的 A 和 B 的质量总和 = 生成的 C 和 D 的质量总和

20g + 10g = 5g + m (D)

m (D) = 25g

即 20g A 完全反应生成 25g D

设生成 5g D 需要 A 的质量为 x：

20g : 25g = x : 5g

x = 4g

答案：A

易错点分析

易错点 1：物理变化与化学变化的判断

  • 错误：根据现象判断，如发光发热就是化学变化

  • 正确：根本依据是是否有新物质生成

易错点 2：化学反应类型的判断

  • 错误：对置换反应和复分解反应的概念理解不清

  • 正确：置换反应是单质与化合物反应，复分解反应是两种化合物相互交换成分

易错点 3：质量守恒定律的应用

  • 错误：忽略 “参加反应” 的条件

  • 正确：只有参加反应的物质质量才计算在内

考点总结

高频考点

  1. 物理变化与化学变化的判断

  2. 物理性质与化学性质的区别

  3. 化学反应类型的判断

  4. 质量守恒定律的应用

解题技巧

  1. 判断物质变化类型时，关键看是否有新物质生成

  2. 区分物理性质和化学性质时，看是否需要通过化学变化表现出来

  3. 判断化学反应类型时，要熟悉各种反应的定义和通式

  4. 应用质量守恒定律时，要注意 “参加反应” 的物质质量

物质的转化规律与常见材料

重点知识点

2.1 物质的分类

纯净物与混合物

  • 纯净物：由一种物质组成，有固定的组成和性质

  • 混合物：由多种物质组成，没有固定的组成和性质

  • 实例：氧气（纯净物）、空气（混合物）

单质与化合物

  • 单质：由同种元素组成的纯净物

  • 化合物：由不同种元素组成的纯净物

  • 实例：铁（单质）、水（化合物）

氧化物、酸、碱、盐

  • 氧化物：由两种元素组成，其中一种是氧元素的化合物

  • 酸：电离时产生的阳离子全部是氢离子的化合物

  • 碱：电离时产生的阴离子全部是氢氧根离子的化合物

  • 盐：由金属阳离子（或铵根离子）和酸根阴离子组成的化合物

2.2 物质的转化规律

金属的化学性质

  1. 与氧气反应：金属 + 氧气 → 金属氧化物
  • 实例：3Fe + 2O₂ → Fe₃O₄
  1. 与酸反应：活泼金属 + 酸 → 盐 + 氢气

  • 条件：金属活动性顺序中排在氢前面的金属

  • 实例：Zn + H₂SO₄ → ZnSO₄ + H₂↑

  1. 与盐反应：金属 + 盐 → 新金属 + 新盐

  • 条件：金属活动性顺序中排在前面的金属能把排在后面的金属从其盐溶液中置换出来

  • 实例：Fe + CuSO₄ → FeSO₄ + Cu

酸的化学性质

  1. 与指示剂反应：使紫色石蕊试液变红，不能使无色酚酞试液变色

  2. 与活泼金属反应：酸 + 活泼金属 → 盐 + 氢气

  3. 与金属氧化物反应：酸 + 金属氧化物 → 盐 + 水

  • 实例：Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O
  1. 与碱反应：酸 + 碱 → 盐 + 水（中和反应）
  • 实例：HCl + NaOH → NaCl + H₂O
  1. 与某些盐反应：酸 + 盐 → 新酸 + 新盐

  • 条件：生成沉淀、气体或水

  • 实例：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

碱的化学性质

  1. 与指示剂反应：使紫色石蕊试液变蓝，使无色酚酞试液变红

  2. 与非金属氧化物反应：碱 + 非金属氧化物 → 盐 + 水

  • 实例：2NaOH + CO₂ → Na₂CO₃ + H₂O

  1. 与酸反应：碱 + 酸 → 盐 + 水（中和反应）

  2. 与某些盐反应：碱 + 盐 → 新碱 + 新盐

  • 条件：反应物都可溶，生成物中有沉淀

  • 实例：2NaOH + CuSO₄ → Cu (OH)₂↓ + Na₂SO₄

盐的化学性质

  1. 与金属反应：盐 + 金属 → 新盐 + 新金属

  2. 与酸反应：盐 + 酸 → 新盐 + 新酸

  3. 与碱反应：盐 + 碱 → 新盐 + 新碱

  4. 与某些盐反应：盐 + 盐 → 两种新盐

  • 条件：反应物都可溶，生成物中有沉淀

  • 实例：NaCl + AgNO₃ → AgCl↓ + NaNO₃

2.3 常见材料

金属材料

  • 纯金属：铁、铜、铝等

  • 合金：由一种金属跟其他金属（或非金属）熔合形成的具有金属特性的物质

  • 特性：硬度比组成它的纯金属大，熔点比组成它的纯金属低

  • 实例：生铁、钢、黄铜、铝合金

无机非金属材料

  • 传统无机非金属材料：陶瓷、玻璃、水泥等

  • 新型无机非金属材料：光导纤维、高温结构陶瓷等

有机合成材料

  • 塑料：聚乙烯、聚氯乙烯、聚苯乙烯等

  • 合成纤维：涤纶、锦纶、腈纶等

  • 合成橡胶：丁苯橡胶、顺丁橡胶等

复合材料

  • 定义：由两种或两种以上不同性质的材料，通过物理或化学的方法，在宏观（微观）上组成具有新性能的材料

  • 实例：玻璃钢、碳纤维复合材料等

典型例题

例 1：物质的分类

下列物质中，属于氧化物的是（ ）

A. O₂ B. H₂O C. KClO₃ D. H₂SO₄

解答：

氧化物是由两种元素组成，其中一种是氧元素的化合物。

A. 单质

B. 氧化物（H 和 O 两种元素）

C. 盐（K、Cl、O 三种元素）

D. 酸（H、S、O 三种元素）

答案：B

例 2：金属的化学性质

将铁丝插入下列溶液中，有明显现象的是（ ）

A. 稀盐酸 B. 硫酸锌溶液 C. 氢氧化钠溶液 D. 氯化钠溶液

解答：

根据金属活动性顺序：

  • 铁排在氢前面，能与稀盐酸反应产生氢气

  • 铁排在锌后面，不能与硫酸锌溶液反应

  • 铁不能与氢氧化钠溶液反应

  • 铁不能与氯化钠溶液反应

答案：A

例 3：物质的转化

下列物质间的转化，不能一步实现的是（ ）

A. CuO → Cu (OH)₂ B. H₂SO₄ → H₂O

C. Fe → FeSO₄ D. CO₂ → CaCO₃

解答：

A. CuO 不能一步转化为 Cu (OH)₂（CuO 不溶于水，不能与水反应）

B. H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O（一步实现）

C. Fe + H₂SO₄ → FeSO₄ + H₂↑（一步实现）

D. CO₂ + Ca (OH)₂ → CaCO₃↓ + H₂O（一步实现）

答案：A

易错点分析

易错点 1：物质的分类

  • 错误：对氧化物的概念理解不清，认为含有氧元素的化合物就是氧化物

  • 正确：氧化物是由两种元素组成，其中一种是氧元素的化合物

易错点 2：金属活动性顺序的应用

  • 错误：忽略金属与盐反应的条件

  • 正确：金属必须排在盐中金属元素的前面，且盐必须可溶

易错点 3：复分解反应的条件

  • 错误：认为两种化合物都可溶就能发生复分解反应

  • 正确：复分解反应发生的条件是生成沉淀、气体或水

考点总结

高频考点

  1. 物质的分类

  2. 金属的化学性质

  3. 酸、碱、盐的化学性质

  4. 物质的转化规律

  5. 常见材料的性质和用途

解题技巧

  1. 物质分类时，要掌握各类物质的定义和特点

  2. 应用金属活动性顺序时，要记住金属活动性顺序表

  3. 掌握酸、碱、盐的化学性质，注意反应发生的条件

  4. 物质转化问题，要熟悉各类物质的化学性质和转化规律

重点知识点

3.1 能量的形式

机械能

  • 动能：物体由于运动而具有的能量

    • 公式：$E_k = \frac{1}{2}mv^2$

    • 影响因素：质量、速度

  • 势能：

    • 重力势能：物体由于被举高而具有的能量

      • 公式：$E_p = mgh$

      • 影响因素：质量、高度

    • 弹性势能：物体由于发生弹性形变而具有的能量

      • 影响因素：弹性形变程度、材料性质

内能

  • 定义：物体内部所有分子热运动的动能和分子势能的总和

  • 影响因素：温度、质量、状态

  • 改变方式：做功和热传递

化学能

  • 定义：物质发生化学反应时所释放或吸收的能量

  • 实例：燃料燃烧、食物消化、电池放电

电能

  • 定义：电流所具有的能量

  • 公式：$W = UIt$

  • 单位：焦耳（J）、千瓦时（kW・h）

光能

  • 定义：光所具有的能量

  • 实例：太阳能、灯光

核能

  • 定义：原子核发生变化时释放的能量

  • 类型：核裂变、核聚变

3.2 能量的转化

机械能与内能的转化

  • 机械能转化为内能：摩擦生热、钻木取火

  • 内能转化为机械能：蒸汽机、内燃机

机械能与电能的转化

  • 机械能转化为电能：发电机

  • 电能转化为机械能：电动机

化学能与内能的转化

  • 化学能转化为内能：燃料燃烧

  • 内能转化为化学能：某些化学反应（如光合作用）

化学能与电能的转化

  • 化学能转化为电能：干电池、蓄电池

  • 电能转化为化学能：电解水、蓄电池充电

3.3 能量守恒定律

定律内容

  • 能量既不会凭空产生，也不会凭空消失，它只会从一种形式转化为另一种形式，或者从一个物体转移到另一个物体，而在转化和转移的过程中，能量的总量保持不变

实际应用

  • 解释各种能量转化现象

  • 分析能量转化效率

  • 指导节能技术的开发

典型例题

例 1：能量的形式

下列关于能量的说法中，正确的是（ ）

A. 机械能只包括动能

B. 物体的温度越高，内能越大

C. 能量是可以凭空产生的

D. 化学能只能转化为内能

解答：

A. 错误，机械能包括动能和势能

B. 正确，物体的温度越高，分子热运动越剧烈，内能越大

C. 错误，能量不能凭空产生，也不能凭空消失

D. 错误，化学能可以转化为多种形式的能量

答案：B

例 2：能量的转化

下列过程中，属于机械能转化为内能的是（ ）

A. 汽油机的做功冲程

B. 冬天搓手取暖

C. 发电机发电

例 3：能量守恒定律的应用

一个质量为 1kg 的物体从 10m 高处自由下落，不计空气阻力，求物体落地时的速度。（g 取 10N/kg）

解答：

根据能量守恒定律，重力势能转化为动能：

$E_p = E_k$

$mgh = \frac{1}{2}mv^2$

两边都除以 m：

$gh = \frac{1}{2}v^2$

代入数据：

$10N/kg \times 10m = \frac{1}{2}v^2$

$100 = \frac{1}{2}v^2$

$v^2 = 200$

$v = \sqrt{200} = 10\sqrt{2} \approx 14.14m/s$

答案：物体落地时的速度约为 14.14m/s。

易错点分析

易错点 1：能量的形式

  • 错误：认为只有运动的物体才有能量

  • 正确：任何物体都具有能量，能量有多种形式

易错点 2：能量的转化

  • 错误：认为能量转化是单向的

  • 正确：能量可以在多种形式之间相互转化

易错点 3：能量守恒定律的理解

  • 错误：认为能量守恒定律只适用于某些特定情况

  • 正确：能量守恒定律是自然界最普遍、最重要的基本定律之一，适用于任何情况

消化过程

  • 物理性消化：牙齿的咀嚼、舌的搅拌、胃肠的蠕动

  • 化学性消化：通过消化酶的作用，将大分子物质分解为小分子物质

吸收过程

  • 主要场所：小肠

  • 吸收方式：

    • 葡萄糖、氨基酸、无机盐、水等：被小肠绒毛毛细血管吸收

    • 甘油、脂肪酸：被小肠绒毛毛细淋巴管吸收

4.2 循环系统

血液

  • 毛细血管：连接动脉和静脉，进行物质交换

心脏

  • 结构：四个腔（左心房、左心室、右心房、右心室）

  • 瓣膜：房室瓣、动脉瓣，防止血液倒流

  • 血液循环：

    • 体循环：左心室→主动脉→全身毛细血管→上、下腔静脉→右心房

  3. 气体在血液中的运输

  4. 组织换气：血液与组织细胞的气体交换

气体交换原理

  • 气体扩散：气体从浓度高的地方向浓度低的地方扩散

4.4 泌尿系统

排泄器官

  • 肾脏：形成尿液

  • 输尿管：输送尿液

  • 膀胱：暂时储存尿液

  • 尿道：排出尿液

  3. 肾小管的分泌作用：排出代谢废物

排泄意义

  • 排出代谢废物

  • 调节体内水和无机盐的平衡

4.5 新陈代谢

同化作用

  • 定义：生物体把从外界环境中获取的营养物质转变成自身的组成物质，并且储存能量的变化过程

异化作用

  • 定义：生物体能够把自身的一部分组成物质加以分解，释放出其中的能量，并且把分解的终产物排出体外的变化过程

能量平衡

  • 能量摄入：通过食物获取能量

  • 能量消耗：基础代谢、体力活动、食物特殊动力作用

  • 能量平衡：能量摄入 = 能量消耗

内分泌系统

激素的作用

  1. 生长激素（垂体分泌）

  • 促进生长发育，主要促进蛋白质合成和骨的生长

  • 幼年时分泌不足：侏儒症

  • 幼年时分泌过多：巨人症

  • 成年时分泌过多：肢端肥大症

  • 促进新陈代谢和生长发育，提高神经系统的兴奋性

  • 幼年时分泌不足：呆小症

  • 成年时分泌不足：甲状腺功能减退症

  • 成年时分泌过多：甲状腺功能亢进症（甲亢）

  • 调节糖代谢，降低血糖浓度

  • 分泌不足：糖尿病

  • 分泌过多：低血糖症

  1. 肾上腺素（肾上腺分泌）
  • 加快心跳，收缩血管，升高血压，促进糖原分解，使血糖升高

神经系统的组成

  • 中枢神经系统：脑和脊髓

  • 周围神经系统：脑神经和脊神经

神经调节的基本方式

  • 反射：人体通过神经系统对刺激作出的规律性反应

  • 反射弧：完成反射的结构基础，包括感受器、传入神经、神经中枢、传出神经、效应器

  • 非条件反射：生来就有的先天性反射（如膝跳反射）

  • 条件反射：在生活过程中逐渐形成的后天性反射（如望梅止渴）

大脑的功能

  • 大脑皮层：神经系统的最高级中枢

  • 功能区：

    • 躯体运动中枢：管理身体对侧骨骼肌的运动

    • 躯体感觉中枢：管理身体对侧皮肤、肌肉等处的感觉

    • 语言中枢：人类特有的神经中枢

    • 视觉中枢：形成视觉

    • 听觉中枢：形成听觉

    • 向化性：植物的根向肥料生长

感性运动

  • 定义：植物受到不定向的外界刺激而引起的局部运动

    • 捕蝇草捕捉昆虫

重点知识点

6.1 电路的基本概念

电路的组成

  • 电源：提供电能的装置（如电池、发电机）

  • 用电器：消耗电能的装置（如灯泡、电动机）

  • 开关：控制电路通断的装置

  • 导线：连接电路元件的导体

电路的状态

  • 通路：处处连通的电路，有电流通过

  • 开路：某处断开的电路，没有电流通过

  • 短路：不经过用电器，直接用导线把电源两极连接起来的电路

电路的连接方式

  • 串联电路：用电器依次连接的电路

  • 并联电路：用电器并列连接的电路

6.2 电流、电压和电阻

电流

  • 定义：电荷的定向移动形成电流

  • 方向：正电荷定向移动的方向（与电子移动方向相反）

  • 单位：安培（A），常用单位有毫安（mA）、微安（μA）

  • 测量工具：电流表，要串联在电路中

电压

  • 定义：使电路中形成电流的原因

  • 单位：伏特（V），常用单位有千伏（kV）、毫伏（mV）

  • 测量工具：电压表，要并联在电路中

电阻

  • 定义：导体对电流的阻碍作用

  • 单位：欧姆（Ω），常用单位有千欧（kΩ）、兆欧（MΩ）

  • 影响因素：材料、长度、横截面积、温度

欧姆定律

  • 内容：导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比

  • 公式：$I = \frac{U}{R}$

  • 变形公式：$U = IR$，$R = \frac{U}{I}$

6.3 电路的计算

串联电路的特点

  1. 电流：处处相等，$I = I_1 = I_2 = \dots = I_n$

  2. 电压：总电压等于各部分电路电压之和，$U = U_1 + U_2 + \dots + U_n$

  3. 电阻：总电阻等于各部分电路电阻之和，$R = R_1 + R_2 + \dots + R_n$

并联电路的特点

  1. 电流：总电流等于各支路电流之和，$I = I_1 + I_2 + \dots + I_n$

  2. 电压：各支路两端的电压相等，$U = U_1 = U_2 = \dots = U_n$

  3. 电阻：总电阻的倒数等于各支路电阻倒数之和，$\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2} + \dots + \frac{1}{R_n}$

7.1 磁现象

磁体和磁极

  • 磁体：具有磁性的物体

  • 磁极：磁体上磁性最强的部分，分为南极（S 极）和北极（N 极）

  • 磁极间的相互作用：同名磁极相互排斥，异名磁极相互吸引

磁场和磁感线

  • 磁场：磁体周围存在的一种特殊物质，能对放入其中的磁体产生力的作用

  • 磁场的基本性质：对放入其中的磁体产生磁力的作用

  • 磁感线：用来描述磁场的假想曲线，磁感线的方向表示磁场的方向

  • 磁感线的特点：

    • 在磁体外部，磁感线从 N 极出发，回到 S 极

    • 磁感线的疏密表示磁场的强弱

    • 磁感线不相交

地磁场

  • 定义：地球周围存在的磁场

  • 地磁场的北极：在地理南极附近

  • 地磁场的南极：在地理北极附近

  • 磁偏角：地理子午线与地磁子午线之间的夹角

7.2 电生磁

电流的磁效应

  • 发现者：奥斯特

  • 内容：通电导线周围存在磁场，磁场的方向与电流的方向有关

通电螺线管的磁场

  • 磁场形状：与条形磁体的磁场相似

  • 磁场方向：用安培定则（右手螺旋定则）判断

    • 右手握住螺线管，四指指向电流的方向，大拇指所指的那端就是螺线管的 N 极

电磁铁

  • 构造：在螺线管内插入铁芯

  • 特点：

    • 磁性的有无可以通过电流的通断来控制

    • 磁性的强弱可以通过改变电流的大小或线圈的匝数来控制

    • 磁极的方向可以通过改变电流的方向来控制

  • 应用：电磁继电器、电铃、电磁起重机等

7.3 磁生电

电磁感应现象

  • 发现者：法拉第

  • 内容：闭合电路的一部分导体在磁场中做切割磁感线运动时，导体中就会产生电流

感应电流的产生条件

  1. 电路必须闭合

  2. 导体必须做切割磁感线运动

发电机

  • 工作原理：电磁感应现象

  • 能量转化：机械能转化为电能

  • 种类：交流发电机、直流发电机

7.4 电动机

磁场对电流的作用

  • 内容：通电导体在磁场中会受到力的作用

  • 力的方向：与电流方向和磁场方向有关（左手定则）

电动机的工作原理

  • 工作原理：通电线圈在磁场中受力转动

  • 能量转化：电能转化为机械能

  • 构造：定子、转子、换向器、电刷等

8.1 波的基本概念

波的产生

  • 波源：产生振动的物体

  • 机械波：需要介质才能传播的波（如声波、水波）

  • 横波：质点的振动方向与波的传播方向垂直的波（如光波、水波）

  • 纵波：质点的振动方向与波的传播方向在同一直线上的波（如声波）

波的描述

  • 波长（λ）：相邻两个波峰或波谷之间的距离

  • 频率（f）：单位时间内完成振动的次数，单位是赫兹（Hz）
