KAT-Coder 开发工具集成指南-文档中心-StreamLake --- KAT-Coder Development Tool Integration Guide-Document Center-StreamLake --知识铺

2025年10月22日
| 阅读

KAT-Coder 系列型号专为代理编码任务而设计。其中，KAT-Coder-Pro 和 KAT-Coder-Air 型号支持与 Claude Code、Cline、Kilo Code 和 Roo Code 集成。从 Vanchin 平台获取 Vanchin API 密钥并创建推理端点后，您可以在各种代理产品中配置和使用它。

获取 Vanchin API 密钥并创建 KAT-Coder 推理端点

一、获取和配置 API Key

获取 API 密钥

登录 Vanchin 平台>左侧边栏菜单 [API 密钥] >点击“创建 API 密钥”

⚠️ 重要提示：请安全地存储生成的 API 密钥。

配置 API 密钥

环境变量配置（推荐）：

export VC_API_KEY="your-api-key-here"

二、准备推理端点

准备在线推理端点

登录 Vanchin 平台 > 左侧边栏菜单 [模型] → [推理] > 点击“ 创建推理端点”

选择 KAT-Coder 型号。

在在线推理页面，找到对应的推理端点，复制推理端点 ID。

克劳德代码

配置以下环境变量，以使用 Claude Code 中 Vanchin 提供的 KAT-Coder 模型：

● ANTHROPIC_BASE_URL： 设置为“https://vanchin.streamlake.ai/api/gateway/v1/endpoints/ep-xxx-xxx/claude-code-proxy”，将“ep-xxx-xxx”替换为您的 Vanchin 推理端点 ID

● ANTHROPIC_AUTH_TOKEN： 设置为您的 Vanchin API 密钥

● ANTHROPIC_MODEL 和 ANTHROPIC_SMALL_FAST_MODEL：设置为“KAT-Coder”

⚠️ 注意：目前仅支持 KAT-Coder 型号。您无法修改通过 ANTHROPIC_MODEL 环境变量使用的模型。

一、Claude 代码安装与配置

二、运行 Claude 代码

在终端中运行以下命令以启动 Claude Code：

claude

克莱恩

I. 安装 Cline 插件

打开扩展市场

    1. 启动 VS Code
    2. 点击 扩展 左侧边栏上的图标
    3. 在搜索框中输入 Cline
    4. 查找 Cline 扩展

安装插件

    1. 单击“安装”按钮进行安装
    2. 安装后，选择信任发布者

二、配置 API 设置

选择 API 密钥方法

选择使用您自己的 API 密钥

按如下方式填写配置详细信息：

  • API 提供商：选择 OpenAI 兼容
  • 基本 URL：输入 https://vanchin.streamlake.ai/api/gateway/v1/endpoints
  • API 密钥：输入您的 Streamlake API 密钥
  • 模型：找到模型 ID，输入万庆推理端点 ID（例如，ep-xxxxxxxxxxxxx）

三、开始使用

配置完成后，您可以在输入框中输入您的请求，模型将帮助您完成各种任务，例如：

  • 创建和编辑文件
  • 生成代码
  • 重构代码
  • 解释代码逻辑
  • 调试问题

Kilo Code

一、安装 Kilo Code 插件

1. 打开扩展市场

  1. 打开 VS Code
  2. 单击左侧边栏上的扩展图标
  3. 在搜索栏中输入 Kilo Code
  4. 找到 Kilo Code 插件

2. 安装插件

  1. 单击“安装”按钮
  2. 安装后，选择信任开发者

二、配置 API 设置

1. 选择 API 密钥选项

选择使用您自己的 API 密钥

2. 填写配置详细信息

请填写以下信息：

  • API 提供商： 选择 OpenAI 兼容
  • 基本 URL：输入 https://vanchin.streamlake.ai/api/gateway/v1/endpoints
  • API 密钥： 输入您的 Streamlake API 密钥
  • 型号： 输入您的万情推理端点 ID（例如 ep-xxxxxxxxxxxxxxx），然后选择使用自定义

三、开始使用

配置完成后，您可以开始在输入框中输入您的请求，AI 模型可以协助完成各种任务，例如：

  • 分析数据库模式
  • 从数据中聚合和计算平均值
  • 生成和优化 SQL 查询
  • 代码生成和重构
  • 项目分析和文档起草

罗代码

一、安装 Roo Code 插件

1. 打开扩展市场

  1. 打开 VS Code
  2. 点击 扩展 左侧边栏上的图标
  3. 在搜索栏中输入 Roo Code
  4. 找到 Roo Code 插件

2. 安装插件

  1. 单击“安装”按钮
  2. 安装后，选择信任开发者

二、配置 API 设置

1. 填写配置详细信息

请输入以下信息：

  1. API 提供商 ：选择 OpenAI 兼容
  2. 基本 URL：输入 https://vanchin.streamlake.ai/api/gateway/v1/endpoints
  3. API 密钥： 输入您的 Streamlake API 密钥
  4. 型号 ：输入您的万情推理端点 ID（例如 ep-xxxxxxxxxxxxxxx），然后选择使用自定义

三、权限设置及使用

1. 配置权限

根据需要选择要允许的权限：

  • 文件读/写作
  • 自动批准执行
  • 项目访问权限

2. 开始使用

在输入框中输入您的请求。Roo Code 可以帮助您：

  • 总结当前项目结构
  • 分析关键模块和功能
  • 代码重构和优化
  • 生成文档和注释
  • 诊断问题并提供修复建议
