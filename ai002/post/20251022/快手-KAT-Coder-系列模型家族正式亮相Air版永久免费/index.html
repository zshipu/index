3.Kilo Code 评测

在10月21日，Kilo Code 发布了一篇 Blog，通过测量速度、成本、代码质量和工具调用等指标对 Haiku 4.5、GLM 4.6 和 GPT5-Mini 三款模型进行了测试对比了三个模型在Al编程任务中的表现。我们沿用 Kilo Code 的测试方法，对 KAT-Coder-Pro V1 进行了实测。KAT-Coder-Pro V1 用约4分钟完成代码生成，成功完成了10项功能，并且全程零工具调用失败，总体花费仅$0.072。

对比来看，KAT-Coder-Pro V1 虽与 Haiku 4.5 和 GLM 4.6 的定价相近，但其在任务中的实际消耗低于这两款模型。这意味着，在相同编程任务中，KAT-Coder-Pro V1 能让用户以更低成本获得稳定且高质量的输出，更好的实现了价格与质量的平衡，无疑是 AI 编程场景下极具竞争力的高性价比之选。

KAT-Coder 系列模型一览：总有一款适合你

今天，我们将带您深入了解 KAT-Coder 模型家族的三款模型的特性，确保每位开发者都能找到最适合自己的模型。

KAT-Coder-Pro V1 高性能AI编码模型，重新定义“极致性价比”

凭借73.4%的解决率，KAT-Coder-Pro V1 在 SWE-Bench Verified 榜单上展现了与全球顶尖闭源模型相媲美的卓越表现。Pro版不仅具备强大的代码生成和工具调用能力，还配备256K超长上下文窗口，轻松应对项目级的代码和智能体任务。您只需用自然语言描述需求，模型便能自主完成从需求分析、架构规划到编码实现和调试纠错的全链路代码解决方案，助力“从0到1”的自动化项目开发。

当前Pro版经过深度适配与优化，完美兼容 Claude Code、Cline、Kilo Code、Roo Code 等多种主流AI编程工具，确保您在最熟悉的环境中无缝集成顶级AI能力，可以说，KAT-Coder-Pro V1专为应对最复杂的工业级和企业级工程场景而设计。

除高性能AI编码模型外，KAT-Coder-Pro V1采用基于上下文窗口的阶梯式定价，极具性价比，大幅降低AI技术使用门槛。

KAT-Coder-Air V1 - 为轻量普惠而生

KAT-Coder-Air V1 是 Pro 版本的轻量级变体，继承了旗舰模型的核心架构，依然保持卓越性能。在 SWE-Bench Verified 榜单上，排名全球同等规模开源模型的第五，是性能与成本的最佳平衡选择。Air 版本支持上下文长度达 128K，最大输出 32K，兼容 MCP 和工具调用，相较于 Pro 版本具有更低的延迟，反应速度更为迅捷。

KAT-Coder-Exp-72B - 为前沿探索而生

KAT-Coder-Exp-72B 是 KAT-Coder 模型强化学习的实验版本，我们希望借助这一开源模型揭秘 KAT-Coder 在大规模强化学习方向上的技术创新，并促进开源社区内的技术交流。该模型在软件开发能力评测基准 SWE-Bench Verified 上取得了74.6%的优异成绩，成功登顶开源模型 SOTA。
