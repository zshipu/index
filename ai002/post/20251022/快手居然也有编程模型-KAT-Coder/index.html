快手居然也有编程模型 KAT-Coder？ --知识铺

2025年10月22日
| 阅读

一直对快手在大模型这块的认知，只停留在可灵 Kling，不曾想快手居然也有编程模型？

今天下午偶然间发现有群友在问KAT-Coder 是否比阿里 Qwen3-Coder 更好用？

心想这哪家公司出的？于是第一时间扒拉了下什么是 KAT-Coder？我勒个豆，居然是快手出品，这可确实是没想到。

看了下， Ask模式、Agent模式、行内chat、ide plugin都有

网址在此：https://kwaipilot.ai

但是看了下社区动态及粉丝关注度，嗯，不太好评～

快手最早编程模型露面是在2025年1月2号，开源了基于DeepSeek-v2-Lite-Base构建的Kwai-Coder-DS-V2-Lite-Base，不得不说还得是DeepSeek

总160亿参数，激活24亿参数，开发成本仅仅5万美元，训练数据是中包含70%的高质量代码数据，最终评测数据如下👇

大家感受下当时 kwai-coder 所处的位置

第二次发声2025年1月26号，推出KwaiCoder-23BA4-v1👇

https://huggingface.co/Kwaipilot/KwaiCoder-23B-A4B-v1，不过没激起什么火花

第三次发声2025年7月21号推出KAI-V1，总40亿参数，可在推理和非推理之间动态切换；👇

同样没什么水花……

最近一次发声 2025 年 9 月 26 号推出 Kat-Dev-32B 和 KAT-Coder👇

KAT-DEV-32B 在开源模型中的位置

KAT-Coder 在闭源模型中的位置

然而社区反馈画风如此👇

1. 你们知道这家公司吗？他们似乎在 HF 上分享了一些简化版本，而把高端版本留给自己？或者他们是通过 API 分享的？没什么关于他们的信息。

2.kat-coder 在哪里？该怎么试用它。

KAT-Dev-32B 是开源版本，在 huggingface 上可使用；

KAT-Coder 是闭源版本，需要在快手万擎中注册使用；

网址：https://www.streamlake.com/product/kat-coder

不但支持 API 调用，还支持 Claude Code 集成

那么，大家有人使用快手编程KAT-Coder 大模型吗？
