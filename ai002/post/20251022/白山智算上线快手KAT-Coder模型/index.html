白山智算上线快手KAT-Coder模型！ --知识铺

2025年10月22日
| 阅读

9月29日，白山云科技旗下“白山智算”平台正式上线快手StreamLake发布的Agentic Coding 闭源旗舰版大模型 ——KAT-Coder，为全球开发者提供突破性代码生成解决方案。

白山智算平台已上架KAT-Coder

即日起，用户可通过访问白山智算平台，以简单易用的API调用方式快速解锁 KAT-Coder 顶尖 Agentic Coding 能力！实名认证并完成首次API调用的新用户，将获得白山智算送出的450元大模型API体验金！

访问地址：

http://ai.baishan.com/website/model-api

模型介绍

KAT-Coder 模型具备强大的代码生成能力，可高效完成从自然语言需求到完整代码交付的端到端开发任务。

模型效果展示

据介绍，该模型的核心优势源于在多个训练阶段进行了创新和优化，包括通过 Mid-Training 注入工具调用与多轮交互能力，SFT（监督微调）阶段深度覆盖算法实现、接口开发、单元测试等八大编程场景，RFT （强化微调）阶段引入 “教师轨迹” 指导优化代码逻辑，最后经大规模 Agentic RL 训练强化复杂任务处理能力，确保代码质量与开发效率双重提升。

在权威评测中，KAT-Coder也展现出顶尖实力。在SWE-Bench Verified 数据集（衡量代码模型解决真实软件工程问题能力的关键指标）中，其解决率高达 73.4%，性能比肩全球顶尖闭源编程模型。

白山云MaaS服务

提供边缘极速AI体验

白山云大模型API服务提供预构建、开箱即用的LLM服务，只需一行代码用户即可完成模型切换与集成。基于白山全球边缘云架构进行就近推理，能够为用户带来＜300ms的超低延时推理体验。

目前，平台已上架DeepSeek系列、通义千问系列、ChatGLM系列等多款大模型，并提供部分免费模型。开发者可在白山边缘算力云平台上对比选择各类模型，以更简单易用、灵活高效的API调用方式迅速开启AI创新之旅。
