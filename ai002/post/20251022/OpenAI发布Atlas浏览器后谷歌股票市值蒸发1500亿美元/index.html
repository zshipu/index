OpenAI发布Atlas浏览器后,谷歌股票市值蒸发1500亿美元

OpenAI 发布 Atlas 浏览器后，相关报道普遍指向谷歌母公司 Alphabet 的股价在新闻发布后短期内出现下挫，市场情绪普遍解读为对 Chrome 主导地位的潜在挑战所致。具体幅度在不同机构与市场时段有差异，大多报道聚焦盘中下跌而非长期趋势，且收盘价通常显示回落或回稳的迹象，影响程度也随地区市场与交易时段波动。以下是对事件要点的梳理。

要点梳理

  • 产品与定位: Atlas 是 OpenAI 基于浏览器的 AI 原生产品，核心特性包括在浏览器中直接使用 ChatGPT、浏览器“记忆”与代理模式等，旨在提升网页检索和自动化任务的效率与便利性。此举直接冲击以谷歌 Chrome 为代表的桌面浏览器市场格局。相关信息源普遍强调 Atlas 的核心集成点与对比 Chrome 的潜在竞争优势。12
  • 市场反应: 多家财经媒体报道，Atlas 发布消息后谷歌母公司 Alphabet 的股价在盘中出现下跌，跌幅在不同报道中大致落在3%到5%区间，部分报道提到盘中最低点与收盘时点的不同表现。整体结论倾向于“短期负面反应”，并指出市场正在评估 Atlas 对 Chrome 的市场份额潜在影响。341
  • 业界解读与影响: 分析普遍认为 Atlas 的推出代表 OpenAI 首次直接进入浏览器生态的尝试，若持续扩大用户生态与功能落地，理论上可能对浏览器市场的竞争格局产生一定压力，尤其在具备代理模式等自动化能力的场景中。不同机构给出的判断和股价反应也反映出对潜在竞争的不同预期。452
