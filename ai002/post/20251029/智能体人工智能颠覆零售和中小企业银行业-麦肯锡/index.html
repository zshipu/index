智能体人工智能颠覆零售和中小企业银行业 麦肯锡 --知识铺

2025年10月29日
| 阅读

代理商在此： “嘿，旅行机器人，预订一个从旧金山直飞巴黎的七日行程，两人同行，经济舱可退款机票，价格低于 2000 美元，并使用奖励最高的信用卡付款。”

智能体人工智能（一种能够自主执行任务和解决问题的技术）将能够处理上述复杂的请求，使人工智能从被动的助手转变为主动的金融代理人，用于购物、支付和投资。

这些发展预计将撼动金融业的经济基础，影响数十亿美元的收入，并对银行、中小企业、信用卡公司等企业的商业模式和收入构成威胁。勇于创新的先行者或将获得颠覆性的机遇，而那些不进行创新的企业则将错失良机。开放银行已扎根的市场将率先感受到这些变化。

信用卡和存款这两种主要的收入“引擎”尤其容易受到影响。它们都严重依赖客户的惯性和品牌熟悉度。但人工智能代理不会在意品牌忠诚度，它们会以结果为导向。当逻辑而非习惯驱动产品选择时，规则将会改变。

OpenAI 的操作员模式1他们已经能够浏览网页、预订酒店并执行多步骤任务。像 Manus AI 这样的初创公司2以及像 Perplexity 这样的工具3他们正进一步拓展业务——跨平台协作，解读用户意图，并采取行动。谷歌4已推出AI模式购物体验，通过Gemini AI提供个性化的商品发现和价格追踪。为了促进这些举措，Visa等支付服务商……5万事达，6和PayPal7最近发布了促进代理商务的新功能，允许代理商代表用户自主触发付款。

一个简单的语音指令——“嘿 SaveBot，请将我的支票账户余额维持在 1000 美元，并将多余的现金转到我的高收益储蓄账户，如果我的余额低于 500 美元，请从我的储蓄账户中补足”——可能会成为日常操作。不久之后，我们甚至可能无需开口。人工智能代理会通过电子邮件、短信或应用程序提醒来识别支付或投资机会，自动执行操作，并在事后告知我们它做了什么。实现这一切的技术已经存在。——格里芬8已开始为其金融科技合作伙伴构建“代理优先”的银行核心系统。

金融机构必须立即做好准备，了解自身价值面临风险的领域、关键的控制点以及如何在以代理为主导的世界中保持竞争力。本文将探讨其中的金融利害关系；代理人工智能将对存款机构、信用卡发行机构及其他相关机构造成的颠覆性影响；勇于创新者将面临的机遇；以及金融机构如何才能更好地为即将到来的代理银行经济做好准备。

所涉价值

全球支付行业每年从存款利差、银行卡经济、跨境资金流动和商户收单等渠道创造超过 2.7 万亿美元的收入。

大约一半的收入集中在两个主要领域：

  • _存款净利息收入。_指银行向账户持有人支付的存款利息与银行投资所赚取的利息之间的差额。
  • _消费信用卡经济学。包括_交易手续费收入、循环余额利息收入以及未兑换的奖励收入。

这两种盈利模式都基于一个简单的道理：大多数消费者并非每天都会把每一分钱都花在刀刃上。利润就来自于消费者潜在消费行为与实际消费行为之间的差距。

智能体人工智能打破了这种平衡。即使只是部分采用这项技术，也可能显著压缩利润空间，或将收入转移到其他业务部门。

存款与流动性：惯性红利

从个人支票账户到中小企业运营账户，存款一直是银行盈利的主要来源。在全球范围内，净利息收入约占零售银行利润的30%。

大多数消费者要么不关注自己获得的利率，要么缺乏时间、工具和动力去优化存款利息收益。他们更关注的是便捷性，即拥有一个维护成本低、免收手续费、集成了ATM、账单支付和财富管理门户等功能的单一银行中心。

美国银行对计息支票账户和储蓄账户支付的全国平均利率分别仅为0.07%和0.38%。9到2025年6月，即使最高的网上储蓄利率超过4%，情况依然如此。此外，过去五年，不计息的活期存款账户余额年均复合增长率为28%，而计息账户余额的年均复合增长率仅为3%。

智能体人工智能系统颠覆了这种逻辑。它们可以实时监控账户余额，比较不同机构的收益率，将闲置资金转入收益更高的账户，然后在账单到期前将资金转回支票账户。这样一来，原本被银行攫取的利差就能更多地流向账户持有人。

近期利率飙升（2022年涌向高收益储蓄）10事实证明，当利率差距明显出现时，消费者会自行转移资金。中国的《余额宝》杂志也证实了这一点。11一只于 2013 年推出的货币市场基金，就体现了这一趋势的增长速度。到 2017 年，用户已将 2680 亿美元的闲置资金投入货币市场基金（参见侧边栏“余额宝：优化规模化之时”）。像 Raisin 这样的平台也正在推动这一趋势的发展。12——一个主要面向欧盟和英国的存款市场——已吸收超过800亿美元的存款，并将这些资金存入250多家银行的高收益储蓄账户。这些案例证明，即使在代理商介入之前，市场优化也是可以实现的。

中小企业已经开始利用API驱动的资金管理自动化技术，实时优化现金和外汇交易。例如，许多中小企业已经采用现金管理平台，实现日常报告、预测、资金划拨以及外汇对冲的自动化。这些平台让客户能够轻松获取实时银行数据。而智能体人工智能则更进一步，将这些功能集成到持续的、以用户偏好为导向的资金管理运营中。

作家威廉·吉布森曾说过：“未来已来，只是尚未普及。” 因此，地理位置将影响变革的速度，一些地区会比其他地区更早地拥抱开放银行。智能体人工智能对存款的影响将首先在欧洲和英国显现，因为在这些地区，支付发起服务提供商 (PISP) 和账户信息服务提供商 (AISP) 的数量正在不断增加。13授权许可，再加上低成本的即时支付渠道，使得账户切换相对容易。

欧洲存款收入总额超过1000亿美元。如果只有10%到20%的消费者采用代理驱动的现金自动转账服务，银行净息差可能会收窄30到50个基点。

代理商将能够超越余额宝，根据收益率、结算时间和用户设定的风险限额，持续地在多家银行和不同利率层级之间重新分配资金。对银行而言，威胁显而易见：低成本资金的流失以及存款粘性下降带来的流动性假设压力。对账户持有人而言，好处同样显而易见：终于可以获得长期以来被忽略的利息收入。

信用卡：已优化

信用卡通过多种途径创造巨额收入——预计到2024年将达到2340亿美元——其中包括来自欠款客户的利息收入、14交换费、年费和罚款费15以及未兑换的奖励（见附件）。

这种利润丰厚的商业模式很大程度上依赖于消费者的惯性。近期调查数据显示16这表明，超过 20% 的持卡人在过去 12 个月内没有兑换任何奖励。“积分失效率”（即因账户关闭或过期而损失的累积奖励积分比例）根据消费者金融保护局的数据，每年约为 3% 至 5%。17

然而，人工智能代理可以帮助被动型消费者，实时自动将消费引导至最佳信用卡，触发新申请以获得更好的信用卡优惠，并在促销利率到期前将余额转移到另一张信用卡。

有些事情已经发生了。Klarna 的“理财故事”功能会根据交易数据调整消费限额；18Curve 的智能规则功能允许用户在其 Curve 卡上设置基于类别或金额的路由；19Apple Wallet 应用引入了圆形拨盘式支付滑块，让持卡人可以立即调整其由高盛发行的 Apple Card 的每月还款额。20这些微型自动化预示着卡片任务的完全委派。

账户间结账威胁

随着开放银行的普及，人工智能代理可以在结账时发起账户间（A2A）支付——绕过银行卡交换系统，从而破坏奖励机制。在欧盟，借记卡交换费上限为0.2%，信用卡交换费上限为0.3%——这限制了套利机会。21但在北美，交换费通常在 1.30% 到 3.25% 之间，这为实时换卡提供了节省成本的动力。22

这种动态优化依赖于通过数字钱包进行的交易，因为实体刷卡交易必须经过交换系统。这使得像 Apple Pay 这样的平台作为人工智能代理渠道拥有先发优势。即使是适度的代理主导型 A2A 普及，也能转移相当一部分交换费和利息收入。

势头与现实相遇

总之，银行业曾经被动的运作方式正变得可编程，也更加动态。原本难以变现的存款变得流动性更强；奖励机制也变得更加透明。即使是小规模的采用，也能显著降低存款利差和信用卡利润率。而存款和信用卡仅仅是开始；个人贷款定价、分期付款费用以及商户收单业务也可能面临与代理优化利差类似的压力。

然而，推广应用并非一帆风顺。代理金融受到诸多现实限制，这些限制将决定其发展速度以及谁将率先受益：

  • _信用评分的界限。_目前的模型会因新查询和账户历史短而扣除信用评分，这限制了代理商在不损害消费者信用评分的情况下更换信用卡或开通余额转移优惠的频率。在出现其他现金流或行为评分模型之前，代理商需要内置信用感知逻辑。
  • _代理错误。_早期版本的代理技术很可能出错。例如，代理可能会支付错误的账单或错误地转移资金，这反映了对用户偏好的误解或软件漏洞。
  • _信任与透明度。_委托重要的金融操作需要的不仅仅是“黑箱”人工智能。代理必须与用户清晰沟通，提供可手动关闭的实时警报，并保留审计跟踪记录，以便用户和监管机构能够准确核实事件经过和原因。
  • _责任与监管。_目前尚缺乏明确的法规来强制执行人工智能行为的问责制。欧盟人工智能法案23（于2024年批准，2026年生效）将代理金融工具归类为“高风险”，要求人工智能具备可解释性、人工控制和第三方审计。在美国，消费者金融保护局（CFPB）的第1033条规定了相关内容。24制定相关规则可以强制要求代理商以标准化的方式访问客户授权的数据，从而为受监管的API使用建立合规框架。在明确的问责机制建立之前，个人和机构都不愿将高风险功能委托给代理商。
  • _用户安全顾虑。_消费者对将账户凭证交给机器人感到担忧是可以理解的。代理需要采用令牌化的零信任架构，仅授予必要的最低权限，并在检测到异常行为时自动撤销访问权限。多因素身份验证——尤其是在大额交易或异常交易的情况下——至少是赢得消费者信任的必要措施。
  • _欺诈和反洗钱 (AML)。_快速洗钱和多账户操作可能触发可疑活动警报或引起对洗钱账户活动的怀疑。为了满足监管机构的要求，代理商必须设置交易速度上限、定期进行客户身份验证 (KYC) 以及实时进行反洗钱监控。
  • _运营轨道。_现代即时支付系统，例如 FedNow。25在美国和欧洲的TIPS/SEPA系统中26Instant几乎完全消除了结算延迟，使代理商能够实时清点资金并进行支付。与此同时，基于稳定币的结算早期试点项目（例如Visa-USDC）也取得了进展。27和圆圈28（例如，与国库整合）可以实现亚秒级可编程转账——尽管此类渠道也引入了链上 KYC 和代币治理方面的考量，发行方需要加以解决。然而，许多机构仍然依赖于传统的支付渠道，例如自动清算系统 (ACH)，这可能会减缓代理支付的普及。

这些障碍都不会阻止代理金融的发展，但它们会决定其发展节奏，并塑造早期的赢家。

代理经济中的控制点

随着代理商承担更多金融决策，零售和中小企业银行及支付业务的价值中心将发生转移。竞争优势可能集中在以下五个关键控制点：

  • _凭证和身份。_代理商需要安全且经用户授权的令牌才能获取余额或跨多个机构发起交易。那些已经管理高信任度凭证的公司——无论是通过欧洲基于 OAuth2 的开放银行 API（PSD2）还是美国的 Visa Token Service 等网络令牌化服务——都拥有明显的优势。同样，大型科技身份提供商和提供委托授权流程的银行可以为每个下游操作提供控制门。因此，成功取决于：
    • _零信任架构_从不假定持久访问。
    • 通过标准化协议（例如 OAuth2/OpenID Connect）实现_动态同意。_
    • _持续的审计追踪_能够同时满足客户和监管机构的需求——掌握这些要素的公司能够控制代理融资中的第一个也是最关键的瓶颈。
  • _信任与责任保障机制。_构建与智能体兼容的产品、提供嵌入式安全防护并分担责任的企业，有望成为生态系统中的首选合作伙伴。值得信赖的品牌可以付费承担责任，从而加速用户对智能体技术的接受度。
  • _商户与平台整合。_实时优化——例如，消费路径规划、优惠触发——将取决于线上线下结账环节的深度整合。如今，大多数交易收单机构对消费者购买行为的了解有限，这意味着以软件为中心的供应商可能获益最大。
  • _决策逻辑。_代理商需要在几毫秒内比较费率、功能、奖励和信用影响。那些为其聚合平台和自身网站构建代理商易读、易于优化的产品的平台，更有可能被代理商选中。目前现有的比价网站可能是最初的赢家，因为它们已经收集了数据和信息，并拥有清晰的联盟营销变现逻辑。
  • _行为数据和意图信号。_第一代智能体需要接受指令，而后续几代智能体则能够在用户表达需求之前就推断出他们的需求。这种学习机制将奖励那些对用户需求有更深刻洞察的机构。像消费者应用那样思考意味着要精心策划每一个触点——推送通知、应用内建议、情境提示——以便智能体能够提供动态引导，使用户持续处于优化循环中。

企业接下来应该怎么做？

向代理人主导的金融模式转变是一个转折点。对于银行而言，在这个新时代蓬勃发展意味着要围绕业绩而非客户忠诚度重新设计产品。以下是制定成功战略的三大支柱：

  • _产品策略。_审核每一项产品是否存在惯性依赖。如果今天由代理人代表客户做决定，你的产品还能胜出吗？
  • _技术基础设施。_使用标准化的 API、透明的定价逻辑和代理商可以解析的丰富元数据，使产品对机器可读且易于优化。
  • _分销和互动。_决定是否以及如何整合到第三方代理生态系统或构建自有生态系统。无论采用哪种方式，产品都必须易于消费者理解和使用。

短期内，银行必须做好迎接全新竞争节奏的准备。它们必须时刻重新赢得自身的相关性——每次代理商更新排名时，它们都必须如此。在制定战略的过程中（参见侧边栏“塑造金融代理未来的五个战略问题”），银行可以通过发布实时利率和流动性API、推出可自动转账的储蓄子账户，以及与客户分享部分增量收益以鼓励客户将资金留在平台内，从而为迎接代理商做好准备。

其他机构也可以效仿，提前做好准备应对变革：

  • _当代理商频繁更换信用卡或默认使用A2A支付渠道时，发卡机构_将面临交换费和利息收入的下滑。发卡机构可以通过以下方式应对：将奖励机制和余额转移规则转换为机器可读格式；通过API提供始终可用的预批准信用额度；以及试点体验式福利——例如VIP活动入场券、惊喜旅行升级或会员专属体验——这些福利虽然算法可以识别，但只有客户才能真正体会到。
  • _如果代理直接与银行API集成，钱包提供商和超级应用_必须防范被边缘化的风险。相反，他们可以将自身定位为代理的首选操作系统：发布开发者工具包，内置透明的授权和覆盖控制机制，并确保用户意图的安全——无论用户是通过语音、聊天还是设备级小部件进行操作。
  • _卡组织_面临着来自 A2A 支付渠道的威胁，这些渠道有可能完全绕过卡组织。为了应对这一挑战，卡组织已经开始部署即时支付渠道，对跨渠道凭证进行令牌化处理，并提供诸如欺诈保障、争议解决和数据洞察等高级便捷服务，供代理商默认启用。
  • _商户收单机构和支付服务提供商 (PSP)_所处的环境中，代理商会在结账时进行小型竞价，在几毫秒内选择成本最低的支付路径，这与当今的在线广告竞价非常相似。为了保持竞争力，他们应该通过 API 提供实时费用报价，向代理商展示促销或返利信息，并与电子钱包集成，提供内置的最低成本路由逻辑。
  • _跨境汇兑专家_将看到外汇价差被压缩，因为代理商会实时比较不同供应商的价格。领先的公司将公布包含所有费用的到岸价格，保证结算时间，并在服务水平协议未达标时自动退款——将可靠性转化为竞争优势。
  • _大型科技平台_可以通过将智能商务嵌入语音助手来巩固其领导地位，语音助手已经拥有强大的消费者使用率——然后通过推荐流、数据洞察或订阅服务来盈利。
  • _如今，商家_在每次结账时都面临着一场微型竞价。大型零售商可以通过提供动态的、基于 API 的优惠券和会员积分来提升代理商的偏好；中小企业可以通过 API 接入与平台无关的优惠活动，或者与提供便于代理商使用的促销工具的收单机构合作。

纵观整个商业格局，一条共同的主线是：让优化引擎能够轻松发现、信任并选择您的产品，同时让用户有充分的理由对这一选择感到满意。智能体人工智能标志着决策方式从用户驱动转向系统主导。它也标志着零售和中小企业融资领域从品牌忠诚度转向绩效导向的选择。率先适应的机构可以将信任和透明度嵌入逻辑层，从而很快能够代表每位客户做出决策。而那些坐等变革的机构则有可能沦为资产负债表上隐形的“公用事业”。这不仅仅关乎利润率的保护，更关乎零售和中小企业融资的重塑。未来将青睐那些业绩卓越的企业，而非那些知名度高的企业。代理商很可能更倾向于选择业绩最佳的产品。他们会找到您的产品吗？
