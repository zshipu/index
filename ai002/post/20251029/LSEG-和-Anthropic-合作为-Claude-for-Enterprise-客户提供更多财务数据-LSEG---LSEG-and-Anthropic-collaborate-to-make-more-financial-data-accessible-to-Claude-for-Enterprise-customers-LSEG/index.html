LSEG 和 Anthropic 合作，为 Claude for Enterprise 客户提供更多财务数据 LSEG --- LSEG and Anthropic collaborate to make more financial data accessible to Claude for Enterprise customers LSEG --知识铺

2025年10月28日
| 阅读

伦敦证券交易所集团 (LSEG) 今日宣布与 Anthropic 达成合作，将授权 Claude 客户访问其通过 LSEG 产品（例如 Workspace 和 Financial Analytics）获得许可的数据。该功能将通过 Anthropic 今日推出的扩展版 Claude for Financial Services 产品线提供。

这是伦敦证券交易所集团人工智能战略“LSEG Everywhere”的最新里程碑，该战略旨在提供可信的授权数据，以扩展金融服务领域的人工智能应用。伦敦证券交易所集团的人工智能就绪内容和分类法拥有无与伦比的深度、广度和质量，涵盖了数十年来的数据集。

LSEG Everywhere 包括采用 MCP，并涉及最近与企业级 AI 应用程序和数据平台的合作，包括 Microsoft、Rogo、Snowflake 和 Databricks。

伦敦证券交易所集团 (LSEG) 正在开放其通过 Workspace 和 Financial Analytics 等产品授权的数据访问权限，以便客户能够自动化财务分析。LSEG 的 AI 就绪内容将分阶段通过 MCP 提供，首先从 LSEG Financial Analytics 开始。在首次发布之后，LSEG 将向 Claude 用户开放更多 AI 就绪内容的访问权限，初始功能将于本月晚些时候推出，其他数据类别也将陆续推出。

伦敦证券交易所集团 (LSEG) 以其开放、不受法律硕士 (LLM) 约束且面向基础设施的合作模式，为金融服务领域安全、可扩展的人工智能树立标杆，并通过 MCP 等开放协议实现工作流程。这有助于客户最大限度地降低成本和定制工程，并加快部署速度。

通过此次合作，伦敦证券交易所集团和 Anthropic 还将嵌入相互的潜在客户开发流程。Claude for Financial Services 客户可以使用 LSEG 旗舰 Workspace 解决方案的许可证来扩充其现有数据。LSEG 正在与 Workspace 客户合作，以确保获得 Claude 的企业级授权访问权限。

**伦敦证券交易所数据与分析联席主管 Ron Lefferts 表示：
**伦敦证券交易所集团长期以来以开放的合作方式以及随时随地满足客户需求而享有盛誉。像 Claude 这样的安全企业级人工智能应用程序以及像 MCP 这样的开放标准正在拓展伦敦证券交易所集团与客户建立深度合作关系的机会。

“借助 Claude for Financial Services，我们的客户现在可以访问 LSEG 无与伦比的财务数据和见解，从而直接在其工作流程中为代理 AI 提供支持和扩展。”

**Anthropic 金融服务产品主管 Nicholas Lin 表示：
**将 Claude 的情报与数据和背景信息相结合，能够带来真正的价值。借助伦敦证券交易所的可靠数据，Claude 能够汇总收益电话会议、浏览尽职调查材料、触发代理工作流程并发现即时市场信号——所有这些都具备企业级控制能力。

此次合作建立在伦敦证券交易所集团 (LSEG) 采用由 Anthropic 率先推出的开放标准模型上下文协议 (MCP) 的基础上。LSEG 的 MCP 服务器将于 10 月 27 日星期一在 Claude MCP 合作伙伴目录中上线，确保安全、无缝地访问 LSEG 授权的 AI 就绪内容。

Claude MCP 合作伙伴目录为企业客户提供了访问一系列值得信赖的合作伙伴的单一访问点，并通过 MCP 实现了与他们自己的 AI 系统和其他第三方应用程序的完全互操作性。

人工智能正在改变金融市场。凭借我们无与伦比的数据、基础设施和合作伙伴关系，伦敦证券交易所集团在这一变革中占据着独特的领先地位。了解更多关于伦敦证券交易所集团人工智能战略的信息

伦敦证券交易所集团是全球领先的金融市场基础设施和数据提供商，在世界金融体系中发挥着重要的社会和经济作用。

凭借开放的理念、值得信赖的专业知识和全球规模，我们助力客户及其社区实现可持续增长和稳定。我们是专业的合作伙伴，在数据和分析、指数、资本形成以及跨资产类别的交易执行、清算和风险管理领域拥有丰富的经验、深厚的知识和全球影响力。

伦敦证券交易所集团总部位于英国，业务遍及欧洲、中东和非洲地区、北美、拉丁美洲和亚太地区的65个国家。集团在全球拥有超过26,000名员工，其中超过一半位于亚太地区。

LSEG 的股票代码为 LSEG。
