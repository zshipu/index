推进克劳德金融服务人类学 --- Advancing Claude for Financial Services Anthropic --知识铺

2025年10月28日
| 阅读

我们正在通过 Excel 插件、实时市场数据和投资组合分析的附加连接器以及新的预建代理技能（如构建折现现金流模型和启动覆盖报告）来扩展 Claude for Financial Services 。

这些更新基于 Sonnet 4.5 在财务任务方面的卓越表现，其准确率高达 55.3%，超越了 Vals AI 的财务代理基准 。这些更新增强了 Claude 的智能，为耗时却关键的财务工作提供了解决方案，并内置于行业首选工具中。

克劳德的 Excel

我们即将发布 Claude for Excel 的 Beta 版研究预览版。用户可以通过 Microsoft Excel 的侧边栏直接使用 Claude 进行操作，Claude 可以读取、分析、修改和创建新的 Excel 工作簿。Claude 的操作完全透明：它会跟踪并解释更改内容，并允许用户直接导航到其解释中引用的单元格。

克劳德分析了包含 Acme Grille, Inc. 2020 年至 2024 年合并损益表的电子表格，为财务建模提供了实时指导。

这意味着 Claude 可以讨论电子表格的工作原理，在保留其结构和公式依赖关系的同时对其进行修改，调试和修复单元格公式，用新数据和假设填充模板，或者从头开始构建新的电子表格。

Claude for Excel 增强了我们与 Microsoft 应用程序的现有集成。在 Claude 应用中，Claude 还可以创建和编辑文件，包括 Excel 电子表格和 PowerPoint 幻灯片，并连接到 Microsoft 365 以搜索文件、电子邮件和 Teams 对话。部分 Claude 模型也可在 Microsoft Copilot Studio 和 Researcher Agent 中使用。

Claude for Excel 现已推出 Beta 测试版，面向 Max、Enterprise 和 Teams 用户提供研究预览。我们将收集 1,000 名初始用户的真实反馈，然后再更广泛地推广该功能。如需加入候补名单， 请点击此处 。

为克劳德提供实时信息

连接器让 Claude 能够直接访问外部工具和平台。7 月 ，我们新增了 S&P Capital IQ、Daloopa、Morningstar 和 PitchBook 的连接器。我们将继续添加新的连接器，让 Claude 能够实时访问更多信息：

  • Aiera 为 Claude 提供实时收益电话会议记录和投资者活动摘要，如股东大会、演示和会议；
  • Aiera 的连接器还支持来自 Third Bridge 的数据馈送，这使得 Claude 可以访问来自专家和前任高管的洞察访谈、公司情报和行业分析库；
  • Chronograph 为私募股权投资者提供投资组合监控和尽职调查的运营和财务信息，包括绩效指标、估值和基金级数据；
  • Egnyte 使 Claude 能够安全地搜索内部数据室、投资文件和批准的财务模型的许可数据，同时保持受管控的访问控制；
  • LSEG 将 Claude 与实时市场数据连接起来，包括固定收益定价、股票、外汇汇率、宏观经济指标以及分析师对其他重要财务指标的估计；
  • 穆迪提供专有信用评级、研究和公司数据（包括超过 6 亿家上市公司和私营公司的所有权、财务和新闻），支持合规、信用分析和业务发展方面的工作和研究；
  • MT Newswires 为 Claude 提供有关金融市场和经济的最新全球多资产类别新闻。

有关 MCP 连接器设置的详细信息以及最大限度发挥每个连接器优势的提示指导，请参阅此处的文档 。

财务任务的新代理技能

本月初，我们推出了 “代理技能” 。技能文件夹包含 Claude 可用于执行指定任务的指令、脚本和资源。这些技能适用于所有 Claude 应用，包括 Claude.ai 、Claude Code 和我们的 API。为了使 Claude 更好地完成金融服务任务，我们添加了 6 项新技能：

  • 可比公司分析 ，包括估值倍数和运营指标，可通过更新数据轻松刷新；
  • 折现现金流模型 ，包括完整的自由现金流预测、加权平均资本成本计算、情景切换和敏感度表；
  • 尽职调查数据包 ，将数据室文档处理成包含财务信息、客户名单和合同条款的 Excel 电子表格；
  • 公司宣传片和简介 、用于宣传册和买家名单的简明公司概述；
  • 收益分析 ，研究季度业绩和财务状况以提取重要指标、指导变化和管理评论；
  • 通过行业分析、公司深度研究和估值框架启动覆盖报告 。

与 Claude for Excel 一样，这些新技能正在预览版中，面向 Max、Enterprise 和 Teams 用户推出。您可以在此处代表您的团队或组织注册。
