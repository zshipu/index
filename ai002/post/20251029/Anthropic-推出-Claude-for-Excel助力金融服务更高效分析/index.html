Anthropic 推出 Claude for Excel，助力金融服务更高效分析 --知识铺

2025年10月29日
| 阅读

近日，Anthropic 公司宣布推出 Claude for Excel，旨在进一步拓展其在金融服务领域的应用。此更新使得 Claude 能够作为金融专业人士的 AI 助手，尤其是那些使用 Microsoft 工具的用户。Claude for Excel 目前处于研究预览的测试阶段，用户可以通过 Excel 侧边栏直接与 Claude 进行互动。

通过这项新功能，Claude 能够读取、分析并修改工作簿，且每一次的更改都能得到清晰的追踪和解释。这项功能尤其适合帮助用户调试公式、生成财务模型以及从零开始创建新的电子表格，从而大大节省了分析师的时间，通常这些工作需要花费数小时才能完成。

此次 Excel 集成是 Anthropic 与 Microsoft 已有合作关系的重要补充。用户在 Claude 应用中不仅能够创建和编辑 Excel 和 PowerPoint 文件，还可以在 Microsoft365中搜索信息，包括电子邮件和 Teams 对话。此外，部分 Claude 模型还嵌入到 Microsoft Copilot Studio 和 Researcher Agent 中。

为了推动新功能的使用，Anthropic 邀请了来自 Max、Enterprise 和 Teams 层级的1000名初期用户进行 Excel 插件的测试，计划在后续进行更广泛的推广。

除了 Excel 功能外，Anthropic 还推出了新的连接器，让 Claude 能够访问实时金融数据。这些连接器包括与 Aiera（用于收益电话会议）、LSEG(市场数据)、Moody’s(信用评级)以及 Egnyte(安全文档访问)等平台的集成。此外，Chronograph、Third Bridge 和 MT Newswires 等更多数据源也进一步增强了 Claude 在投资组合监控、估值和研究方面的能力。

Anthropic 还发布了六个为金融专业人士设计的预构建代理技能。这些技能能够帮助生成折现现金流模型、进行尽职调查、制作公司概况以及汇编启动覆盖报告。这些更新基于 Sonnet4.5在金融基准测试中的卓越表现，得分为55.3%，为银行、资产管理公司和投资公司提供更全面的支持，涵盖前台、中台和后台的各项职能。

📊 新功能支持实时金融数据接入，助力专业人士决策。  

🤖 六种预构建代理技能提升 Claude 在金融领域的应用。
