Claude Skills：为您的工作流程定制 AI Anthropic --- Claude Skills Customize AI for your workflows Anthropic --知识铺

2025年10月28日
| 阅读

Claude 现在可以使用_技能_来改进其执行特定任务的方式。技能文件夹包含指令、脚本和资源，Claude 可以在需要时加载它们。

Claude 只会在技能与当前任务相关时才会使用。使用技能后，Claude 能够更好地完成特定任务，例如使用 Excel 或遵循组织的品牌准则。

您已经在 Claude 应用中体验过技能的运作，Claude 使用它们来创建电子表格和演示文稿等文件。现在，您可以构建自己的技能，并在 Claude 应用、Claude Code 和我们的 API 中使用它们。

技能如何发挥作用

在执行任务时，Claude 会扫描可用的技能，寻找相关的匹配项。一旦匹配成功，它只会加载所需的最少信息和文件，从而确保 Claude 在获取专业知识的同时保持快速运行。

技能有：

  • 可组合 ：技能可以叠加。Claude 会自动识别所需技能并协调它们的使用。
  • 可移植性 ：技能在所有地方使用相同的格式。一次构建，即可在 Claude 应用、Claude 代码和 API 中使用。
  • 高效 ：仅在需要时加载所需的内容。
  • 强大 ：技能可以包括可执行代码，用于传统编程比令牌生成更可靠的任务。

您可以将技能视为定制的入门材料，将其与专业知识整合在一起，让 Claude 成为您最关心的领域的专家。如需深入了解代理技能的设计模式、架构和开发最佳实践，请阅读我们的工程博客。

克劳德·阿普斯

专业版、旗舰版、团队版和企业版用户均可使用技能。我们提供用于常见任务（例如文档创建）的技能、可自定义的示例以及创建自定义技能的功能。

Claude 会根据您的任务自动调用相关技能，无需手动选择。您甚至会在 Claude 的思维链中看到技能。

创建技能非常简单。“技能创建器”技能提供交互式指导：Claude 会询问您的工作流程、生成文件夹结构、格式化 SKILL.md 文件，并打包您所需的资源。无需手动编辑文件。

在 “设置” 中启用技能。对于团队和企业用户，管理员必须首先在组织范围内启用技能。

Claude 开发者平台（API）

代理技能（我们通常简称为技能）现在可以添加到消息 API 请求中，新的 /v1/skills 端点使开发人员能够通过编程控制自定义技能的版本控制和管理。技能需要代码执行工具 Beta 版，该工具可提供运行技能所需的安全环境。

使用 Anthropic 创建的技能，让 Claude 读取并生成包含公式的专业 Excel 电子表格、PowerPoint 演示文稿、Word 文档和可填写的 PDF。开发者可以创建自定义技能，扩展 Claude 的功能，以满足特定用例的需求。

开发者还可以通过 Claude Console 轻松创建、查看和升级技能版本。

浏览文档或 Anthropic Academy 以了解更多信息。
