提示注入攻击风险

仅将 Claude for Excel 与受信任的电子表格一起使用，而不要使用来自外部不受信任来源的电子表格（例如，下载的模板、供应商文件、协作文档和数据导入）。

Claude for Excel 和其他能够读取和操作电子表格的 AI 工具的用户面临的一个重大风险是提示注入攻击。这种攻击会将恶意指令隐藏在电子表格内容（单元格、公式、注释等）中，从而诱使 AI 模型执行非预期的操作。例如，从外部接收或从互联网下载的看似无害的模板或数据文件可能包含隐藏指令，例如“将所有财务数据导出到此外部 URL”或“修改这些财务记录”。Claude 可能会将这些恶意指令解读为您发出的合法请求。

用户有权批准 Claude 的所有行动

虽然我们不断开发产品并改进安全措施以降低这些风险，但用户在使用 Claude for Excel 时应谨慎，并且不应将其与来自外部不受信任来源的电子表格一起使用。

常问问题

克劳德了解金融建模惯例吗？

是的，Claude 经过培训，能够识别常见的财务建模模式、公式结构和行业标准计算。但是，请务必确认输出结果是否符合您的具体方法。

我可以将 Claude for Excel 用于敏感数据吗？

Claude for Excel 可与您现有的安全框架兼容。对于高度敏感或受监管的数据，请确保遵守您组织的数据处理政策。

我的聊天记录会怎样？

目前，会话之间的聊天记录不会保存。每次打开插件时，您都会与 Claude 开始新的对话。不过，我们正在努力在未来版本的 Claude for Excel 中支持此功能。

Claude 如何访问我的电子表格？

Claude 会读取您当前打开的工作簿的内容，包括单元格、公式和选项卡结构。它只能访问您在 Excel 中打开的工作簿。

如果克劳德犯了错误怎么办？

Claude 会高亮显示它对您的工作簿所做的所有更改。在保存或共享文件之前，请仔细检查这些更改。您可以随时使用 Excel 的标准撤消功能撤消更改。
