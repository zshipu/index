Excel 的 COPILOT 函数：为什么我之前的 AI 建议突然改变了 --- Excel’s Game-Changing COPILOT Function Why My Previous AI Recommendations Just Changed --知识铺

2025年10月28日
| 阅读

简而言之： 微软刚刚在 Excel 中推出了革命性的 COPILOT 函数，将 AI 直接引入电子表格单元格。这彻底改变了我之前在 Copilot 与 Claude 对比中所推荐的所有内容。虽然 Claude 仍然擅长处理复杂的 Power Query 工作，但 Excel 的新 COPILOT 函数提供了速度、灵活性和原生集成，彻底改变了我们处理数据的方式——无需 OneDrive 自动保存。COPILOT 函数由大型语言模型和先进的 AI 模型提供支持，使其在 Excel 中具备生成和探索能力。

就在我发表 Copilot 与 Claude 在 Excel 公式构建方面的对比分析几周后，微软又抛出一枚重磅炸弹，彻底改变了 Excel 中的 AI 格局。全新的 COPILOT 函数彻底重塑了人工智能与电子表格的集成方式。值得注意的是，COPILOT 最初是作为 Excel Labs 及其插件的一部分进行测试的，目前正处于 Beta 测试阶段，仅向部分用户开放，微软正在持续完善其功能。

在之前的比较中，我注意到 Copilot 虽然提供了无缝集成，但其在自动保存要求、响应时间较慢以及公式选择有限等方面的局限性，使得 Claude 成为高级用户的最佳选择。新的 COPILOT 功能不仅解决了这些问题，还引入了之前两款工具均未提供的功能。

COPILOT 功能的革命性之处

Excel 的全新 Copilot 函数是一款 AI 驱动的工具，它将 AI 辅助功能从外部聊天界面转换为原生 Excel 函数，其功能与 VLOOKUP、SUMIF 或任何其他公式类似。这项新功能允许您直接在单元格中输入 AI 提示并获得即时结果，从而取代传统公式。

基本语法如下：

=COPILOT(提示部分 1，[上下文 1]，[提示部分 2]，[上下文 2]，…)

通过此新功能，用户可以直接在单元格中输入自然语言提示来与 Copilot 进行交互，从而无需技术专业知识即可轻松生成、分类或分析数据。

但真正的魔力是当你亲眼看到它发挥作用时才会发生。

轻松实现数据分类

还记得花费数小时对费用报告或客户反馈进行分类吗？COPILOT 函数可以立即处理这些任务。它还可以自动对数据（例如调查回复或反馈）进行分类，从而简化分析。正如微软的公告视频所示，您可以分析信用卡信息并自动生成费用类别：

=COPILOT（“分析信用卡描述并列出可用于分组支出的简短类别列表”，B2：B50）

COPILOT 只需几秒钟即可扫描您的数据并推荐相关类别。该功能具有情境感知功能——它能够理解您的业务背景并相应地提供定制建议，甚至融入常见的业务命名惯例和商户类别。

数据生成和内容创建

最令人印象深刻的演示之一是 COPILOT 生成电子表格中尚未包含的数据。您可以要求它创建特定行业的公司列表，根据产品描述生成营销关键词，甚至集思广益，提出项目创意——所有这些都直接在 Excel 单元格中完成。COPILOT 函数还可以激发新项目或营销策略的灵感，使其成为一个非常实用的头脑风暴工具。

例如，您可能会问：“列出按收入和主要业务重点排名前十的 SaaS 公司。” COPILOT 从其训练数据中提取这些信息，并直接在您的单元格中创建结构化输出，而这之前需要外部研究和手动数据输入。此外，该功能还可用于汇总来自客户或团队成员的反馈，为您的分析提供快速洞察。

公式解释以及如何用简单的英语引用单元格值

COPILOT 函数擅长简化复杂的公式。使用 FORMULATEXT 函数，您可以引用电子表格中的任何公式，并让 COPILOT 用通俗易懂的语言进行解释：

=COPILOT（“用简单的英语解释这个公式”，FORMULATEXT（D5））

这样就消除了从同事那里继承电子表格或几个月后重新查看自己的复杂公式时的猜测。

来自 Web 和企业数据的数据分析和洞察

或许最强大的是，COPILOT 可以分析整个数据范围并提供智能摘要。该功能根据用户的提示生成所需的输出，提供定制的洞察。您无需手动浏览损益表，只需让 COPILOT 分析数据并突出显示关键发现即可：

=COPILOT（“分析损益表并总结要点”，A1：C20）

该函数使用 TEXTJOIN 和 CHAR 函数来格式化具有适当换行符的响应，从而创建可读的段落样式输出，这些输出看起来更像是分析师报告而不是公式结果。

数据清理变得简单

数据清理，一直以来都是 Excel 中最繁琐的任务之一，现在变得几乎毫不费力。演示展示了 COPILOT 如何从格式不一致的地址数据中提取邮政编码，然后标准化这些通常需要复杂正则表达式或手动更正的格式任务。

常规数据处理的重要提示 ：虽然 COPILOT 函数擅长一次性数据清理任务，但它并非重复性数据清理工作的最佳解决方案。如果您定期收到需要每月或每季度执行相同清理步骤的外部报告，Power Query 仍然是最佳选择。Power Query 允许您构建可重复使用的数据转换工作流，并自动使用新数据进行刷新。COPILOT 可用于临时清理任务和探索性数据工作，但对于重复性数据处理工作流，则需要投资 Power Query 解决方案。

这如何改变我之前的建议

COPILOT 功能面向未来，随着微软继续开发其 AI 功能，预计将持续改进并推出新功能。

速度：游戏规则的改变者

在我之前的分析中，我批评了 Copilot 的响应速度比 Claude 慢。COPILOT 函数的运行速度与 Excel 的计算引擎相同。这得益于 Excel 的计算引擎，它确保了实时更新，并与 COPILOT 等 AI 功能无缝集成。当您的数据发生变化时，结果会自动更新——就像任何其他 Excel 函数一样。这代表着从较慢的基于聊天的界面到即时原生性能的根本性转变。

不再需要自动保存

我对原版 Copilot 最大的不满在于它需要 OneDrive 自动保存，这强制要求用户采用云优先的工作流程，很多用户觉得这很不方便。COPILOT 函数适用于任何 Excel 文件，无论是本地存储、网络驱动器还是云端。这解决了 Claude 对那些喜欢控制保存工作流程的用户更具吸引力的主要顾虑。

集成与外部工具

虽然 Claude 仍然需要在应用程序之间复制粘贴，但 COPILOT 函数将 AI 功能直接引入电子表格网格。结果将成为工作簿的一部分，在源数据发生变化时自动更新，并可供其他公式引用。通过利用电子表格参考单元格值，用户可以确保 AI 生成的结果根据特定单元格或范围的变化动态更新。这种程度的集成是外部 AI 工具无法实现的。

解读的最佳实践，而非精确的最佳实践

微软强调了一个重要的局限性：COPILOT 函数擅长解释、分类和洞察，而非精确数学。然而，它可能无法始终提供准确的数值计算，因此对于关键数据，手动验证至关重要。这完全符合大多数用户实际需要 AI 辅助的需要——理解数据模式、分类信息并生成洞察，而不是执行复杂的计算。

更新的建议：何时使用什么

在以下情况下选择 Excel 的新 Copilot 功能：

  • 数据分类任务 ：对费用、客户反馈或调查回复进行分类
  • 内容生成 ：创建列表、集思广益或生成结构化数据
  • 数据分析 ：总结趋势、突出关键发现或解释模式
  • 公式解释 ：理解他人构建的复杂公式
  • 数据清理 ：从杂乱的文本中提取信息或标准化格式
  • 在本地处理文件 ：当您不想使用 OneDrive 自动保存时
  • 单细胞使用 ：COPILOT 函数可以在单个细胞中使用，进行快速分析或内容生成。
  • 内部数据处理 ：该函数对工作簿内的内部数据进行操作。请确保引用或导入当前或内部数据以获得准确的结果，因为 COPILOT 无法访问实时网络或企业数据。

选择 Claude 的时机：

  • Power Query 开发 ：COPILOT 函数在 Power Query 编辑器中不起作用
  • 复杂 M 代码 ：Claude 在数据转换脚本方面仍然占据优势
  • 多种公式选择 ：当你想看到解决同一问题的不同方法时
  • 跨平台工作 ：在 Excel 之外的不同应用程序上工作时
  • 教育深度 ：当您需要详细解释 Excel 概念和最佳实践时

在以下情况下选择传统副驾驶：

  • 对话协助 ：当您更喜欢通过聊天互动来获得指导时
  • 模板创建 ：用于生成整个工作簿结构和布局
  • 多步骤工作流程 ：当您需要指导完成复杂的多阶段流程时

公式和语法：如何编写和使用 COPILOT

Excel 中的 COPILOT 函数引入了一种与数据交互的全新方式——直接在电子表格中使用自然语言提示。使用此函数，您可以在熟悉的 Excel 环境中引用单元格值、提出问题并获得 AI 生成的结果。其基本语法非常简单：

=COPILOT(prompt_part1, [context1], [prompt_part2], [context2], …)

这种结构允许您将指令（提示）与工作表中的特定数据（上下文）结合起来。例如，您可以要求 Excel 中的 Copilot 函数“总结这些调查回复的要点”，并引用包含客户反馈的单元格区域。然后，该函数会处理您的请求并直接在电子表格中返回结果，让您无需离开 Excel 即可轻松整理数据、生成摘要或对信息进行分类。

无论您是想分析趋势、集思广益，还是自动执行重复性任务，Copilot 功能的自然语言界面都能让各种技能水平的用户轻松上手。通过将 AI 与 Excel 强大的计算引擎相结合，您可以从数据中解锁新的效率和洞察。

了解 COPILOT 公式结构

COPILOT 公式的结构旨在实现最大的灵活性和易用性。您可以根据具体需求混合搭配提示和上下文引用。例如，如果您想快速对客户反馈进行分类，可以输入类似“分类此反馈”的提示，并引用一列回复。Copilot 函数会分析每个条目，并将分类结果直接输出到您选择的单元格中。

这种方法无需手动读取、标记或汇总大量数据。相反，Copilot 公式会承担繁重的工作，使其成为客户反馈分析、调查回复分类以及从复杂数据集生成简洁摘要等任务的理想选择。通过利用 Copilot 函数，您可以专注于解释结果和做出决策，而不必陷入手动数据处理的泥潭。

底线：Excel 的新时代

COPILOT 函数代表了 Excel 自引入动态数组以来最重要的创新。它将 AI 从外部助手转变为原生的电子表格功能，解决了 Claude 等外部工具在许多用例中更具吸引力的主要限制。COPILOT 函数利用先进的 AI 模型和大型语言模型，根据用户提示和引用数据生成响应，从而在 Excel 中实现自然语言理解和内容生成。

与某些外部工具不同，COPILOT 函数无法直接访问实时网络数据，因此用户必须根据需要导入其他数据进行全面分析。任何输入都将保密，不会用于训练或改进 AI 模型，从而确保用户隐私和数据安全。这标志着与 INDEX MATCH 等传统工具的重大转变，因为从公式到 AI 驱动功能的演变正在改变 Excel 中的数据分析方式。

对于大多数用户来说，这彻底改变了 AI 辅助的格局。原生集成、自动更新、本地文件支持和即时性能的结合，为使用 Excel 内置 AI 功能而非外部替代方案提供了强有力的理由。

然而，这并不意味着外部 AI 工具就过时了。Claude 在 Power Query 开发、教学讲解以及需要多种方法的复杂分析方面仍具有显著优势。现在，理想的工作流程是使用 COPILOT 函数进行日常数据操作和分类，同时利用 Claude 进行高级开发工作和学习。

正如微软首席执行官萨蒂亚·纳德拉所说：“我就是喜欢这个。” COPILOT 功能感觉就像电子表格 AI 辅助的自然演变——无缝集成、情境感知、立即可用。

在集成和功能之间进行选择的时代已经结束。Excel 现在同时提供两者。

常见问题解答

  1. 我可以相信 COPILOT 函数输出的准确性吗？
    COPILOT 函数擅长解释、分类和生成洞察，但它并非专为精密数学或监管报告而设计。请务必审查并验证其结果，尤其是在进行关键业务决策时。应将其视为探索和模式识别的强大助手，而非结构化数据验证的替代品。

  2. COPILOT 函数如何影响共享工作簿中的协作？
    共享工作簿时，COPILOT 函数的行为与其他公式相同——其他用户将结果视为单元格值。但是，只有拥有适当 Copilot 许可证的用户才能刷新或修改基于 COPILOT 的公式。这确保了跨团队的兼容性，同时满足 Microsoft 的许可要求。

  3. COPILOT 函数可以替代 Power Query 进行数据清理和转换吗？
    并非完全如此。虽然 COPILOT 可以更轻松地进行临时数据清理（例如提取邮政编码或对费用进行分类），但它并非为可重复的工作流程而设计。如果您需要可重复使用的自动化转换功能，并每月或每季度刷新新数据，那么 Power Query 仍然是最佳选择。COPILOT 函数最适合快速洞察和一次性清理，而 Power Query 则处理重复的数据处理。
