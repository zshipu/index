Codex 介绍

Codex 是 OpenAI 出品的开源代码神器，用 Rust 语言写的，速度快，效率高，能读懂、修改和运行代码，你可以更快地完成开发，轻松搞定 bug，还能理解那些你看不懂的代码。

Codex 主要有以下 3 种使用方式：

  • Codex CLI —— 直接可以在任意终端里运行；

  • Codex IDE —— 在 IDE 中以插件形式运行；

  • Codex Cloud —— 绑定 Github 在云端执行任务；

也就是说，CodeX 不仅可以在任意终端运行，如果你装上 Codex VS Code 插件，你就能在 IDE 里直接用 Codex，或者把任务交给云端处理，简直不要太方便！

还有一个好消息，CodeX 不用单独开付费 API 了，Codex 已支持 ChatGPT Plus、Pro、Team、Edu 或企业版账号，每个类型的账户，对本地和云端任务的使用限制不同。

安装 CodeX CLI

Codex 的安装和运行非常简单，和 Claude Code 一样，仅需要一条命令即可安装。

使用 npm 进行安装：

npm install -g @openai/codex

Mac 也可以使用 Homebrew 安装：

brew install codex

后续升级也是一样的命令。

安装完成后使用下面的命令查看版本：

codex –version

运行 CodeX CLI

运行 Codex 如此简单：

codex

第一次运行 Codex CLI 的时候，会提示你进行身份验证：

如果你已经订阅 ChatGPT Plus/ Pro 等，强烈建议你用自己的 ChatGPT 账号登录，这样不需要用付费 API，这样就能在原来的 Plus、Pro 等付费账号中使用 Codex 的全部功能啦。

授权成功后，我们就可以来测试一下，在某个项目路径下执行：

codex “解释一下这个项目”

它会检索所有项目文件并解释项目功能：

安装 CodeX IDE 插件

Codex 的 IDE 插件可以和 VS Code 的分支版本一起使用，比如 Insiders、Cursor 或 Windsurf。

你可以从 Visual Studio Code 商店搜索并安装 Codex 插件：

安装完后右上角会多一个 Codex 专用图标：

点击该图标就能快速进入 Codex 对话：

装好 CodeX 插件之后，第一次使用系统会提示你用 ChatGPT 账号登录，强烈推荐这种方式。如果你想用 API 密钥，可以选择相应选项，然后把 API 密钥配置到环境变量里。

后续在你的编辑器里用 Codex，就能无缝聊天、编辑和预览修改啦！它能理解你打开的文件和选中的代码，这样你就能用更短的提示，更快地获得更相关的结果。

另外，在 IDE 插件中更改模型、推理等级什么的也更加方便。

目前，Codex 的 VS Code 扩展在 macOS 和 Linux 上都能用，Windows 的支持还在试验阶段。

总结

Codex 使用起来一点也不输 Claude Code，它还是 OpenAI 出品，同时支持本地和云端运行，能读懂、解释、修改和执行代码，安装简单，速度飞快，还能接入 VS Code 插件无缝协作。

不管你是写代码、改 bug、看不懂旧项目，Codex 都能帮上大忙。

更关键的是，你的 Plus 和 Pro 账号可直接授权使用 CodeX，不用额外付费 API，免费账号也能通过配置接入 API 付费使用。

如果喜欢用 ChatGPT 的，现在不用开多个会员了，现在就能同时使用 OpenAI 的 CodeX 和 GPT-5 旗舰模型来实现 AI 编程了。

我已经全面拥抱 CodeX 了，强烈建议大家都试试。
