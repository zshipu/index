MiniMax 发布 M2 开源型号，速度翻倍，价格仅为 Claude Sonnet 的 8_ · TechNode --- MiniMax releases M2 open-source model , offering double speed at 8_ of Claude Sonnet’s price · TechNode --知识铺

2025年10月28日
| 阅读

中国人工智能初创公司 MiniMax 推出了其开源大型语言模型 MiniMax M2，该模型专为代理工作流程和编码任务构建。据称，该模型的推理速度是 Anthropic 的 Claude Sonnet 4.5 的两倍，而 API 成本仅为其 8%，目前可在限定时间内免费使用。

MiniMax 表示，M2 在 Claude Code、Cursor 和 Cline 等平台上的代码生成和工具使用方面表现出色。该公司已在 Hugging Face 上发布了模型权重，并支持 vLLM 和 SGLang 的部署，同时还提供基于 M2 的代理平台，该平台具有“Lightning”和“Pro”模式，可用于通用和高级自动化。[TechNode 报道]
