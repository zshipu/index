做T的手续费及补手续费点数的具体计算如下，可以参考一下：
1.做T的成本计算
假设您通过某券商进行一次做T操作，买入和卖出各5000股，具体费用如下：
买入成本：
成交金额 ：6.13元/股 × 5000股 = 30,650元
券商佣金 ：30,650元 × 0.0015% = 4.5975元（假设佣金率为万分之1.5）
过户费：30,650元 × 0.002% = 0.613元（仅上交所股票）
总买入成本：4.5975 + 0.613 = 5.2105元
卖出成本：
成交金额：6.24元/股 × 5000股 = 31,200元
券商佣金：31,200元 × 0.0015% = 4.68元（假设佣金率为万分之1.5）
印花税：31,200元 × 0.1% = 31.2元
过户费：31,200元 × 0.002% = 0.624元（仅上交所股票）
总卖出成本：4.68 + 31.2 + 0.624 = 36.504元
总交易成本：5.2105 + 36.504 = 41.7145元
2. 盈亏平衡点计算
为了不亏手续费，您需要确保做T的盈利能够覆盖上述总交易成本。假设您做T的操作是在同一天内完成的，那么盈亏平衡点可以通过以下公式计算：
每股差价 = 总交易成本 / 股数
每股差价 = 41.7145元 / 5000股 ≈ 0.00834元（约等于0.834分）
因此，为了不亏手续费，您需要在做T时至少赚取0.834分/股的差价。换句话说，股价上涨超过0.834分才能覆盖手续费。
3. 做T的盈亏平衡点总结：根据上述计算，一般情况下，做T赚0.8个点左右可以抵消手续费。具体来说，对于高价股，由于每手交易金额较大，即使差价较小也能覆盖手续费。对于低价股，由于每手交易金额较小，差价需要稍大一些才能覆盖手续费。对于极低价股，如股价低于5元的股票，做T的难度较大，因为差价需要更大才能覆盖手续费。此时，建议谨慎操作，避免频繁交易导致手续费过高。
